    ▇  資安講座（慢霧團隊 Lisa）

    ▍受害者現身說法

    • 一位受害者加入了一個偽裝成幣安交易所的群組，群內成員和官方客服實際上是由同一批騙子假冒，他們利用與「真正幣安近期活動類似的假活動名稱」來誘騙，使受害者在網上查證時誤以為該活動真實，從而落入圈套。

    • 一開始（前兩次）可以正常提領，等受害者入金越來越多，提現開始出現問題，當向假客服反映時，會被要求再投入更多代幣作為保證金，以解鎖提款功能。

    • 延伸閱讀：再傳詐騙！假裝是「幣安交易所」釣魚，用戶遭詐騙 50 萬。
      　

    ▍在區塊鏈黑暗森林安身立命三大方向

    • Refer to 創建錢包與私鑰備份 ⚠️

      • 確保下載錢包的官網網址無誤（差一個 / 或 s 就有差），以免下載到假錢包、被篡改程式碼的錢包、惡意軟體等。

      • 不要連線傳輸私鑰（用「手寫」的方式備份，不要截圖、拍照或存在雲端）。

      • 即使離線，也要避免在公開場合觀看私鑰，防止被攝影機窺視。

    • Refer to 錢包使用過程中的陷阱 ⚠️

      • 盲簽釣魚：不懂簽署內容就簽下去。

      • 假錢包地址：轉帳目標被惡意修改。

      • 智能合約詐騙：詐騙者將惡意規則以程式碼的方式嵌入到智能合約裡。

      • 地址中毒（Address poisoning）：詐騙者建立一個山寨地址（開頭 5 碼及結尾 4 碼相同，但中間不同），然後用這個地址空投代幣給被害人，這樣的手法很像是在「投毒」，拿來魚目混珠「感染」受害人的交易紀錄。未來受害人在做錢包內地址之間的資金劃轉時，如果不小心複製到了詐騙者的山寨地址，就有可能將資金誤轉。

    • Refer to 若不幸中標該怎麼辦 ⚠️

      • 止損：先趕快把剩下的幣轉走。

      • 鏈上凍結：尋求發行商／交易所協助。

      • 聯繫區塊鏈安全團隊。

      • 做好自己的功課 區塊鏈黑暗森林自救手冊，事前預防更勝於治療。

    ▇  正課：AI 繪圖工具，介紹與使用入門（講師：陳財貓）

    ▍DALL-E3

    • OpenAI 公司開發的文生圖模型。

    • 訂閱 ChatGPT Plus 以後，直接在 ChatGPT 裡面使用。

    • 現代文生圖模型往往忽略掉某些詞語或描述，迫使用戶學習 prompt engineering。DALL-E3 在「根據你提供的文本，精確生成圖像」的能力大躍進。

    • 特性

      • 最容易上手（自然的溝通方式）。

      • 可操作的空間較小（控制、調整或自定義的選項和功能有限）。

      • 能夠嚴格遵守 prompt 中提到的細節。

      • 有 ChatGPT 加持幫忙幹活（當我們透過 ChatGPT 使用 DALL-E3 時，ChatGPT 會作為「中介」演繹我們的想法）。

    • 文生圖

      • 描述畫面，生成圖片（ex 畫一隻紫色的倉鼠）。

      • 請 ChatGPT 發揮創意（ex 如果你是一隻倉鼠，請畫出你看到的世界，第一鼠稱）。

    • 圖生圖

      • 上傳圖片給 ChatGPT 看，基於這張圖片生成新的圖片。

      • 因為 ChatGPT 支援多模態（multi-modal）。

      • 意思是除了傳統的文字對話外，用戶可以透過「拍照」或「錄音」來與 ChatGPT 進行更多元的互動。用單一模型就可以處理多方面的任務。

    • 保持一致性（ex 請保持當前圖片種子不變，讓小倉鼠手上捧一個瓜子）

    種子（feed）通常指的是隨機生成過程中，用來初始化生成器的一個數值。因此，在剛才的範例中指的是，無論進行何種操作，都不要改變用於生成圖片的初始參數，以確保結果的一致性或可重現性。

    • 你可以做一個難纏的甲方

      • 要求特定的圖片比例（ex 橫）。

      • 拍攝角度（ex 放大視野，鏡頭拉遠）。

      • 多輪對話，逐步改進。

    ▍Midjourney

    • 2022 年，奪下美國科羅拉多州博覽會數位藝術類首獎的〈太空歌劇院〉（Théâtre D’opéra Spatial）是利用 Midjourney 產生的作品，也讓這款工具爆火。

    • 作為一個 bot，寄生在一款名為 Discord 的社交軟體上。

    • 新手上路

      • 安裝並註冊 Discord（也有免安裝的網頁版）
        https://discord.com/

      • 加入 Midjourney Discord 伺服器
        https://discord.gg/midjourney

      • /subscribe：訂閱以後可以 1 on 1 小房間

    • 最好的學習資源來自於 Midjourney 官方入門指南。

    • /image（想像）

      • 基本操作只要接 text prompt 就可以了。

      • 進階操作還可以設定 image prompt（墊圖）和 parameters（參數）。

    墊圖必須是線上圖片的超連結。先把圖片上傳到 Discord，然後在圖片上長按或右鍵按滑鼠，然後選擇「複製圖片網址」以取得 URL。

    • 生成圖片後，2 x 2 網格的下方出現兩行按鈕

      • Upscale（放大影像，從 2 x 2 網格獨立分離出來）

      • Variation（變化）

      • Re-run or Re-roll（重新運算）🔄

    • 獨立出單張圖片後，你可以進一步：

    • /settings（設定）

      • Midjourney 有很多版本，例如針對「二次元、動漫風格」所預訓練的 🌈 Niji-journey V5 模型。

    • MidJourney 要做 prompt engineering 但不會太複雜，訣竅在於：

      • 你的腦子裡，要知道世界上還有這樣用來描述的詞彙。

      • 知道那些詞彙背後映射到什麼樣的特徵。
        　

    • 如何發想 prompt？

      • 觀摩｜Midjourney 雜誌
        mag.midjourney.com

      • 觀摩｜藝術家風格
        midlibrary.io/categories

      • 觀摩｜Midjourney 社群畫廊
        https://www.midjourney.com/showcase

      • 餵食 ChatGPT 幾張樣本圖片，讓它也產出一個 prompt。
        　

    • - - parameter（參數）

      • 圖片預設 1:1，用 --aspect 或 --ar 接「縱橫比」。

      • --stylize 接數值，數值越低越似童繪，數值越高越寫實。

      • --chaos 或 --c 接數值，數值越低越中規中矩，數值越高越天馬行空。

    • Fast, Relax, & Turbo Modes

      • Fast Mode 是預設模式，會消耗你訂閱方案下的「每月 GPU 時間」。

      • Relax Mode 不會消耗 GPU 時間，而是將你放入等待隊列中，服務的優先順序將根據你使用此模式的頻率（偶爾或頻繁）來決定。

      • 如果你很急，開啟 Turbo Mode 會讓你的執行速度提升四倍，但是消耗的訂閱 GPU 分鐘數是 Fast Mode 模式的兩倍。

    ▍Stable Diffusion

    • 與其說是一款工具，Stable Diffusion 更像是圍繞著「開源（open source）模型」建造的龐大生態系。

    • 開源，意味著你可以把模型運行在自己的電腦上，也可以自由修改模型的原始碼——不過對硬體（顯示卡）有一定程度的要求。

    • Stable Diffusion 3 震撼發布，採用 Sora 同源技術，文字終於不亂碼了

      • 可以用更自然的語言去描述 prompt。

      • 未來，針對圖片生成的 prompt engineering 可能會越來越不重要。
