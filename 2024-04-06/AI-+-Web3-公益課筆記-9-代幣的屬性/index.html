    ▇  正課：代幣的屬性（講師：Ashley）

    ▍Coin（原生代幣）

    • 區塊鏈底層直接發行，用以實現區塊鏈基本運作的代幣。

    • 用來支付主網轉帳費用。

    在區塊鏈的世界中有兩種不同的網路：主網（Mainnet）和測試網（Testnet）。主網是真正的金融交易發生的地方，如各種 DeFi 和 NFT 活動；所有在主網上的代幣，都是「能在市場上交易」的代幣。

    ▍Token（通證）

    • 若要較真，Token 翻譯成「通證」會比「代幣」來的理想些（現況是，後者是比較直觀、順口的稱呼）。

    • 通證是覆蓋比較大的超集合，除了轉帳，多了委託（delegate）、投票（Vote）等功能；代幣是覆蓋比較小的子集合，只有轉帳的概念。

    社群通證的金錢性固然比不上反映一般商品價值而且歷史悠久的法定貨幣，但反過來也擁有諸多過往無法想像的優勢，最明顯的是精準地反映和承載社群所重視的價值，把抽象的價值化為具體的行動清單（actionable items），讓體現核心價值不只流於遙不可及的願景，產出階段性成果，過程變得比較有趣 [1]。

    • 雖然 Token 項下可以做一些分類（例如 Protocol Tokens、Utility Tokens 等），但我們也要知道，一個 Token 往往同時具備若干屬性，沒有辦法只用一個標籤去定義它。

    　

    ▍Protocol Tokens

    ERC 全稱 Ethereum Request for Comments，是用來描述以太坊編程標準。EIP 全稱 Ethereum Improvement Proposals，與比特幣的 BIP 一樣的道理，目的是改進協議本身。

    • 例如市場上最知名的 ERC-20 Token，這是從 EIP-20 提案經過以太坊社區不斷討論驗證通過而來的。

    • 由以太坊創辦人 Vitalik Buterin 於 2015 年提出，這是一種基於以太坊的 Token 格式。開發人員可以利用現有的基礎來構建自己的代幣，而不是從零開始。

    • 只有按照這個格式開發的 Token，才能被各種以太坊錢包、地址、智能合約兼容。

    　

    ▍Utility Tokens

    • 去中心化應用程式（DApp）發行的代幣。

    • 其常見用途包括了：社區治理投票權、項目營收分潤權、作為獎勵發放等。

    　

    ▍NFT（Non-Fungible Tokens）和 FT（Fungible Tokens）

    • NFT 和許多加密貨幣一樣，是使用 區塊鏈技術 來記錄資料的一種形式，也是一個 Token。

    • 每個 Token 可以代表一個「非同質化」資產，例如圖像、藝術品、影音、體育、飲食、遊戲、甚至是一則社群貼文⋯⋯。

    • 總的來說，NFT 是一種專門紀錄非同質化資產的資料單位。

    「同質化」vs「非同質化」首先「同質化」是指資產可與同類型資產互換，或是可拆分的特性。舉例來說，比特幣、以太幣和其它加密貨幣，又或是你手邊的新台幣，都是一種同質化的資產，因為與同類型資產之間，價值相等且能夠切割成更小的單位。如果今天我拿 1 張千元大鈔，跟你的 10 張枚百元鈔票做替換，你會覺得無所謂，因為兩者的價值相同。而「非同質化」自然是指獨一無二，無法與同類型資產互換，拆分的資產。舉例來說，畢卡索的畫作，蘇富比的拍賣品，又或是不動產權狀，每件資產都與眾不同，你無法找到第二件一模一樣的資產。就算出自於同一人（畢卡索）之手，他的每一件作品都擁有不同的價值，彼此無法互相替換。要注意的是，「同質化」與「非同質化」的定義是可以改變的。如果一件同質化資產被賦予內在價值或獨有特徵，便可重新定義為一件獨一無二的非同質化資產。例如，我的千元大鈔擁有周杰倫的簽名，那它就變成非同質化資產，反之亦然。

    • 我們可以把 NFT 想像成原創者為數位資產發行的「身分證」或「防偽證書」，因為被放在區塊鏈上的緣故，我們能夠簡單驗證作品的出處、真偽、稀缺性。

    • 換句話說，當你在買賣 NFT 時，你買的其實不是作品本身，而是指引作品所在的數據，是原創者為作品發行的「身分證」或「防偽證書」。

    ▍這些代幣、通證帶來了什麼好處？

    • 價值發現

    • 落實 Web3 理念
      把數據還給個人（區塊鏈技術只是其中一種實現手段）。

    ▍穩定幣

    • 穩定幣（Stablecoin）是一種與標的（例如美元或黃金）掛鉤的加密貨幣。與一般加密貨幣相比，穩定幣的波動性較小，價格平穩，得名穩定幣。
