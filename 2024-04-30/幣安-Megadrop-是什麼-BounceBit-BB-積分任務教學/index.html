    幣安 Megadrop 是什麼？BounceBit（BB）積分任務教學 ——

    本文轉錄自區塊鏈科普網站《鏈習生 Chainee》幣安 Megadrop 是什麼？教你完成 BounceBit（BB）積分任務擼空投 一文。

    哇！不只 Launchpool，加密貨幣交易所龍頭幣安 Binance 能「擼羊毛」的地方又多一個！

    這篇文章要告訴你，幣安最近推出的 Megadrop 是什麼；以及手把手教你參與 Megadrop 平台首發的 BounceBit（BB）積分活動。讓我們馬上開始吧！

    幣安 Megadrop 是什麼？

    幣安 Megadrop 是一種全新「代幣發行平台」，結合空投活動的舉行，為用戶帶來全新體驗。

    同樣是和新幣發行有關，相較於幣安原有的 Launchpool／Launchpad，用戶在使用 Megadrop 時感受最不同的地方是，必須連接幣安 Web3 錢包。

    • 延伸閱讀：幣安 Web3 錢包一鍵創建，不再需要使用助記詞，自己持幣好輕鬆

    合理推測，幣安推出 Megadrop，應是為了刺激用戶的幣安 Web3 錢包採用率，充滿戰略性意義的一步棋。

    幣安 Megadrop 上線 BounceBit（BB）

    目前，幣安在 Megadrop 平台也已經上線首發項目 BounceBit（BB），這是一個專門為比特幣設計的「再質押」服務。

    跟據 幣安研究院 的介紹，BounceBit（BB）是一個專注於為比特幣資產賦能的創新區塊鏈平台，透過中心化金融（CeFi）和去中心化金融（DeFi）的融合，以及再質押（restaking）的策略，要將比特幣從一種被動資產，轉變為加密生態系統中的積極參與者。

    BounceBit 具備以下亮點：

    • BTC 再質押
      增加了資產的流動性和收益機會。

    • 雙幣 PoS 共識機制
      允許驗證者同時接受 BBTC 和 BB 代幣，者種機制稱強了網路的彈性和安全性，同時擴大了參與者基礎。

    • BounceClub
      不必具備程式設計基礎，用戶也能使用內建工具創建 DeFi 產品。

    • 流動託管
      BounceBit 引入流動託管概念，使抵押的資產保持流動性，並提供更多收益機會。

    幣安 Megadrop BounceBit（BB）積分任務怎麼參與？

    首先，你得在幣安交易所有一個完成 KYC 的帳戶，以及開通 幣安 Web3 錢包，這些是前置條件。

    還未註冊幣安交易所的朋友，使用「閱讀筆耕」專屬 推薦連結 註冊，或是手動輸入閱讀筆耕專屬推薦碼「L3BAGL6Y」，享有交易手續費終身 20% 返現優惠。

    活動期間為即日起至 5 月 13 日，詳見 幣安 BounceBit（BB）Megadrop 任務公告辦法，為大家梳理成友善的 SOP 操作步驟，讓你輕鬆完成這些積分任務。

    ≣ 積分是做什麼的？積分是怎麼計算？

    活動期間持續累積積分，用戶最終積分將獲得 Megadrop 獎勵。

    積分計算公式為 =（鎖倉 BNB 積分 x Web3 任務乘數）+ Web3 任務積分

    • 鎖倉 BNB 積分（基於 BNB 定期產品快照期間的每小時快照）=（120 天 BNB 平均鎖倉數量 x 130）+（90 天 BNB 平均鎖倉數量 x 120）+（60 天 BNB 平均鎖倉數量 x 110）+（30 天 BNB 平均鎖倉數量 x 100）

    • Web3 任務乘數：1.5

    • Web3 任務積分：1000

    如果以上描述看得頭疼也沒有關係，只要記得有「兩種」積分任務要做就可以了！

    第一種任務是在「幣安交易所」裡完成，第二種任務是在「幣安 Web3 錢包」裡完成。

    只要完成一種任務，就可以掌握一部分積分入袋（當然，建議你兩種任務都完成）。例如，如果你只有完成 Web3 任務，沒有做鎖倉 BNB 任務，屆時，你的積分是 0 x 1.5 + 1000 = 1000 積分。

     

    ≣ 積分任務一：幣安「交易所」購買 BNB 定期產品

    類似傳統金融中「定存」的概念，操作路徑是幣安 App 首頁 → 左上角幣安 logo → 找到 Megadrop 圖示 → 鎖倉 BNB。

    有各種期間（120天、90天、60天、30天）可以參與，鎖倉的金額愈大、天數愈長，所創造的積分效果也愈高。

    ≣ 積分任務二：幣安「Web3 錢包」裡體驗 BounceBit 質押服務

    如果你的幣安 Web3 錢包餘額是零，轉一些 BNB 代幣到幣安 Web3 錢包（大約等值 10U 就夠了，是為了下一動要 swap 和當作 gas fee 用）。

    在 Web3 錢包裡，切換到 BSC 鏈，把 BNB 代幣 swap 成 0.0001 BTCB 代幣（等值不到 10U）。

    從稍早展示過的 Megadrop 圖示進入 BounceBit 任務頁面「完成 Web3 任務」，連接幣安 Web3 錢包，並質押 0.0001 BTCB 代幣。

    最後，記得回到 BounceBit 任務頁面「完成 Web3 任務」按下「驗證」鈕才算數。

    結語

    隨著幣安推出全新的 Megadrop 平台，加密貨幣市場又添了一處新的「擼羊毛」機會。做為先發球員，參與比特幣再質押的 BounceBit（BB）積分活動，不僅是一個賺取額外收益的機會，也讓你能夠體驗最前沿的 Web3 技術。

    希望這篇手把手教學，能幫助你掌握如何參與活動並從中獲益。也別忘了投資前，記得進行適當的研究，以確保每一步都符合你的財務目標和風險承受能力。祝你玩得愉快！
