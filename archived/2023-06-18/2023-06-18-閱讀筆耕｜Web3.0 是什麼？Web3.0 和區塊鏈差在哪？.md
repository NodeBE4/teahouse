是什麼？Web3.0 和區塊鏈差在哪？</figcaption></figure><p>根據數據網站 statista 估計，全球約有 50 億人口透過網路與世界交流，但你有沒有想過，你在網路上留下的足跡、數據，那些屬於你的數位蹤跡本應是屬於你的資產，在現行的網路體制卻並不屬於用戶，而是被大型網路服務商、社群媒體企業所壟斷，<strong>換句話說，整個網路生態正被極少數的科技巨頭所掌握。</strong></p><p>為了改變這個困境，Web3.0 作為一個新世代的網路概念被提出，許多開發者紛紛加入 Web3.0 這個新興產業。究竟 Web3.0 是什麼？Web3.0 概念和我們現在的 Web2.0 網路有什麼差異？Web3.0 應用又有哪些呢？本文將一一介紹。</p><hr><hr><h2><strong>Web3.0 是什麼？和區塊鏈差在哪？</strong></h2><h3><strong>≣ Tim Berners-Lee：語意網是我心中的 Web3.0</strong></h3><p>全球資訊網（WWW）之父 Tim Berners-Lee 是早期 Web 3.0 的倡議者之一。他在 1998 年首次提出「語意網」（Semantic Web）的概念，並在 2005 年的一次採訪中表示，語意網就是他心中的 Web 3.0。語意網的理念是讓電腦以模擬人類的方式處理內容，使所有的數據能夠根據上下文和概念進行連接和理解。</p><p>在推廣語意網時，Berners-Lee 也一直強調一個思想，即網路應該是「去中心化」的，任何人無需中央機構的許可就能在網路上發布內容，不應受到無謂的審查和監視。<br class="smart"><br class="smart"></p><h3><strong>≣ Gavin Wood：Web3.0 是向 Web2.0 發起一場革命</strong></h3><p>2014年，以太坊的共同創辦人Gavin Wood 重塑了 Web 3.0 的概念。他認為應該有一種低門檻、不受審查和壟斷的基礎網路傳輸協議，取代傳統的 Web 技術（如 HTTP、AJAX 和 MySQL），以強大而可驗證的方式保護網路使用者的數據。</p><p><strong>他表示，Web 3.0 的出現是為了應對 Web 2.0 這種「把用戶的個資當成石油一樣開採和利用」的現象。</strong>巨頭公司擁有大量的用戶數據，一方面用這些數據訓練人工智慧（AI）並為用戶提供便利工具，另一方面則利用數據交換廣告營收。更嚴重的是，這些巨頭過於「中心化」，擁有審查和控制權力。</p><p>Web3.0 希望打破現有的「中心化」架構，也就是將網路服務的控制權分散，避免少數企業壟斷網路生態的現狀，創造一個更開放、民主的網路環境。<br class="smart"><br class="smart"></p><h3><strong>≣ Web3.0 與區塊鏈技術一拍即合</strong></h3><p>無論如何，Tim Berners-Lee 和 Gavin Wood 都不約而同地表達 Web3.0 必須是「去中心化」的思想，要打破現有的「中心化」架構，達到網路服務的控制權分散；這個願景與區塊鏈的種種特性（去中心化、不可篡改、公開透明、匿名、共識機制、無需許可）正好一拍即合。</p><p>延伸閱讀：<a target="_blank" rel="noopener noreferrer nofollow" href="https://chainee.io/what-is-blockchain/">區塊鏈是什麼？一文搞懂區塊鏈的九大必備知識</a></p><p>如果我們用比較廣義的觀點來看待，Web 3.0 泛指建構在區塊鏈、分散式網路以及相關技術上的網路服務，這種新興的網路生態呈現出更開放、協作和民主化的特性；人們透過創新的應用方式將 Web 3.0 的精神融入到生活的各種領域，例如 NFT、DeFi、GameFi、DAO 和 DID，這些沿革重新定義了我們對於數位資產、金融服務、遊戲、組織治理和身份驗證的理解。</p><p>總的來說，Web 3.0 描繪的是一種理念和哲學，建立一個讓每個人都擁有自己數位資產的網路體系，而區塊鏈則是實現這一理念的關鍵技術革新。</p><hr><hr><h2><strong>為什麼我們需要 Web3.0？</strong></h2><h3><strong>≣ 少數巨頭的壟斷剝削</strong></h3><p>Web2.0 用戶的數據由中心化實體構管控，為其帶來巨大利益，但用戶卻無法從中獲得任何價值。</p><p>例如 Facebook、Instagram 和 Twitter 等社交媒體巨頭，累積了大量用戶生成的數據，包括個人資訊、社交關係和行為偏好等。這些數據可用於廣告精準投放和個人化推薦，幫助這些平台獲得廣告收入，但用戶卻無法分享到這些利潤。</p><p>而 Web3.0 提倡將網路服務的控制權分散，<strong>避免少數企業壟斷網路生態的現狀</strong>。<br class="smart"><br class="smart"></p><h3><strong>≣ 資訊不透明造成效率低落</strong></h3><p>Web3.0 致力於解決中心化環境下資訊不透明所帶來的效率問題，以下面的例子來做說明：</p><p>例如美國零售業巨頭沃爾瑪 Walmart 指出，在沒有引進 Web3.0 區塊鏈技術的情況下，追蹤食物來源通常需要一周的時間，但運用區塊鏈技術後只需，花上 2.2 秒即可快速追查，大大改善了時間效率。這不僅降低了食品安全疑慮和食材浪費，還能降低企業成本。<br class="smart"><br class="smart"></p><h3><strong>≣ 單點失效隱憂</strong></h3><p>在 Web2.0 環境下，若中心化實體發生故障或遭受攻擊，整個系統的數據完整性和安全性將受到威脅，這種情況被稱為「單點失效」；而在 Web3.0 環境下，消除了中心化管理的需求，使「單點失效」問題可以得到解決。即使某些節點出現故障或遭受攻擊，其他節點仍然可以繼續運行，保障數據的安全性、完整性和可用性。<br class="smart"><br class="smart"></p><h3><strong>≣ 管理自主權、資產所有權缺乏</strong></h3><p>回顧 2008年，中本聰（Satoshi Nakamoto）在發明 <a target="_blank" rel="noopener noreferrer nofollow" href="https://chainee.io/what-is-bitcoin/">比特幣</a> 時描繪了一個願景，<strong>即每個人自己持有自己的資產，而不是把資產放在第三方中心化機構裡被託管</strong>。這也是 Web3.0 的信念之一。</p><p>在 Web3.0 的世界中，用戶使用自託管 <a target="_blank" rel="noopener noreferrer nofollow" href="https://chainee.io/what-is-cryptocurrency-wallet/">加密貨幣錢包</a> 來管理他們的加密貨幣投資組合。由於私鑰儲存在用戶的設備上，並由用戶自行保管，讓我們貨真價實地掌握了自己的錢包，從而實現了用戶真正的管理自主權／資產所有權。<br class="smart"><br class="smart"></p><h3><strong>≣ 隱私權缺乏</strong></h3><p>Web2.0 對於用戶隱私的保護相當有限。當你使用金融卡線上支付產品或服務時，你必須提供個人資訊。此外，對於企業如何儲存這些資訊以及是否與第三方分享，你也完全無法控制。</p><p><strong>許多用戶擔憂這些中心化機構如何處理他們的數據，甚至侵犯到他們的隱私</strong>。這正是 Web3.0 能幫助我們改善的問題之一。</p><p>與 Web2.0 不同，Web3.0 平台不需要用戶提供個人資訊。你只需使用自託管 <a target="_blank" rel="noopener noreferrer nofollow" href="https://chainee.io/what-is-cryptocurrency-wallet/">加密貨幣錢包</a> 來訪問 Web3.0 的去中心化應用程式（dApp）。無需註冊，也沒有人要求你提供姓名、地址或其他個資。你的加密貨幣錢包即是標識你作為 Web3.0 獨特用戶的方式。</p><hr><hr><h2><strong>從 Web1.0 到 Web3.0 發展史</strong></h2><p>We3.0 作為網路發展的第三階段，要了解它究竟為網路帶來什麼樣的革新，勢必要先簡單回顧一下網際網路的歷史。</p><h3><strong>≣ Web1.0：Read</strong></h3><p>泛指 1999 年之前的網路環境，當時人們只能閱讀網頁內的資訊，無法與網頁進行互動。用戶僅能查看與簡單的文字留言。<a target="_blank" rel="noopener noreferrer nofollow" href="https://www.cameronsworld.net/">Cameron’s World</a> 收藏了許多早期 Web1 網頁的樣式，不但用戶介面較為陽春，更重要的是，網頁本身並不提供用戶反饋、互動的管道。</p><h3><strong>≣ Web2.0：Read, Write</strong></h3><p>指現今的網路環境，用戶可以在科技巨頭打造的平台上創作個人內容，與全世界的網民互動。<strong>不過這些資訊皆儲存在「中心化」的資料庫裡，企業擁有更改、刪除用戶內容的權利。</strong>比如我們可以在 FaceBook 上發布貼文，評論他人的內容，並與平台用戶即時交流，但同時受到平台審查、管控。</p><h3><strong>≣ Web3.0：Read, Write, Own</strong></h3><p>Web3.0 除了能夠 Read, Write 之外，更強調用戶對於數位資料的「擁有權」。</p><p>在傳統的 Web2.0 中，用戶通常將數位資料儲在「中心化」的伺服器上，導致用戶失去了對自己數據的控制權；而 Web3.0 使用區塊鏈技術和智能合約，使得用戶能夠將數據儲存在「去中心化」的環境中，並透過私鑰進行控管，這意味著用戶可以真正「擁有」自己的數位資料。</p><p>在此 Web3.0 基礎上，用戶可以利用智能合約發行自己的數位作品，或是向其他人買賣數位作品。<strong>Web3.0 為用戶創造了全新的創作、擁有和交易數位資產的機會</strong>。</p><figure 概念不同之處</figcaption></figure><hr><hr><h2><strong>Web3.0 帶來的社會變革</strong></h2><p>透過區塊鏈技術驅動去中心化的願景，Web3.0 可以實現許多在 Web2.0 世界裡無法做到的事情，以下將舉一些社會上 Web3 應用的範例，讓大家了解 Web3 概念在許多開發者的努力下，正在逐漸改善我們的生活。</p><h3><strong>≣ 跨國轉帳業務</strong></h3><ul><li><p>Web2.0<br class="smart">在傳統銀行用法定貨幣跨國轉帳，首次匯款需平日至臨櫃辦理手續，轉帳過程需經較昂貴且緩慢的 SWIFT 系統和人工審核，整體流程需 1～2 工作天。</p></li><li><p>Web3.0<br 和萬事達卡（Mastercard）也都積極 <a target="_blank" rel="noopener noreferrer nofollow" href="https://www.blocktempo.com/visa-crypto-strategy-remains-intact/">布局</a> 在公共區塊鏈上運行（由法幣支持的）加密貨幣支付。</p></li></ul><figure 差異範例</figcaption></figure><h3><strong>≣ 金融借貸服務</strong></h3><ul><li><p>Web2.0<br 差異範例</figcaption></figure><h3><strong>≣ 數位創作領域</strong></h3><ul><li><p>Web2.0<br class="smart">當我們將數位創作放到網路上時，原作者難以主張版權所有，容易被別人「複製貼上」、「抄襲」。</p></li><li><p>Web3.0<br class="smart">我們可以為數位創作發行 技術找到了解答，這讓許多藝術品收藏家有了令人眼睛為之一亮的新賽道；隨著疫情推動人們日常生活朝向數位化的影響，使用 NFT 技術驅動的數位藝術創作，未來勢必蔚為主流。<br class="smart"><br class="smart">以文字創作為例，華人部落格平台 Matters 和方格子（vocus）也開始探索 Web3.0「創作者經濟」的新嘗試，與 IP、藝術家、創作者接洽，協助打造其專屬 NFT，為既有商業模式開創全新面貌。</p></li></ul><hr><hr><h2><strong>Web3.0 面臨的五大挑戰</strong></h2><p>Web3.0 概念及 Web3.0 應用依然處於早期階段，在社群與項目方共同探索 Web3.0 各種可能性之後，目前 Web3.0 距離真正願景的實現，依然面臨著以下挑戰：</p><h3><strong>≣ 進入門檻</strong></h3><p>Web3.0 作為一個新興的概念，使用者體驗目前仍不夠友善和成熟。使用 Web3.0 應用之前，需要具備基礎的區塊鏈、加密貨幣相關知識，例如 <a target="_blank" rel="noopener noreferrer nofollow" href="https://chainee.io/what-is-cryptocurrency-wallet/">創建加密貨幣錢包</a>、<a target="_blank" rel="noopener noreferrer nofollow" href="https://chainee.io/how-to-buy-cryptocurrency/">購買加密貨幣</a> 等。這些門檻造造成了 Web3.0 普及和推動上的困難。</p><p>目前許多 Web3.0 相關開發團隊（如：Reddit、LINE），正透過簡化使用者介面，降低用戶使用 Web3.0 應用的門檻。</p><h3><strong>≣ 缺乏殺手級應用</strong></h3><p>呼應上一段提到的進入門檻，Web3.0 目前還缺乏一個能夠被廣泛接納的「殺手級應用」場景，或許這將是最關鍵的挑戰。</p><p>如果以人工智能（AI）的發展來做對照，長年以來，AI 對於大多數人來說仍然是一個相對抽象的概念，難以直觀地理解和使用。直到最近，隨著聊天機器人 ChatGPT 的出現，我們見證了一個「殺手級應用」的誕生。</p><p>同樣地，Web3.0 也需要一個類似的「殺手級應用」。目前最值得期待的殺手級應用是各大科技巨頭正在努力的「元宇宙」領域。元宇宙是一個虛擬的平行世界，允許人們在其中建立自己的虛擬身份、探索虛擬世界、參與社交互動，甚至進行虛擬商業活動等。</p><p>總而言之，將 Web3.0 概念具體轉化為現實生活中的實際體驗，讓一般用戶能夠從中受益並感受到其優勢。</p><h3><strong>≣ 去中心化喊假的</strong></h3><p>知名區塊鏈以太坊創辦人 Vitalik 曾說過：「真正 100% 的『去中心化』只是一盤散沙，去中心化與中心化之間的關係，並不是非黑即白。」</p><p>Vitalik 這番話的意思是說，儘管大多數的 Web3.0 項目宣稱追求去中心化，但實際上，我們必須承認許多項目仍然受到少數人的控制。這種現象可能源自於技術限制（過度依賴中心化的基礎設施或服務提供商）或是治理模型的不完善（治理社區中的巨鯨成員，把持發展和營運的主導地位）。</p><h3><strong>≣ 難以兼顧效率性</strong></h3><p>如果一條區塊鏈去中心化程度極高，開發團隊每一項決策都需經過社群投票，勢必會造成開發效率低下，甚至做出錯誤決策；但若開發團隊一意孤行，違背社群意願，該區塊鏈就不符合 Web3.0 的願景。</p><p>去中心化程度與效率性（決策和協調）之間要如何達到良好的平衡，一直是各大區塊鏈與項目方的難題。</p><h3><strong>≣ 政府監管</strong></h3><p>Web3.0 的願景是「去中心化」，不受第三方機構（如企業、政府）所控制，但這同時也代表並沒有第三方能夠介入 Web3.0 並保護用戶權益，像是當加密貨幣被駭客盜取時，我們難以透過法律管道來取回資產。</p><p>與此同時，因 Web3.0 產業高速發展，各國政府其實也正在積極地制定監管事宜，以保護用戶與國家利益，但此舉又會影響到 Web3.0 去中心化的願景，因此 Web3.0 該如何面對政府的監管，也是目前所面臨的挑戰之一。</p><hr><hr><h2><strong>結語：Web3.0 的未來是什麼？</strong></h2><p>許多科技趨勢研究報告顯示，<strong>Web 3.0 是未來的發展趨勢</strong>。根據 <a target="_blank" rel="noopener noreferrer nofollow" href="https://www.globenewswire.com/news-release/2019/11/21/1950706/0/en/Blockchain-Market-size-to-expand-at-a-massive-CAGR-of-over-69-from-2019-to-2025-Global-Market-Insights-Inc.html">環球電訊社</a>（GlobeNewswire）預估，直至 2025 年，Web3.0 產業將以驚人的近 70% 年複合成長率（CAGR）速度持續擴大，區塊鏈技術帶來了廣泛的功能。</p><p>與此同時，<strong>Web3.0 的「去中心化」願景將對我們的生活產生深遠的影響</strong>，Web3.0 正在對數位資產管理、金融服務、遊戲、組織形式、身份驗證等領域帶來全新的應用。</p><p>不過，Web3.0 目前面臨著一些重要挑戰，其中最關鍵的就是需要「殺手級應用」問世，讓一般用戶能夠從現實生活中實際體驗 Web3.0 帶來的效益。目前，許多人尚未瞭解 Web3.0 的好處，<a target="_blank" rel="noopener noreferrer nofollow" href="https://triple-a.io/crypto-ownership-data/">Triple A</a> 市場報告指出全球使用加密貨幣的用戶只有 3 億人。</p><p>Web3.0 的未來充滿了潛力和可能性，但這些願景是否能夠實現，還待時間考證。而我們須<strong>以開放和包容的態度看待 Web3.0</strong>，理解它不僅僅是一個技術的演進，而是一個能夠重塑我們社會運作方式的革命性改變。</p><hr><hr><figure class="image"><img
