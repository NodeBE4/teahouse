<p>每個月的第三個禮拜天上午，是 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E7%96%AB%E6%83%85%E4%B8%8B%E7%9A%84%E5%A4%A9%E7%A9%BA-%E9%81%A0%E8%B7%9D%E6%9C%83%E8%AD%B0-bafyreihgwwi33f5smq5k6qtmqp7kmnn63i5kk673aylijqanldhbwkqk3m" rel="noopener noreferrer" target="_blank">線上口語表達讀書會</a> 的日子，這回輪到我作為（其中一名）主講人，而主持人之前拋出的關鍵字是「<strong>投資</strong>」⋯⋯。</p><hr><hr><h2><strong>■ 我的準備方向</strong></h2><p><strong>✅ 食材挑選</strong></p><p>我設定的主題朝著「<strong>是否納入數位資產，為資產配置的一環</strong>」開展。</p><p>起心動念可以追朔至年初，聽聞方舟投資（ARK Invest）的一份研究報告，建議數位資產（比特幣）的配比應在 <a href="https://finance.ettoday.net/news/1908847" rel="noopener noreferrer" target="_blank">2.55% ～ 6.55％</a> 之間，科技女股神金口一出，便把這件事放在心上。</p><p>適逢本月初，進行一系列 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E7%82%BA%E9%A6%AC%E7%89%B9%E5%B8%82-nft-%E8%A8%88%E7%95%AB%E5%81%9A%E6%BA%96%E5%82%99-01-%E9%80%99%E6%98%AF%E5%95%A5-bafyreibeeboww4h45g36p3cgizyxtgjprkno6dn6oxwxpj4e7iv2q7pbru" rel="noopener noreferrer" target="_blank">馬特市 NFT 大基建</a> 的研究，最至又補完 <a href="https://www.youtube.com/watch?v=cDk1FPoCfqI" rel="noopener noreferrer" target="_blank">老高</a> 的影片，覺得他的講述很貼切易懂：<strong>在虛擬世界（元宇宙）若要消費，得靠加密貨幣。而那些「虛擬物品」的歸屬權得靠 NFT 去界定，有歸屬權才會有人花錢去買嘛</strong>！</p><figure class="image"><img src="https://assets.matters.news/embed/d9258bcb-fb5b-40e8-b730-7631e79b49f8.png" data-asset-id="d9258bcb-fb5b-40e8-b730-7631e79b49f8" referrerpolicy="no-referrer"><figcaption><span>電影《一級玩家》描繪出未來元宇宙的概念</span></figcaption></figure><p>登愣，頃刻間豁然開朗，把元宇宙、加密貨幣、NFT、區塊鏈⋯⋯都大串連在一起，找到了分享用的素材。</p><blockquote>⚠️ 食材挑選：專有名詞的大串連。<br class="smart"></blockquote><p>⋯</p><p><strong>✅ 烹調手法</strong></p><p>先前拜讀《<a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E6%9C%80%E9%AB%98%E5%AD%B8%E7%BF%92%E6%B3%95-%E8%81%B2%E9%9F%B3%E7%9A%84%E7%93%B6%E9%A0%B8%E6%95%88%E6%87%89%E8%88%87%E9%9A%A8%E6%A9%9F%E5%85%B1%E6%8C%AF-bafyreig7opxofsr7txzrsh7gujwgrh42phcdhr7i4sci2zgh4s6vezlhwq" rel="noopener noreferrer" target="_blank">最高學習法</a>》，得知聲音之間有 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E6%9C%80%E9%AB%98%E5%AD%B8%E7%BF%92%E6%B3%95-%E8%81%B2%E9%9F%B3%E7%9A%84%E7%93%B6%E9%A0%B8%E6%95%88%E6%87%89%E8%88%87%E9%9A%A8%E6%A9%9F%E5%85%B1%E6%8C%AF-bafyreig7opxofsr7txzrsh7gujwgrh42phcdhr7i4sci2zgh4s6vezlhwq" rel="noopener noreferrer" target="_blank">瓶頸效應</a>，閱讀也算數。</p><pre class="ql-syntax" spellcheck="false">如果你要進行簡報，那麼投影片不宜使用太多文字，推薦的做法是少量的關鍵字，因為聽眾毋須在心裡默念，不必頻繁切換（布洛卡／威尼克）網路開關，減少注意力（在兩種聲音之間）轉換而產生的摩擦（錯失掉部分訊息）。</pre><p>所以這次我刻意嘗試，讓簡報<strong>只留下關鍵字和圖片</strong>，<strong>避免出現句子</strong>。聽者不用（在心裡）誦讀，可以專注地接收講者的聲音 ⇩。</p><figure class="image"><img src="https://assets.matters.news/embed/dfad9c38-08de-42fd-9381-f14953e23e81.png" data-asset-id="dfad9c38-08de-42fd-9381-f14953e23e81" referrerpolicy="no-referrer"><figcaption><span>只留關鍵字和圖片</span></figcaption></figure><blockquote>⚠️ 食材挑選：只留關鍵字和圖片。</blockquote><hr><hr><h2><strong>■ </strong>一兼多顧，摸蜊仔兼洗褲</h2><p>其實有點取巧，沒有做投影片。</p><p>我是在前一天，利用 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E5%A6%82%E4%BD%95%E7%B9%AA%E8%A3%BD%E5%BF%83%E6%99%BA%E5%9C%96-%E6%96%87%E6%9C%AB%E5%BD%A9%E8%9B%8B-bafyreiackdwlh337h4icb53zen7naocts24xfckfcwdgxgn2lmx7nbfu5y" rel="noopener noreferrer" target="_blank">心智圖</a>（思維導圖）軟體，在腦海中順過一遍，過程中用到哪些字或圖，就把它們添加進去。由於「節點」可以收放，全部聚攏在一起就很小巧（一個手機屏幕的尺寸），<strong>變成我的數位提示手卡</strong>。</p><p>遠距會議的好處是可以共享螢幕，屆時只要操作自己的畫面，<strong>先讓人見樹</strong>，先對整個框架有有個初步掌握，<strong>再逐一見林</strong>，再依序「開枝散葉」說到哪就展到哪，目的是不讓台下目光跑偏，<strong>這也能在投影片裡的動畫效果來達成，我省下這道功夫</strong>。</p><p>注意用上心智圖軟體，<strong>從發想、製作、順過、呈現</strong>，可謂是一兼多顧，摸蜊仔兼洗褲。</p><hr><hr><p>謝謝各位看到這邊，隨文附上我的〔<strong>讚賞公民
