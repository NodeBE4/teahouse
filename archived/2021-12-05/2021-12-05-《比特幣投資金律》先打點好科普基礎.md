<p>作者：黃文逸（腦哥）<br class="smart">出版社：一心文化</p><figure class="image"><img src="https://assets.matters.news/embed/2de6b5d7-653e-4322-9c00-5985a5aef5c9.jpeg" data-asset-id="2de6b5d7-653e-4322-9c00-5985a5aef5c9" referrerpolicy="no-referrer"><figcaption><span>《比特幣投資金律：加密貨幣實戰教學與進階獲利法》</span></figcaption></figure><p>《<a href="http://moo.im/a/1ahMOQ" rel="noopener noreferrer" target="_blank">比特幣投資金律</a>》是 YouTube 頻道 <a href="https://www.youtube.com/channel/UC1EG67JfgUKaZ4Z_l4euycw/featured" rel="noopener noreferrer" target="_blank">腦哥 Chill 塊鏈</a> 主持人黃文逸的著作。這書在博客來開賣時，推出 500 本限量「幣安幣（BNB）贈品」版，依稀記得當天凌晨，成功把書加進購物車，但結帳時手速不夠快，失之向隅，可惜。</p><hr><hr><h2><strong>■ 這本書在說什麼？</strong></h2><p>雖書名曰比特幣，卻只佔大約四分之一篇幅。</p><p>就我理解，是以【<strong><u>① 比特幣</u></strong>】的誕生、特性、發展⋯⋯做為引子，先鑑古；後知今，帶到【<strong><u>② 加密貨幣</u></strong>】實戰教學，從最基礎的交易所介紹、開戶、出入金、買賣、幣種介紹⋯⋯至比較進階的【<strong><u>④ 投資策略</u></strong>】，也千叮萬囑，在學習如何獲利以前，必須先建立起【<strong><u>③ 風險意識</u></strong>】才行。</p><hr><hr><h2><strong>■ 為何比特幣是數位黃金？</strong></h2><p>不是第一次聽到比特幣是「數位黃金」之說，但第一次聽到完整的論述，從書中。</p><p>原來比特幣在創始之初，就設定好了發行總量（上限 2100 萬顆）與規則，產出量大約每四年減半一次，越挖越少，挖完就沒了，足見其<strong>稀缺性</strong>；再者，它是數位的——<strong>不易（不會）衰敗</strong>——存在於區塊鏈（Blockchain）上的一串數字代碼，不肖人士無法造假，也沒有央行印真，<strong>沒人有發行權</strong>，是去中心化（decentralization）的；最後，只要能受到<strong>多數人認可</strong>，比特幣便具有類似黃金「價值儲存」的功能。</p><figure class="image"><img src="https://assets.matters.news/embed/9304b044-af81-4cbf-a748-cf19490b0519.jpeg" data-asset-id="9304b044-af81-4cbf-a748-cf19490b0519" referrerpolicy="no-referrer"><figcaption><span>最後一顆比特幣預計在 2140 年被挖出｜書中圖表</span></figcaption></figure><blockquote>⚠️ 黃金（比特幣）四大屬性：<br class="smart">1. 稀缺性。<br class="smart">2. 不易衰敗。<br class="smart">3. 沒人有發行權。<br class="smart">4. 多數人認可。</blockquote><p>對於「多數人認可」這點深有感觸，憶起《<a href="http://moo.im/a/9fpFLS" rel="noopener noreferrer" target="_blank">人類大歷史</a>》中提到「<strong>想象的現實</strong>」概念，指的是某件事人人都相信，而且只要這項共同的信念仍然存在，力量就足以影響世界。</p><p>至今，我們基於政府（中心化）的背書，相信「一張紙」是有價值的；對照看看，你願不願意信任演算法及程式碼（去中心化）的機制，相信「一串數字代碼」是有價值的。<br class="smart"></p><hr><hr><h2><strong>■ 入市實戰：知風險，有策略</strong></h2><p>之前「習而時學之」已經有相關操作經驗，故實戰篇的閱讀速度飛快，很能進入狀況。順勢釐清一些概念，列出幾則印象深刻的：</p><h2><strong>✅ 託管 vs 自主</strong></h2><p>你在加密貨幣交易所註冊的帳戶，並沒有拿到「私鑰」，所以<strong>資金其實是「借放」在交易所的錢包</strong>，這與在物理世界買賣證券（的交割戶）是同樣概念；會冒著一定風險，輕則遇上暫時性凍結，重則平台倒閉或管理人捲私鑰逃跑，你的資產便會消失。</p><p>比較穩妥的作法是把持有的幣，存放在私人「完整的」錢包，有私鑰，<strong>保有更大自主權，也能對接區塊鏈上的其它應用</strong>；我也陸續把自己的 LikeCoin，慢慢從 <a href="https://liker.land/getapp" rel="noopener noreferrer" target="_blank">Liker Land</a> 分批轉到 <a href="https://www.keplr.app/" rel="noopener noreferrer" target="_blank">Keplr</a> 錢包，除了 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E9%87%8D%E6%96%B0%E8%AA%8D%E8%AD%98-like-coin-%E7%9A%84%E5%B6%84%E6%96%B0%E6%87%89%E7%94%A8-%E9%99%84%E6%95%99%E5%AD%B8-bafyreibvzxilf5zxwt6unn663dlfpuqntnxp4jczfgc7btumboqvmq3qke" rel="noopener noreferrer" target="_blank">對議案投票和註冊 ISCN</a> 之外，最吸引人的誘因莫過於 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E8%B2%93%E8%B2%93%E5%B7%A5%E4%BD%9C%E5%9D%8A%E9%9A%A8%E7%AD%86-%E8%AA%BF%E6%9F%A5%E5%85%B5%E5%9C%98%E5%9C%A8-osmosis-%E5%8D%9A%E5%A3%AB%E5%AE%B6%E7%9A%84%E5%9C%B0%E4%B8%8B%E5%AE%A4%E7%99%BC%E7%8F%BE-bafyreiewyar7botfhxrxyraklzjj6i6twadbti47nbglryzagag4seixgq" rel="noopener noreferrer" target="_blank">流動性挖礦</a> 了吧！</p><p>⋯</p><h2><strong>✅ 幣圈詐騙</strong></h2><p>我們常有 FOMO（Fear of Missing Out）的心理，害怕錯過千載難逢的「上車」機會，一聽到【<u>❶ 名額、時間有限</u>】就奮不顧身地進場；再加上幣圈的預期報酬，普遍比傳統金融的行情高，一種【<u>❷ </u><strong><u>快速獲利</u></strong>】的印象容易讓人放鬆戒心，供養成為詐騙溫床。</p><p>例如，對方只會拿比特幣出來【<u>❸ 吹噓</u>】，卻【<u>❹ 講不清楚背後運作機制</u>】，很可能他們的獲利就是來自於你的本金；最後一點，也是最中肯的，那就是【<strong><u>❺ 異常熱心的客服或推廣者</u></strong>】，除非這個人本來就是你現實中的朋友，否則一般陌生人誰有空跟你私聊這麼多，很可能別有居心，要把你的本整碗端走。</p><p>⋯</p><h2><strong>✅ 量化交易</strong></h2><p>這是書中介紹的 7 種投資策略其中之一。</p><p>事先指定好策略（相當多元），透過機器人（不用休息）自動化執行的交易方式，適合紀律欠佳（避免人性誤區）或沒時間自己操盤的投資人（好好吃飯、好好生活）。</p><p>為了實際體驗，我也在 <a href="https://www.pionex.com/zh-TW/sign/ref/fag3HmhO" rel="noopener noreferrer" target="_blank">Pionex 派網</a> 交易所註冊，這裡的 USDT 蠻便宜，用一樣多的法幣入金，換得較高的穩定幣數額。</p><figure class="image"><img src="https://assets.matters.news/embed/41194889-0ddf-4b30-bd0f-0579e11513c8.png" data-asset-id="41194889-0ddf-4b30-bd0f-0579e11513c8" referrerpolicy="no-referrer"><figcaption><span>Pionex 派網</span></figcaption></figure><p>想配置一些資金試試「<strong>網格交易</strong>」，其邏輯簡單，但操作繁瑣。原理同字面，把幣價切成一格一格，每當目標貨幣上漲觸及一格，機器人就幫你出清一些（分批獲利賣出），反之，下跌觸及一格，機器人就幫你進貨一些（分批進場抄底），是一種適合用在「<strong>上下震盪盤整格局</strong>」的交易策略，反覆刷取價差收益。</p><hr><hr><h2><strong>■ 尾聲：先打點好科普基礎</strong></h2><p>這是我讀過的第三本與加密貨幣有關的書，感覺像是參加教學工作坊，上了一場有系統的線上課程，淺顯易懂。閱畢，可以對幣圈建立起一個「<strong>通盤且全面</strong>」的大框架。</p><p>以時序而言，我推薦先看《<a href="http://moo.im/a/1ahMOQ" rel="noopener noreferrer" target="_blank">比特幣投資金律</a>》，畢竟比特幣是一切的淵源，也先打點好科普基礎；書中第二章談到的 16 種主流加密貨幣，你會在《<a href="https://matters.news/@penfarming/%E4%BB%A5%E5%A4%AA%E5%A5%87%E8%A5%B2-%E5%B0%8E%E8%AE%80-%E4%BB%A5%E5%A4%AA%E7%82%BA%E9%91%91-%E5%8F%AF%E4%BB%A5%E7%9F%A5%E8%88%88%E6%9B%BF-bafyreibe5e2rhzdmtex5xeqc4g4hlfdwjxli74gtqknoqahngr4em46sca" rel="noopener noreferrer" target="_blank">以太奇襲</a>》中倍感親切，咦！怎麼好多都跟以太坊創辦團隊扯上邊；最後再看《<a href="https://www.books.com.tw/products/0010867255" rel="noopener noreferrer" target="_blank">區塊鏈社會學</a>》，那是一種更上位，超脫金錢，馬斯洛 <a href="https://zh.wikipedia.org/zh-tw/%E9%9C%80%E6%B1%82%E5%B1%82%E6%AC%A1%E7%90%86%E8%AE%BA" rel="noopener noreferrer" target="_blank">需求層次理論</a> 中自我實現的追求。</p><hr><hr><p>謝謝各位看到這邊，隨文附上我的〔<strong>讚賞公民 2.0</strong>〕〔<strong>圍爐</strong>〕〔<strong>其它平台</strong>〕鏈結，歡迎來逛逛 ❤️。</p><p>🌱 第三季社區活動進行中 <a
