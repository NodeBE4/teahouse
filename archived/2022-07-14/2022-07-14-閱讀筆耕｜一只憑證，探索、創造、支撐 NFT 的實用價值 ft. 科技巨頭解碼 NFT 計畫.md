<div>
<p>以前上經濟學，最常出現的概念之一是「<a href="https://zh.wikipedia.org/zh-tw/%E6%95%88%E7%94%A8" rel="noopener noreferrer" target="_blank">效用</a>」（Utility）。課堂上，還記得教授再三強調，效用也可以來自於精神層面——無法被量化——例如快樂（Happiness）。</p><p>奇了，怎麼進到 Web3 世界，很多人改口說「賦能」（Utility），特別指的是 NFT 的實用性。剛好藉由這次 <a href="https://matters.news/@crypton3/304249-nft-%E8%A8%8E%E8%AB%96%E5%8D%80-%E6%B4%BB%E5%8B%95-%E4%B8%80%E8%B5%B7%E6%8E%A2%E7%B4%A2-nft-%E6%9C%89%E5%93%AA%E4%BA%9B%E5%AF%A6%E7%94%A8%E5%83%B9%E5%80%BC-bafyreig54vtz7hjgdmab7sv5ko2wzxgp6nf4cgwepcjpzpnpzqlmnl2enu" rel="noopener noreferrer" target="_blank">徵文</a> 活動，來聊聊「實用價值」這碼事。</p><hr><hr><h2><strong>■ 一只憑證</strong></h2><p>基於 NFT 的「獨一無二、不可篡改」性質，我們可以把那些想像得到，項目方所端出的各種牛肉收束為兩個字——<strong>憑證。</strong></p><pre class="ql-syntax" spellcheck="false">一張兌換券，是憑證。一張門票，是憑證。一張選票，是憑證。一紙證書，是憑證。一紙合約，是憑證。表彰會員資格，是憑證。表彰社群身份組，是憑證。表彰利潤分配，是憑證。</pre><p>憑證用以<strong>確權</strong>，確保每名 NFT 持有者的權利，姑且稱之為「<strong>受益權</strong>」，為了呼應「實用價值」的討論。</p><blockquote>憑證 → 確權 → 受益歸屬。</blockquote><p>順著這邏輯，<strong><u>各種能套用上述動線，打造出來的商業模式，都可謂是 NFT 實用價值的發想空間，天馬行空</u></strong>；通常，項目方募得一筆資金，做為未來向出資者「兌現承諾」的基底，實現那些 <a href="https://www.youtube.com/watch?v=mLFhTFiX0uM" rel="noopener noreferrer" target="_blank">說好的幸福</a>（說好的路線圖）。</p><p>問題來了，<a href="https://udn.com/news/story/7088/6419491" rel="noopener noreferrer" target="_blank">雖然我不是數學家</a>，但資金總有燒完的一天，對吧！</p><p>所以除了探索 NFT 有哪些實用價值，亦看重項目方能不能幫 NFT 找到一個「實打實」的支撐，而非「後金補前金」淪為龐氏經濟。<br class="smart"></p><hr><hr><h2><strong>■ 探索、創造、支撐實用價值</strong></h2><p>最近很關注的項目是「<a href="https://vocus.cc/vocuverse/collections/TechGiantsDecode" rel="noopener noreferrer" target="_blank">科技巨頭解碼 NFT</a>」計畫，它是一份將近萬人訂閱的「主程式」付費專欄，所衍生出來的獨立「資料片」概念。以前，是由主筆 Mulia「一個人」鎖定幾間科技產業大公司來寫，難免有些市場上的遺珠沒有被討論到；現在，希望凝聚熱愛分析科技產業的「一群人」，大家各有所長，集結社群的力量把相關領域研究地更透徹。</p><p>想當然爾，購買 NFT 的人——<strong>解碼機器人（Decoder BOT NFT）憑證</strong>——將會被邀請進入專屬 Discord 社群（科技巨頭研究院）。至此，都與坊間所見的專案差不多。</p><figure class="image"><img src="https://assets.matters.news/embed/31d9d38d-e806-41ee-9c43-c4ec65ef0d12.png" data-asset-id="31d9d38d-e806-41ee-9c43-c4ec65ef0d12" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>不過拜讀完 <a href="https://vocus.cc/vocuverse/collections/TechGiantsDecode/whitePaper" rel="noopener noreferrer" target="_blank">白皮書</a>，也聽了幾場 <a href="https://blog.hootsuite.com/social-media-definitions/ama-ask-me-anything/" rel="noopener noreferrer" target="_blank">AMA</a>（Ask Me Anything）下來。我看見幾個令人耳目一新的亮點，<strong><u>對於 NFT 實用價值的探索、創造、支撐</u></strong>，具有正面助益：<br class="smart"><br class="smart"></p><h2><strong>１・社群金庫：探索實用價值</strong></h2><p>發行 NFT 募得的資金，不是由項目方「單方面」定奪該怎麼使用。特別保留總金額的 40% 形成社群金庫，由大家「集思廣益」<u>提案與投票，一起</u><strong><u>探索</u></strong><u> NFT 實用價值</u>。<br class="smart"><br class="smart"></p><h2><strong>２・分析挖礦：創造實用價值</strong></h2><p>平時，社群金庫有個例行性任務，用來支應<strong>分析挖礦（Analyze to Earn）</strong>稿費。</p><p>很直覺的想法，一個社群內每個人出力不同。例如有些人是純讀者，有些人是分享者，必須打造某種「誘因」機制，獎勵辛勤付出的人。採取的作法是，針對那些好的研究文章投放代幣——<strong>科技點（Tech Point Token）</strong>——<u>讓分析有價，</u><strong><u>創造</u></strong><u> NFT 實用價值</u>。<br class="smart"><br class="smart"></p><h2><strong>３・挹注補充：支撐實用價值</strong></h2><p>不知道你看到這邊，會不會產生這樣的疑惑，一直「坐吃山空」怎麼辦？</p><p>我覺得這個專案最酷的點子，來自於定期補充社群金庫的手段——<strong>洗腦機器人（Brain Washer NFT）憑證</strong>——它是給業主競標的曝光欄位，得標者可以在「主程式」付費專欄每月釋出的一篇免費公開文裡置放廣告，得標金額的 80% 將會挹注社群金庫，<u>持續的現金流，</u><strong><u>支撐</u></strong><u> NFT 實用價值</u>。</p><p>於是原先的「試用品」被重新活化，有了全新定位。完全不浪費這（因為免費而帶來的）高流量，眾人（業主、項目方、NFT 持有者）皆大歡喜，一舉數得；也呼應 <a href="https://matters.news/@penfarming/295223-%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E5%9C%A8%E5%B9%A3%E5%9C%88-%E6%88%91%E6%83%B3-%E8%85%B3%E8%B8%8F%E5%AF%A6%E5%9C%B0-%E7%9A%84%E8%B3%BA%E9%8C%A2-%E5%BE%9E-stepn-nft-%E5%BE%97%E5%88%B0%E5%9B%9B%E5%80%8B%E8%A7%80%E5%AF%9F%E8%88%87%E6%80%9D%E8%80%83-bafyreic7uqhdj7vowmnmtzdseqwsiwyyxnnmo3o3rbccbv46cwiic4x3mi" rel="noopener noreferrer" target="_blank">在幣圈，我想「腳踏實地」的賺錢</a> 文末「虛實結合到線下」的想法，為求永續經營，<strong>需要有「虛」幣以外的「實打實」金流。</strong></p><hr><hr><h2><strong>■ 尾聲：三足鼎立，不能偏廢</strong></h2><p>回顧本文，我認為在觀察一個 NFT 項目時——主打「賦能」的專案——必須仔細探究下列問題：</p><ol><li>誰定義它的實用價值（單方決定 vs 一齊<strong><u>探索</u></strong>）？</li><li>實用價值的<strong><u>創造</u></strong>（誘因機制是否完善）？</li><li>如何<strong><u>支撐</u></strong>實用價值，永續經營（持續挹注金流）？</li></ol><p>三足鼎立，不可以偏廢；一只憑證，無垠地想像。</p><hr><hr><figure
