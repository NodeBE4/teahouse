<p>作者：土井英司<br class="smart">譯者：歐凱寧<br class="smart">出版社：天下雜誌</p><figure class="image"><img src="https://assets.matters.news/embed/d34d6746-8b51-47b2-b5ab-ca3e3b3732c3.jpeg" referrerpolicy="no-referrer"><figcaption><span>《一流的人讀書，都在哪裡畫線？：菁英閱讀的深思考技術》</span></figcaption></figure><p>這本書教人讀商業書——從挑選、內容吸收、化為實際行動——畫上專屬於你的一條線。</p><hr><hr><h2><strong>■ 王牌採購教你「投資」</strong></h2><p>作者 <a href="https://bookzone.cwgv.com.tw/authors/details/11790" target="_blank">土井英司</a> 在日本亞馬遜任職期間，被稱作「王牌採購」，擅長企劃行銷、一手打造暢銷書。</p><p>之後自立門戶，成立 <a href="https://eliesbook.co.jp/" target="_blank">Elies 書本顧問公司</a>，發行電子報，針對忙碌的企業主與商務人士，介紹優質商業書，是在哪裡畫了線？也經營養成班，從中發掘並推廣新作家，像是近藤麻理惠的《<a href="http://moo.im/a/chjlyH" target="_blank">怦然心動的人生整理魔法</a>》就出自於此，全球銷量達 700 萬冊，是最成功的一次出版。</p><figure class="image"><img src="https://assets.matters.news/embed/3859b368-2554-47db-b1d3-5c5020c87cc3.png" referrerpolicy="no-referrer"><figcaption><span>圖片來源 ©2021《Elies Book Consulting Co》官網</span></figcaption></figure><p>基於這樣的背景，本書在序言就開門見山，闡明聚焦於<strong>讀商業書</strong>。不求娛樂，追求「<strong>目的</strong>」。不是消費，而是「<strong>投資</strong>」。</p><hr><hr><h2><strong>■ 讀前認知</strong></h2><p><a href="https://bookzone.cwgv.com.tw/authors/details/11790" target="_blank">土井英司</a> 讀商業書，自有他的一套哲學，分述如下：</p><figure class="image"><img src="https://assets.matters.news/embed/0b253be5-53b2-42d9-b6f7-ea98c10d0c1b.jpeg" referrerpolicy="no-referrer"><figcaption><span>Photo by Austin Distel on Unsplash</span></figcaption></figure><h2>✅ 改變視野的鏡頭</h2><p>好的商業書必定要包含「<strong>作者的濾鏡</strong>」。</p><p>彷彿與大人物當面聊天，學到偉大作者們的觀點與思維，這可以說是一種濾鏡。透過它來觀察世界，可以幫助你找出自己與他人的差異點，促進自我成長。<br class="smart"></p><h2>✅ 自己才是主角</h2><p>作者反思寫「書評」這件事。</p><p>書評要有自我意識，說明自己從書中吸收了什麼，重點放在<strong>自己閱讀之後的變化</strong>，而非書本身的內容。</p><h2>✅ 就像在挖礦</h2><p>別執著於一定要「把書全部看完」。</p><p>商業書只要有一個小地方對讀者有幫助，其他全部派不上用場也沒關係；<strong>就像在挖礦</strong>，只要挖到鑽石，其他砂石都不重要。</p><h2>✅ 不挑食</h2><p>刻意去接觸價值觀相左的作者成功經驗，或自己不擅長的知識領域，從中獲得影響與刺激；列出<strong>八種領域</strong>（詳見附錄），以資參考。</p><hr><hr><h2><strong>■ 所以，該在哪裡畫線？</strong></h2><p>這個問題可以從反面來回答，先說說「哪裡不該畫線」吧！以下是作者希望我們避開的雷區：</p><ul><li><strong>自我陶醉線</strong><br class="smart">讓你覺得「對！沒錯！我想的果然沒錯！」的部分，這只是確認既有信念，毫無意義。</li><li><strong>自稱的內容</strong><br class="smart">商業書的作者通常都是功成名就才會寫書，他們提出<strong>後見之明</strong>（詳見《<a href="https://matters.news/@penfarming/%E5%85%89%E7%92%B0%E6%95%88%E6%87%89-%E8%AA%8D%E7%9F%A5%E6%80%9D%E7%B6%AD%E8%A3%A1%E5%B8%B8%E8%A6%8B%E7%9A%84%E4%B9%9D%E5%A4%A7%E9%99%B7%E9%98%B1-bafyreihv6afoj7ttgnbxml6gnyhod2scr7uas7kcivmc7xk3jado23jcfe" target="_blank">光環效應</a>》的探討<strong>）</strong>，其中有些說法，往往出自主觀，缺乏驗證；面對這些「自稱」的內容，都該打個折扣，或全數跳過。</li><li><strong>佳話</strong><br class="smart">被佳話感動不是壞事，佳話是心靈的養分。但務必搞清楚，佳話與做生意是沒有關係的。</li><li><strong>結果</strong><br class="smart">原因比結果重要。<br class="smart"></li></ul><p>所以，若有一句話（新發現、新思維）令你感到不太舒服，但印象深刻，務必要畫上一條線；不懂的地方，記住這個不懂，畫上一條線；他怎麼把生意做成功了？他採取了哪些作法？原因和行動，也值得畫上一條線。</p><hr><hr><h2><strong>■ 心得：想完成「三部曲」</strong></h2><p>在讀《<a href="http://moo.im/a/kpGQTX" target="_blank">一流的人讀書，都在哪裡畫線？</a>》時，時而聯想到樊登的《<a href="https://matters.news/@penfarming/%E8%AE%80%E6%87%82%E4%B8%80%E6%9C%AC%E6%9B%B8-%E6%A8%8A%E7%99%BB%E7%9A%84%E8%AE%80%E6%9B%B8%E5%93%B2%E5%AD%B8-bafyreidivqipcbtz2ngj6kkgnb3h6bzudsxgfsgrya47i4gz4afc2gobnq" target="_blank">讀懂一本書</a>》。</p><p>兩本書都信奉「不舒服」的讀書原則，也都提到建立「理解力池子」的概念（背景知識庫的通用分類），也都親睞「科學性」的書（可驗證）。不過，<strong>對文學作品的看法</strong>似乎有所歧異？</p><p>土井英司認為小說很有趣，所以能夠用被動的心態去讀，<strong>這是在消費小說中的趣味</strong>，是一種娛樂；樊登則是提到，千萬不要以為自己讀一些詩、讀一些哲學，似乎在生活起不到多大作用，<strong>其實它會潛在地影響我們對事物的判斷</strong>。</p><p>由於土井英司對這個議題點到即止，無從得知更多，但至少在讀者心中點入一絲漣漪，起了一道發想。</p><figure class="image"><img src="https://assets.matters.news/embed/3a956567-18b3-43cd-a0c5-a61eeb941a30.jpeg" data-asset-id="3a956567-18b3-43cd-a0c5-a61eeb941a30" referrerpolicy="no-referrer"><figcaption><span>Photo by Sigmund on Unsplash</span></figcaption></figure><p>拉回正題，談談畫線這件事。</p><p>樊登說他讀書時不太畫線，讀完後放下，給自己一段時間沈澱。經過了回憶的篩選，洗滌出印象最深刻的內容，然後「二刷」時才會畫線；但並未多著墨。</p><p>土井英司則是提出了一套具體的畫線原則，滿足了我看完《<a href="https://matters.news/@penfarming/%E8%AE%80%E6%87%82%E4%B8%80%E6%9C%AC%E6%9B%B8-%E6%A8%8A%E7%99%BB%E7%9A%84%E8%AE%80%E6%9B%B8%E5%93%B2%E5%AD%B8-bafyreidivqipcbtz2ngj6kkgnb3h6bzudsxgfsgrya47i4gz4afc2gobnq" target="_blank">讀懂一本書</a>》後的意猶未竟。也覺得這方法，不單單只適用於商業書，<strong>任何「經世致用」之學都派得上用場</strong>。</p><p>我從樊登身上，一個說書人的角度，學得心智圖的編纂思維，談如何解構一本書；又在土井英司身上，一個投資人的立場，習得一流人士的畫線取捨，論如何讀商業書。</p><p>此時，突然還想拜讀《<a href="https://www.books.com.tw/products/0010736991" target="_blank">如何閱讀一本書</a>》這本經典。</p><p>猶如蒐集最後一塊拼圖，貌似完整了「三部曲」的儀式啊！</p><hr><hr><h2><strong>■ 附錄：如何讀商業書</strong></h2><p>文末筆記，附上〖爐內精裝版〗心智圖 👉 <a href="https://www.xmind.net/m/qqZnRV/" target="_blank">由此去</a>。摘錄作者讀商業書的心法，可作為「攻城掠地」的教戰指南。</p><figure class="image"><img src="https://assets.matters.news/embed/26f5bc7d-2947-4902-b53b-dd4b4fbba1ce.png" referrerpolicy="no-referrer"><figcaption><span>以 XMind 編纂，新視窗開啟可獲得較佳解析度</span></figcaption></figure><h2>✍️ 1. 不要「速讀」，要「精讀」</h2><p>遇到不擅長或不熟悉的內容，讀得慢是理所當然，慢慢讀才會懂，才會理解，才會吸收；<strong>由不知邁向知之</strong>，懂得愈深，讀書速度自然就愈快。</p><p>此外，<strong>只要有明確的目標</strong>，讀書的品質與時間都會改變。目標之外的資訊都不需要花時間讀，讀書速度也會提升。</p><blockquote>⚠️「快」只是一個結果，本身毫無價值。</blockquote><p>⋯</p><h2>✍️ 2. 別看「整體」，多看「部分」</h2><p>以打網球為例，一對一持續對打，這叫「整體練習」，順手的招式會一直沿用，怠慢了自己的弱項；然針對每一個小部分——例如發球、正手、反手、殺球、高吊球等——徹底強化，是所謂的「部分練習」。</p><p><strong>極力推薦部分練習，亦即見樹不見林</strong>。是一種強化優點，消除缺點的閱讀策略。</p><blockquote>⚠️ 部分練習的天龍八部：會計與金融、策略、行銷、營運、管理與領導、商品研發、統計、經濟學。</blockquote><p>⋯</p><h2>✍️ 3. 別看「結果」，要看「原因」</h2><p>在閱讀同時不斷探索成功的原因，書裡總會提及一兩個最關鍵的部分，然後反覆思考自己該如何運用這關鍵。</p><blockquote>⚠️ 原因就像保齡球的「第一瓶」。</blockquote><p>用第一瓶當作思考的起點，提升找原因的直覺。<strong>各行各業的第一瓶都大不相同</strong>。</p><p><strong>也要小心「導因為果」的誤判</strong>，例如鬥志和工作表現之間的關聯性。</p><p>這裡提到《<a href="https://www.books.com.tw/products/0010836749" target="_blank">管理學</a>》的研究分析：員工並不是因為充滿鬥志才有好的工作成果，反而是工作表現優良之後，才會產生鬥志。也就是說，有沒有鬥志只是結果，原因在於工作表現好壞。</p><p>⋯</p><h2>✍️ 4. 別製造「相同」，要創造「差異」</h2><p>學習書中前輩們如何創造差異，並運用在自己身上。</p><blockquote>⚠️ 差異來自於「搭配」。</blockquote><p>不僅加深自己的專業知識，還要接觸乍看之下與自己專長無關的知識，接著搭配組合。例如能談節稅的業務員（保險、汽車）。</p><blockquote>⚠️ 水往低處流。</blockquote><p>熟悉一門生意從上游到下游的全部流程，就能掌握在哪個階段做點什麼變化，這也是差異化的來源。</p><p>⋯</p><h2>✍️ 5. 留意「背景」</h2><p>作者以讀暢銷書為例，光是思考「它為何暢銷」就非常有價值。</p><p>別急著發表後悔受騙的心得——不好看、不值這個錢、看不懂、早就懂了——應該先確認該書的作者與編輯鎖定了怎樣的讀者群而暢銷？為什麼能引起普羅大眾的興趣？</p><blockquote>⚠️ 在「書本之外」畫線的概念也很重要。</blockquote><hr><hr><p>謝謝各位看到這邊，隨文附上我的〔<strong>讚賞公民 2.0</strong>〕〔<strong>圍爐</strong>〕〔<strong>其它平台</strong>〕鏈結，歡迎來逛逛 喔！</p><p>🌱 成為我的讚賞公民 👇 <a href="https://liker.land/leo7283/civic"
