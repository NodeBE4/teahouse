<p>在前一篇文章聊到 <a href="https://matters.news/@penfarming/366737-%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-nft-%E6%98%AF%E4%BB%80%E9%BA%BC-nft-%E8%A7%A3%E6%B1%BA%E4%BA%86%E4%BB%80%E9%BA%BC%E5%95%8F%E9%A1%8C-nft-%E4%B8%89%E5%A4%A7%E5%83%B9%E5%80%BC-bafybeicbky3tarwhkxm2kngso5ojvhmv2nwhtdfl4dw7vzipxsbjmcdzoy" rel="noopener noreferrer" target="_blank">NFT 三大價值來源</a>：</p><ol><li>藝術創作本身有收藏價值</li><li>基於情感意義的社群價值</li><li>融合線上線下的應用價值</li></ol><p>備註：這三點並非只能選邊站，它更像是一種比例調和。</p><figure class="image"><img src="https://assets.matters.news/embed/f229aa36-1230-4971-9ed2-f051163d2075.jpeg" data-asset-id="f229aa36-1230-4971-9ed2-f051163d2075" referrerpolicy="no-referrer"><figcaption><span>Photo by Andrey Metelev on Unsplash</span></figcaption></figure><p>根據反饋，貌似多數人更認同第三點這種「實打實」的應用價值，我自己也是這麼認為。而且就客觀事實來說，隨著有愈來愈多大品牌進入 NFT 領域，未來有很大部分 NFT 活動將發生在大企業的專案中，<strong>這會讓 NFT 變得更像商品——而不是投資</strong>——因此價格裡當不會太貴才是，更貼近普羅大眾的生活。</p><p>當中又以 Polygon 區塊鏈，與諸多 Web2 企業合作（Meta、Reddit、迪士尼、可口可樂、星巴克、國泰人壽⋯⋯），表現最奪人眼球。當中有一些專案是協助企業發行他們的 NFT，所以我在想在 2023 年，與其單押某個 NFT 項目，<strong>不如投資背後打造 NFT 基礎設施的團隊</strong>，也就是直接去買 MATIC 代幣，用 DCA（Dollar Cost Averaging）的方式，把它納入加密資產配置組合中，立刻付諸實行。</p><p>至於前兩點比較「虛」的 NFT 還是會繼續存在，不否認其價值。只是因為那虛無飄渺的特性，難以進行估值（要去估算一件數位藝術品的價值，遠比估算一張數位會員卡的價值困難，對吧！），自然就容易被價格炒作，我是比較少去碰的。</p><p>總的來說，我看好 NFT 這個技術（我會把資金投在 MATIC 上），NFT 將生成更多「商品」走入我們的日常，而猜測價格的炒作時代將逐漸式微。</p><hr><hr><figure
