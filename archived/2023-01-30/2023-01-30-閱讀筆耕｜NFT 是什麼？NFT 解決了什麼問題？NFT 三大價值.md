👈</p><hr><hr><p>「賣的出去是 NFT，賣不出去就是 JPG。」</p><p>不知道你是否納悶，我們在網路上看到的龐克頭、猴子、日系動漫頭像，不都是簡單的 JPEG 檔案嗎？為什麼它們「變成」NFT 以後，隨隨便便都可以賣到幾萬甚至幾 10 萬美元？更重要的是，這些 NFT 到底是單純的炒作泡沫，還是正在對世界產生顛覆性的影響？</p><figure class="image"><img src="https://assets.matters.news/embed/bc6bdd5d-10d0-42be-abf0-d1e4452dff52.png" data-asset-id="bc6bdd5d-10d0-42be-abf0-d1e4452dff52" referrerpolicy="no-referrer"><figcaption><span>資料來源：OpenSea</span></figcaption></figure><p>本篇文章將用簡單的方式帶你了解 NFT，包括 NFT 是什麼，NFT 的發行方式與解決了什麼問題，以及 NFT 三大價值來源。</p><hr><hr><h2>■ <strong>NFT 是什麼？</strong></h2><p>NFT 全稱 Non-Fungible Tokens，中文譯作「非同質化代幣」。</p><p>NFT 和許多加密貨幣一樣，是 <a href="https://chainee.io/what-is-blockchain/" rel="noopener noreferrer" target="_blank">區塊鏈</a> 上的一種資料形式，是一枚「代幣」。每枚代幣可以代表一個「非同質化」資產，例如圖像、藝術品、影音、體育、飲食、遊戲、甚至是一則社群貼文⋯⋯。</p><p>總的來說，NFT 是一種專門紀錄非同質化資產的資料單位。</p><pre class="ql-syntax" spellcheck="false">💡 區塊鏈小知識：「同質化」vs「非同質化」首先「同質化」是指資產可與同類型資產互換，或是可拆分的特性。舉例來說，比特幣、以太幣和其它加密貨幣，又或是你手邊的新台幣，都是一種同質化的資產，因為與同類型資產之間，價值相等且能夠切割成更小的單位。如果今天我拿 1 張千元大鈔，跟你的 10 張枚百元鈔票做替換，你會覺得無所謂，因為兩者的價值相同。而「非同質化」自然是指獨一無二，無法與同類型資產互換，拆分的資產。舉例來說，畢卡索的畫作，蘇富比的拍賣品，又或是不動產權狀，每件資產都與眾不同，你無法找到第二件一模一樣的資產。就算出自於同一人（畢卡索）之手，他的每一件作品都擁有不同的價值，彼此無法互相替換。要注意的是，「同質化」與「非同質化」的定義是可以改變的。如果一件同質化資產被賦予內在價值或獨有特徵，便可重新定義為一件獨一無二的非同質化資產。例如，我的千元大鈔擁有周杰倫的簽名，那它就變成非同質化資產，反之亦然。</pre><p>我們可以把 NFT 想像成原創者為數位資產發行的「身分證」或「防偽證書」，因為被放在區塊鏈上的緣故，我們能夠簡單驗證作品的出處，真偽，稀缺性。</p><p>換句話說，當你在買賣 NFT 時，你買的其實不是作品本身，<strong>而是指引作品所在的數據</strong>，是原創者為作品發行的「身分證」或「防偽證書」。</p><hr><hr><h2>■ <strong>NFT 的發行方式</strong></h2><p>我們常聽到的「將一件作品上鏈做成 NFT」，乍聽之下是將某個電子檔案放到區塊鏈上，成為一枚區塊鏈的代幣，其實不然。</p><p>創作一件 NFT，或說將一件作品做成 NFT，指的是將該作品的電子檔案儲存在一個去中心化的雲端空間，並在區塊鏈上鑄造一個代幣，<strong>而這個代幣就成了獨一無二、永遠代表該作品（電子檔案）的資料數據</strong>。這個代幣的擁有者，就是這件 NFT 的所有人。</p><p>目前主流發行 NFT 的方式有兩種。第一種是直接透過例如 <a href="https://opensea.io/zh-TW" rel="noopener noreferrer" target="_blank">OpenSea</a>、<a href="https://www.oursong.com/" rel="noopener noreferrer" target="_blank">Oursong</a> 等 NFT 平台直接發行，<strong>這些平台擁有簡單的使用者介面及導覽</strong>，即使是不懂區塊鏈技術的用戶，也能輕鬆在平台上製作 NFT。</p><p>第二種則是透過智能合約（Smart Contract）來發行，把它想像成放在區塊鏈上運行的一串程式碼，交易不需透過第三方，只要滿足特定條件將會自動執行；相較於第一種方式，<strong>使用智能合約來製作 NFT 可獲得完整的發行控制權</strong>，例如有名的 CryptoPunk、無聊猿、Azuki 都是使用智能合約發行的 NFT。</p><figure class="image"><img src="https://assets.matters.news/embed/ea95c3a2-2d77-4dff-8b9e-49b293b55a8e.jpeg" data-asset-id="ea95c3a2-2d77-4dff-8b9e-49b293b55a8e" referrerpolicy="no-referrer"><figcaption><span>NFT 的發行方式</span></figcaption></figure><hr><hr><h2>■ <strong>NFT 解決了什麼問題？</strong></h2><p>以前，只有實體世界裡的作品能被區分真品和贗品，數位世界裡的「非同質化資產」一向難以被重視，這是因為這些這些數位作品通常很難被辨識何者是最原始的正版，缺乏稀缺性或獨特性。</p><p>舉例來說，作為一個數位藝術創作家，嘔心瀝血創造的作品竟然可以簡單地被 Ctrl+C（複製）、Ctrl+V（貼上） 剽竊，又或是網路上出現假冒的詐騙帳號⋯⋯種種原因造成數位藝術創作一直無法好好地發展。</p><p>而 NFT 就能很好地解決這些問題，讓我們來聊聊 NFT 的創新之處：</p><h3><strong>１・允許他人輕鬆驗證真偽</strong></h3><p>NFT 是原創者在區塊鏈上為自己的作品鑄造的身分證，因為身分證被紀錄在區塊鏈上，其他人能夠輕鬆利用區塊鏈技術確認 NFT 的真偽，來看看這些 NFT 是不是由原創者發行的。<br class="smart"><br class="smart"></p><h3><strong>２・確保稀缺性以及獨特性</strong></h3><p>收藏家可以透過檢查發行 NFT 的智能合約，來確認 NFT 的發行總量與每個 NFT 的獨特性。這些資料因為被放在區塊鏈上，所以擁有不可竄改，並且完全公開透明的特性。<br class="smart"><br class="smart"></p><h3><strong>３・帶來持續性的版稅收入</strong></h3><p>實體世界裡，藝術家把作品賣掉後，之後作品的轉手交易就跟他毫無關係了，就算作品被炒到天價，原創者也不會因此拿到任何好處。</p><p>但是 NFT 就不一樣了，只要在智能合約上訂定版稅收取的條件（通常是交易金額的 10%），那麼以智能合約發行的 NFT 就能夠以自動化的方式，讓 NFT 的每一次轉手，都可讓原創者獲得一定比例的版稅收入，它意味著可以維持生計，這就能夠鼓勵創作者持續創作。</p><figure class="image"><img src="https://assets.matters.news/embed/136e8e12-351e-4329-bdad-01015f3e2fc2.jpeg" data-asset-id="136e8e12-351e-4329-bdad-01015f3e2fc2" referrerpolicy="no-referrer"><figcaption><span>NFT 解決了什麼問題？</span></figcaption></figure><hr><hr><h2>■ <strong>NFT 三大價值來源 </strong></h2><p>想要了解 NFT 的價值來源，可以從「人們為什麼想買入這些 NFT」來探討，撇開投機炒作不說，主要可分為三個部分：</p><h3><strong>１・藝術創作本身有收藏價值</strong></h3><figure class="image"><img src="https://assets.matters.news/embed/a35b1059-c2e0-4d36-a6c6-4998007b0405.jpeg" data-asset-id="a35b1059-c2e0-4d36-a6c6-4998007b0405" referrerpolicy="no-referrer"><figcaption><span>藝術創作本身有收藏價值</span></figcaption></figure><p>就跟現實生活中，人們喜歡收藏各種東西一樣，<strong>NFT 技術確保了數位資產的稀缺性、獨特性、可追溯性以及防偽性，讓收藏數位作品一事變得有意義</strong>；雖然任何人依舊能 Ctrl+C（複製）、Ctrl+V（貼上）來拷貝數位作品，但卻無法取得原創者為這件數位作品製作的「身分證」或「防偽證書」。</p><p>把作品製作成 NFT 的方式，能讓許多收藏狂熱者有了令人眼睛為之一亮的新賽道。隨著疫情推動人們日常生活朝向數位化的影響，使用 NFT 技術驅動的數位藝術創作，未來勢必蔚為主流。<br class="smart"><br class="smart"></p><h3><strong>２・基於情感意義的社群價值</strong></h3><figure class="image"><img src="https://assets.matters.news/embed/4c55381c-0674-4b16-add6-4a8904a9b4be.jpeg" data-asset-id="4c55381c-0674-4b16-add6-4a8904a9b4be" referrerpolicy="no-referrer"><figcaption><span>基於情感意義的社群價值</span></figcaption></figure><p>以 PFP（Profile Picture）類別的 NFT 來說，可以把他想像成參加某個私密社群的驗證手段，<strong>滿足人們有情感交流的本能需求。</strong></p><p>例如，持有無聊猿 NFT 就能進入無聊猿俱樂部，與其他成員在推特上聊天互動。想像一下，能加入有歌手林俊傑以及 NBA 球星 Stephen Curry 在內的社群，是不是非常吸引人呢？</p><p>以此前提，未來還可能延伸發展出以 NFT 社群為基礎的去中心化自治組織（Decentralized Autonomous Organization, DAO），讓同一系列的 PFP 頭像，恰好反映出具有同樣價值觀或信念的一群人，一起完成共同的目標。<br class="smart"><br class="smart"></p><h3><strong>３・融合線上線下的應用價值</strong></h3><figure class="image"><img src="https://assets.matters.news/embed/a44feeb3-1b50-4fe4-a5b8-3786deb8f4a4.jpeg" data-asset-id="a44feeb3-1b50-4fe4-a5b8-3786deb8f4a4" referrerpolicy="no-referrer"><figcaption><span>融合線上線下的應用價值</span></figcaption></figure><p>NFT 除了本身收藏價值，以及提供社群價值以外，還有結合實體世界與數位世界的應用價值，<strong>意味著 NFT 能夠提供實用性</strong>。</p><p>Domain 就是其中一個例子。允許使用者不再需要填寫一長串的收款地址，而是輸入簡短的網址代替，就可以進行後續支付動作，這是域名 NFT 提供的應用價值。</p><p>此外，現在有越來越多的 NFT 主打「賦能」（Utility）效果，也就是發行者透過一系列行為，為該 NFT 增添實質益處，希望帶給 NFT 持有者不同的價值。不光是現實生活，亦包涵數位世界。以現實生活舉例，買入某間餐廳發行的限量 NFT，也許從此能夠讓你在點餐時，享受特殊優惠；以數位世界舉例，買入 NFT 也許能夠讓你在玩遊戲之餘，還能賺取加密貨幣。</p><hr><hr><figure class="image"><img src="https://assets.matters.news/embed/235d2f9b-ab0a-4fe2-8ead-49b2b898ca47.png"
