是什麼？BounceBit（BB）積分任務教學</figcaption></figure><p>哇！不只 Launchpool，加密貨幣交易所龍頭幣安 Binance 能「擼羊毛」的地方又多一個！</p><p>這篇文章要告訴你，幣安最近推出的 Megadrop 是什麼；以及手把手教你參與 Megadrop 平台首發的 BounceBit（BB）積分活動。讓我們馬上開始吧！</p><hr><h2><strong>幣安 Megadrop 是什麼？</strong></h2><p>幣安 Megadrop 是一種全新「代幣發行平台」，結合空投活動的舉行，為用戶帶來全新體驗。</p><p>同樣是和新幣發行有關，相較於幣安原有的 Launchpool／Launchpad，用戶在使用 Megadrop 時感受最不同的地方是，<strong>必須連接幣安 Web3 錢包</strong>。</p><ul><li><p>延伸閱讀：<a target="_blank" rel="noopener noreferrer nofollow" href="https://chainee.io/binance-web3-wallet/">幣安 Web3 錢包一鍵創建，不再需要使用助記詞，自己持幣好輕鬆</a></p></li></ul><p>合理推測，幣安推出 Megadrop，應是為了刺激用戶的幣安 Web3 錢包採用率，充滿戰略性意義的一步棋。</p><hr><h2><strong>幣安 Megadrop 上線 BounceBit（BB）</strong></h2><p>目前，幣安在 Megadrop 平台也已經上線首發項目 BounceBit（BB），這是一個專門為比特幣設計的「再質押」服務。</p><p>跟據 <a target="_blank" rel="noopener noreferrer nofollow" href="https://www.binance.com/zh-CN/research/projects/bouncebit">幣安研究院</a> 的介紹，BounceBit（BB）是一個專注於為比特幣資產賦能的創新區塊鏈平台，透過中心化金融（CeFi）和去中心化金融（DeFi）的融合，以及再質押（restaking）的策略，要將比特幣從一種被動資產，轉變為加密生態系統中的積極參與者。</p><p>BounceBit 具備以下亮點：</p><ul><li><p><strong>BTC 再質押</strong><br class="smart">增加了資產的流動性和收益機會。</p></li><li><p><strong>雙幣 PoS 共識機制</strong><br class="smart">允許驗證者同時接受 BBTC 和 BB 代幣，者種機制稱強了網路的彈性和安全性，同時擴大了參與者基礎。</p></li><li><p><strong>BounceClub</strong><br class="smart">不必具備程式設計基礎，用戶也能使用內建工具創建 DeFi 產品。</p></li><li><p><strong>流動託管</strong><br class="smart">BounceBit 引入流動託管概念，使抵押的資產保持流動性，並提供更多收益機會。</p></li></ul><hr><h2><strong>幣安 Megadrop BounceBit（BB）積分任務怎麼參與？</strong></h2><p>首先，你得在幣安交易所有一個完成 KYC 的帳戶，以及開通 <a target="_blank" rel="noopener noreferrer nofollow" href="https://chainee.io/binance-web3-wallet/">幣安 Web3 錢包</a>，這些是前置條件。</p><blockquote><p>還未註冊幣安交易所的朋友，使用「閱讀筆耕」專屬 <strong><a target="_blank" rel="noopener noreferrer nofollow" href="https://accounts.binance.info/register?ref=L3BAGL6Y">推薦連結</a></strong> 註冊，或是手動輸入閱讀筆耕專屬推薦碼「L3BAGL6Y」，享有交易手續費終身 20% 返現優惠。</p></blockquote><p>活動期間為即日起至 5 月 13 日，詳見 <a target="_blank" rel="noopener noreferrer nofollow" href="https://www.binance.com/zh-TC/support/announcement/a0a11e79467e4ee6a937348005f7c84f">幣安 BounceBit（BB）Megadrop 任務公告辦法</a>，為大家梳理成友善的 SOP 操作步驟，讓你輕鬆完成這些積分任務。</p><h3><strong>≣ 積分是做什麼的？積分是怎麼計算？</strong></h3><p>活動期間持續累積積分，用戶最終積分將獲得 Megadrop 獎勵。</p><blockquote><p>積分計算公式為 =（鎖倉 BNB 積分 x Web3 任務乘數）+ Web3 任務積分</p></blockquote><ul><li><p>鎖倉 BNB 積分（基於 BNB 定期產品快照期間的每小時快照）=（120 天 BNB 平均鎖倉數量 x 130）+（90 天 BNB 平均鎖倉數量 x 120）+（60 天 BNB 平均鎖倉數量 x 110）+（30 天 BNB 平均鎖倉數量 x 100）</p></li><li><p>Web3 任務乘數：1.5</p></li><li><p>Web3 任務積分：1000</p></li></ul><p>如果以上描述看得頭疼也沒有關係，只要記得有「兩種」積分任務要做就可以了！</p><p>第一種任務是在「幣安交易所」裡完成，第二種任務是在「幣安 Web3 錢包」裡完成。</p><p><strong>只要完成一種任務，就可以掌握一部分積分入袋（當然，建議你兩種任務都完成）</strong>。例如，如果你只有完成 Web3 任務，沒有做鎖倉 BNB 任務，屆時，你的積分是 0 x 1.5 + 1000 = 1000 積分。</p><p> </p><h3><strong>≣ 積分任務一：幣安「交易所」購買 BNB 定期產品</strong></h3><p>類似傳統金融中「定存」的概念，操作路徑是幣安 App 首頁 → 左上角幣安 logo → 找到 Megadrop 圖示 → 鎖倉 BNB。</p><p>有各種期間（120天、90天、60天、30天）可以參與，鎖倉的金額愈大、天數愈長，所創造的積分效果也愈高。</p><figure class="image"><img src="https://imagedelivery.net/kDRCweMmqLnTPNlbum-pYA/prod/embed/8749456a-5748-42a9-9ffb-b2d20c4f8669.png/public" referrerpolicy="no-referrer"><figcaption>積分任務一：幣安「交易所」購買 BNB 定期產品</figcaption></figure><h3><strong>≣ 積分任務二：幣安「Web3 錢包」裡體驗 BounceBit 質押服務</strong></h3><p>如果你的幣安 Web3 錢包餘額是零，轉一些 BNB 代幣到幣安 Web3 錢包（大約等值 10U 就夠了，是為了下一動要 swap 和當作 gas fee 用）。</p><p>在 Web3 錢包裡，切換到 BSC 鏈，把 BNB 代幣 swap 成 0.0001 BTCB 代幣（等值不到 10U）。</p><p>從稍早展示過的 Megadrop 圖示進入 BounceBit 任務頁面「完成 Web3 任務」，連接幣安 Web3 錢包，並質押 0.0001 BTCB 代幣。</p><figure class="image"><img src="https://imagedelivery.net/kDRCweMmqLnTPNlbum-pYA/prod/embed/9b826399-ffa8-4f96-956c-094d0b114fcd.png/public" referrerpolicy="no-referrer"><figcaption>積分任務二：幣安「Web3 錢包」裡體驗 BounceBit 質押服務</figcaption></figure><p>最後，記得回到 BounceBit 任務頁面「完成 Web3 任務」按下「驗證」鈕才算數。</p><hr><h2><strong>結語</strong></h2><p>隨著幣安推出全新的 Megadrop 平台，加密貨幣市場又添了一處新的「擼羊毛」機會。做為先發球員，參與比特幣再質押的 BounceBit（BB）積分活動，不僅是一個賺取額外收益的機會，也讓你能夠體驗最前沿的 Web3 技術。</p><p>希望這篇手把手教學，能幫助你掌握如何參與活動並從中獲益。也別忘了投資前，記得進行適當的研究，以確保每一步都符合你的財務目標和風險承受能力。祝你玩得愉快！</p><hr><blockquote><p>🌱 免費訂閱【<strong><a target="_blank"
