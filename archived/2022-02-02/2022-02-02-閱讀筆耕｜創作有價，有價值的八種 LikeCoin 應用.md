<div>
<p>寫作可以理解為「創造力挖礦」，付出創意，產生價值，服膺於「創造力證明（Proof of Creativity）」的共識機制下，得到名曰 LikeCoin 的通證（token）。</p><p>本文試著梳理各種打理 LikeCoin 的方式，一共有八種，俯瞰全局，可以做為新用戶入門的引導地圖。</p><hr><hr><h2><strong>■ 八種打理 LikeCoin 的方式</strong></h2><p>不囉唆，先上圖，跟著思考脈絡，依序介紹每個「菱形」區塊。</p><figure class="image"><img src="https://assets.matters.news/embed/ad0ca8bb-75f7-47d6-8449-f9452bd931a5.jpeg" data-asset-id="ad0ca8bb-75f7-47d6-8449-f9452bd931a5" referrerpolicy="no-referrer"><figcaption><span>打理 LikeCoin 的八種方式（中文版）</span></figcaption></figure><p>一開始，大家的 LikeCoin 都會落在 <a href="https://liker.land/getapp" rel="noopener noreferrer" target="_blank">Liker Land</a> 應用程式裡頭，你可以選擇：</p><ul><li>① <strong>單純持有（HODL）</strong><br class="smart">什麼都不做，順帶連結「I AM HODLING」的 <a href="https://bitcointalk.org/index.php?topic=375643.0" rel="noopener noreferrer" target="_blank">典故</a> 出處，拼錯乃刻意為之。<br class="smart"><br class="smart"></li><li>② <strong>委託、質押（Stake）</strong><br class="smart">可於 Liker Land 就地進行（詳見 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-like-coin-%E7%94%9F%E5%88%A9%E6%81%AF-%E6%A6%82%E5%BF%B5%E5%8F%8A%E6%93%8D%E4%BD%9C-%E7%B2%BE%E8%8F%AF%E7%89%88-bafyreietrraadnieba5ojo4lv6cltnfyd6khlcatml5okczvl3pc2yhlbm" rel="noopener noreferrer" target="_blank">LikeCoin生利息，概念及操作</a> 教學）。但有件事必須知悉，就是你並未取得「<a href="https://matters.news/@erica072/%E8%B2%93%E7%89%B9%E5%8B%99%E5%A0%B1%E5%A0%B1-%E9%80%B1%E5%A0%B1-and-%E6%9C%89%E4%BA%86-like-land-%E9%8C%A2%E5%8C%85-%E7%82%BA%E4%BD%95%E9%82%84%E8%A6%81%E7%94%A8-keplr-2021-10-03-2021-10-09-bafyreic6cdwhxrkzezvmc3ygtrf7m72rzk7kzwz4mjahuaixj5j7erynim" rel="noopener noreferrer" target="_blank">私鑰</a>」，所以 $LIKE 是被託管，嚴格來說 <a href="https://matters.news/@erica072/%E8%B2%93%E7%89%B9%E5%8B%99%E5%A0%B1%E5%A0%B1-%E9%80%B1%E5%A0%B1-and-%E6%9C%89%E4%BA%86-like-land-%E9%8C%A2%E5%8C%85-%E7%82%BA%E4%BD%95%E9%82%84%E8%A6%81%E7%94%A8-keplr-2021-10-03-2021-10-09-bafyreic6cdwhxrkzezvmc3ygtrf7m72rzk7kzwz4mjahuaixj5j7erynim" rel="noopener noreferrer" target="_blank"><strong>控制權不在己身，手續費也比較昂貴</strong></a>。<br class="smart"><br class="smart">若想解決以上問題，把 $LIKE 提領至（withdraw to）一個你擁有「私鑰」的 <a href="https://www.keplr.app/" rel="noopener noreferrer" target="_blank">Keplr</a> 錢包是個好辦法（詳見 <a href="https://matters.news/@daisy/bafyreifgvlva5xmncblzm5afdvykws4cahgnutxxnbx5lbpjz4c4hgtzjy" rel="noopener noreferrer" target="_blank">Keplr錢包全攻略，大家K噴發大財</a> 教學），這裡也可以質押 $LIKE。<br class="smart"></li></ul><p>倒也不用一股腦兒把家當全數都搬去 Keplr 錢包，留一些資金在 Liker Land 用於：</p><ul><li><strong>③ 打賞（Donate）</strong><br class="smart">按下文章下方的「支持作者」黃色按鈕，<strong>用 #LIKE 鼓勵創作者</strong>，<s>本文剛好可以做為大家練習的對象，快拿 #LIKE</s><strong><s> </s></strong><s>砸我</s>。<br class="smart"><br class="smart"></li><li><strong>④ 市集（Market）</strong><br class="smart"><strong>用 $LIKE 交易其它市民提供的商品或勞務</strong>，見於 <a href="https://matters.news/tags/VGFnOjE4MjQ5" rel="noopener noreferrer" target="_blank">#Matters市集</a>、<a href="https://matters.news/tags/VGFnOjg1NzM0" rel="noopener noreferrer" target="_blank">#LikeCoin市集</a>⋯⋯等標籤。</li></ul><p>以上四種用途，只消透過 Liker Land 即可完成。喘口氣，容我先曬一下英文版的架構圖，接下來要從 Keplr 錢包的「右側支脈」做開展。<br class="smart"></p><figure class="image"><img src="https://assets.matters.news/embed/bf3b1656-33fb-4dd1-b392-afcfcfbf4497.jpeg" data-asset-id="bf3b1656-33fb-4dd1-b392-afcfcfbf4497" referrerpolicy="no-referrer"><figcaption><span>打理 LikeCoin 的八種方式（英文版）</span></figcaption></figure><p>以下四種用途，可以用「<strong>幣幣交換（Swap）</strong>」一以貫之，想要換的東西不同罷了。</p><pre class="ql-syntax" spellcheck="false">精簡用詞◍ 提領至（withdraw to）以下簡稱轉帳、轉到、轉入◍ 中心化交易所（Centralized Exchange）以下簡稱 CEX◍ 去中心化交易所（Decentralized Exchange）以下簡稱 DEX</pre><ul><li><strong>⑤ 法幣（Flat Money）<br class="smart"></strong>想換成「真錢」。<br class="smart"><br class="smart">從 Liker Land 把 $LIKE 轉到支援「<strong>LIKE／USDT 交易對</strong>」的 CEX，目前符合這樣條件的 CEX 有三間：<a href="https://matters.news/@thumbb13555/%E7%A2%BC%E8%BE%B2%E6%97%A5%E5%B8%B8-%E7%B7%9A%E4%B8%8A%E5%B7%A5%E4%BD%9C%E5%9D%8A-%E5%A6%82%E4%BD%95%E5%B0%87like-coin%E6%8F%9B%E5%87%BA%E7%82%BA%E6%B3%95%E5%B9%A3%E5%91%A2-bafyreidwd6ka6hiky355bxqzowbqrq36bdo4p5qvmapwpe6zafh7ojcpma" rel="noopener noreferrer" target="_blank">DigiFinex／Liquid／ZB</a>，<strong>一但換成穩定幣（Stablecoin）就穩了</strong>；再轉入支援「法幣出金」的 CEX，若以台灣人視角（法幣為新台幣）來看，轉帳至 <a href="https://www.johntool.com/taiwan-crypto-exchange/" rel="noopener noreferrer" target="_blank">ACE／BitoPro／MAX</a> 御三家。<br class="smart"><br class="smart"><strong>由於 $LIKE 在 CEX 的交易量小，搓合不易</strong>，故提出另一種方案，先把 $LIKE 換成交易量大的主流幣，事已至此，就送去更大規模的 CEX 變賣掉。<br class="smart"><br class="smart">現在拉回之前提過的 Keplr 錢包，<strong>它支援許多去中心化金融的對接</strong>，例如連上 <a href="https://app.osmosis.zone/?from=CRO&to=LIKE" rel="noopener noreferrer" target="_blank">Osmosis</a> DEX，輕鬆（免掛單）把 $LIKE 換成 $UST，然後轉到像是 <a href="https://accounts.binance.com/zh-TW/register?ref=296791741" rel="noopener noreferrer" target="_blank">幣安</a> 之類的大型 CEX，找尋「UST／USDT」交易對，<strong>一但換成穩定幣（Stablecoin）就穩了</strong>。<strong><br class="smart"><br class="smart"></strong></li><li><strong>⑥ 加密貨幣簽帳金融卡（Crypto-linked Cards）</strong><br class="smart">想換成「糧草」。<br class="smart"><br class="smart">據我所知，目前市面上與加密貨幣連結的卡片均是「<strong>簽帳金融卡</strong>」的概念（若有錯請指正），意思是說你把手中持有的加密貨幣，轉換成法幣——儲值——來做支付。<br class="smart"><br class="smart">端看你辦的是哪一張？若以 <a href="https://crypto.com/app/qks2bb35ts" rel="noopener noreferrer" target="_blank">Crypto.com 所推出的 CRO 卡</a> 為例，我會把 #LIKE 轉成 $USDT 備著，<strong>鎖住匯率波動</strong>，一遇上刷卡需求，就消耗「糧草」化為額度。<br class="smart"><br class="smart"></li><li><strong>⑦ 換幣並質押（Swap and Stake）</strong><br class="smart">想換「新標的」。<br class="smart"><br class="smart">不論是嫌質押 $LIKE 給的利率太低，或是長線更看好別的幣種，還是想分散持有降低風險，例如於 Osmosis DEX 把 $LIKE 換成其它虛擬幣，並順勢（透過 Keplr 錢包）完成質押，押注新投資標的。<br class="smart"><br class="smart"></li><li><strong>⑧ 流動性挖礦（Farm）</strong><br class="smart">想換「流動性提供者憑證（Liquidity Provider token, LP token）」。<br class="smart"><br class="smart">像是連上 Osmosis DEX，把 $LIKE 用 50／50 比例換成兩種幣，為某一組交易對背後的池子（Pool）注水，換得表彰受益權的 LP token（參與分配 swap fee 之證明）；更進一步，還能把它拿去質押，是謂流動性挖礦（Liquidity mining），有的地方改以 farming 取代 mining 一詞，意思相同。<br class="smart"></li></ul><hr><hr><h2><strong>■ 後記：集體智慧</strong></h2><p>行文前，幸好有先把本文打算用上的「架構圖」扔上 <a
