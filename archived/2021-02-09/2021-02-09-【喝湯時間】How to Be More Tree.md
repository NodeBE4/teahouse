<figcaption><span>Amazon.com</span></figcaption></figure><p><a href="https://www.google.com/url?sa=i&url=https%3A%2F%2Fwww.amazon.com%2FHow-More-Tree-Essential-Perennial%2Fdp%2F1912785137&psig=AOvVaw0_MlFr-Xj1BmQTOJo1qRfv&ust=1612973910291000&source=images&cd=vfe&ved=0CA0QjhxqFwoTCKiLiZqa3e4CFQAAAAAdAAAAABAc" target="_blank"><strong><em>How to be More Tree</em></strong></a> 的翻譯邀約找上門來時，我確實有點驚訝，因為簽下這本書的是三民書局。來信的編輯在信中說明，三民也開始涉足外文版權書，正在為 <em>How to be More Tree</em> 尋找譯者，而他們找我是因為看到我翻譯的《故道》。在這之前，也有作者因為看到《故道》的譯本，而直接找我洽談翻譯，我從《故道》的翻譯獲益匪淺，不過那並非我一人之力，大家出版社的編輯更是功不可沒。</p><p>回到<em> How to be More Tree</em> 。這本書是以一個跨頁為單位，一幅插畫搭配一段短文，講一個淺顯的道理。說起來篇幅短小，文句平順，翻譯難度並不高，只是譯文必須「優美」的要求多少還是費些思量。</p><p>譯稿進入編輯程序之後，三民出於市場的考量，希望把書名和篇名都改成比較勵志的「雞湯」風格。我好像能夠感覺到編輯來信詢問時的為難，可能編輯預期會遭遇譯者的「頑抗」吧，不過我回信表示尊重編輯的意見，同意了他們全部的變更。</p><p>這樣的事情若是放在十年前，我可能確實會不滿，會認為翻譯由我具名，我有權拒絕被改成心靈雞湯型的譯者。其實我今天還是覺得這個理由說得過去，但十多年的經驗讓我變得能夠少在乎自己一點。要清高儘可在自己獨自的領域清高，犯不上為了清高而令共事的人進退兩難。</p><p>《<a href="https://www.books.com.tw/products/0010878443" target="_blank"><strong>願我化作一棵樹</strong></a>》在去年年底出版了，不過編輯部忙亂了一陣子，現在才通知我，因此我也現在才知道。這本書的道理很淺顯，但是透過樹木來比喻表達，別具巧思，而且插圖真的很漂亮，就算不是來喝雞湯的，看看插圖，體會他人的哲思，也很有趣味。（或者也很適合送給喜歡喝雞湯的朋友。）</p><figure class="image">
