data-asset-id="59b2215c-5fba-4a15-9776-b24d2da5684e" referrerpolicy="no-referrer"><figcaption><span>《投資最重要的事：一本股神巴菲特讀了兩遍的書》</span></figcaption></figure><p>投資最重要的事是什麼？希望從《<a href="http://moo.im/a/krsyLN" rel="noopener noreferrer" target="_blank">投資最重要的事</a>》找到答案。此外，本人亦是加密貨幣市場的參與者，也好奇從這本書——是 <a href="https://g.co/kgs/EbEZgv" rel="noopener noreferrer" target="_blank">橡樹資本管理</a>（Oaktree Capital Management）創辦人 <a href="https://g.co/kgs/yDF7av" rel="noopener noreferrer" target="_blank">霍華・馬克斯</a>（Howard Marks）為了傳統金融投資人而寫——所學到的知識，究竟能不能在幣圈中也派上用場？一起來瞧瞧。</p><pre class="ql-syntax" spellcheck="false">說明①②③⋯⋯這些標號是本書章節，我試著用自己的脈絡把它們收束成四大類並串接起來，同時用區塊（block）疊加的方式，定期小結回顧。</pre><hr><hr><h2><strong>■ </strong>投資最重要的事</h2><p>2003 年 7 月，作者以「最重要的事」為標題寫了一份備忘錄，整理出他覺得成功投資該具備的要素，之後就圍繞這個概念發展成本書。談的是投資生涯中恪遵的信條（非工具書），有如路標一樣指引著他⋯⋯閱畢，我把書裡所觸及的面向區分為四大類：<strong>人的思維與行為、市場、風險、策略</strong>，依序探討如下：</p><h3><strong>１・人：心理因素在市場上扮演主要角色</strong></h3><p>開宗明義，書裡明確指出<strong>投資「不是」一門科學，因爲心理因素在市場上扮演主要角色</strong>，而且變動很大，導致因果關係的推演變得不可靠，這有違於科學的概念（可以控制變因，可以複製過去產生的結果，因果關係可靠不變）。</p><p>所以作者希望大家培養【① 第二層思考】的洞察力，<strong>此乃投資的藝術成份</strong>。呼應方才提到的心理因素，集結多數人的心理因素形成市場共識，而比市場共識更深入思考就是「第二層思考」的精髓：</p><pre class="ql-syntax" spellcheck="false">釋例◍ 第一層思考會說：「這是一家好公司，就買這支股票吧！」◍ 第二層思考則會說：「這是一家好公司，但每個人都認為這家公司很好，所以這不是好公司。這支股票的股價被高估，市價過高，所以賣出！」</pre><p>但作者也強調，不應該把「與眾不同」當成目標，<strong>而該把它當成一種思考方式</strong>，我想這應該就是所謂的 <a href="https://wiki.mbalib.com/zh-tw/%E9%80%86%E5%90%91%E6%80%9D%E7%BB%B4%E6%B3%95" rel="noopener noreferrer" target="_blank">逆向思考</a> 功夫吧！</p><blockquote>疊加區塊：<br class="smart">∵ 心理因素在市場上扮演主要角色<br class="smart">➨ 第二層思考 works</blockquote><p>繼續往下探，我還看見了 <a href="https://matters.news/@penfarming/225277-%E9%AB%98%E5%8B%9D%E7%AE%97%E6%B1%BA%E7%AD%96-%E9%81%8B%E7%94%A8%E4%B8%8B%E6%B3%A8%E6%80%9D%E7%B6%AD-%E6%89%93%E6%95%97%E4%B8%89%E5%80%8B%E5%A4%A7%E9%AD%94%E7%8E%8B-bafyreialokkxjr7n5xdeo3afmrzjkqht34bo3ditwah5iwjmw4ihyur754" rel="noopener noreferrer" target="_blank">機率思維</a>（下注思維），散見於【⑭ 認清預測的局限】與【⑯ 認識運氣所扮演的角色】兩章。</p><p>面對投資，我們得警覺那些號稱「神預測」的人（或自己萌生出這樣的想法），他們被作者歸類為「我知道」學派，根據一兩次結果，就斷定決策品質的好壞；其實不然，我們應該擁抱以《黑天鵝效應》一書聞名於世的 <a href="https://zh.m.wikipedia.org/zh-tw/%E7%B4%8D%E8%A5%BF%E5%A7%86%C2%B7%E5%B0%BC%E5%8F%AF%E6%8B%89%E6%96%AF%C2%B7%E5%A1%94%E9%9B%B7%E4%BC%AF" rel="noopener noreferrer" target="_blank">塔雷伯</a> 提出的「另類歷史」或「不確定世界」觀點，<strong>把未來世界視為一個隨機分布，並根據這個條件來投資</strong>。承認自己所知有限，理解到即使做出最好的決策，仍可能會輸掉比賽，<strong>因為任何結果都牽涉運氣</strong>，這種信念是作者所提倡的「我不知道」學派。</p><p>所以 <a href="https://g.co/kgs/EbEZgv" rel="noopener noreferrer" target="_blank">橡樹資本管理</a> 堅持要做好防禦性投資，並且要【 ⑱ 避開投資陷阱】，這當中有很高的比重是【⑩ 對抗情緒帶來的負面影響】，包含了人性中的貪婪、恐懼、質疑、從眾傾向、嫉妒、自負、投降心態等來自心理面（感性面）的因素。還有一件事情，<strong>沒有認識到市場周期和狂熱⋯⋯</strong>。</p><blockquote>疊加區塊：<br class="smart">∵ 心理因素在市場上扮演主要角色<br class="smart">➨ 第二層思考 works<br class="smart"><strong><u>並加入「我不知道」學派，派擁抱機率思維</u></strong><br class="smart"></blockquote><hr><h3><strong>２・市場：</strong>周期最後總是會占到優勢</h3><p>投資學中一個重要的理念叫做 <a href="https://zh.wikipedia.org/wiki/%E6%95%88%E7%8E%87%E5%B8%82%E5%A0%B4%E5%81%87%E8%AA%AA" rel="noopener noreferrer" target="_blank">效率市場假說</a>，這「效率」指的是資產價格會「迅速」反應市場對那些訊息的重要性所產生的共識，<strong>但作者不相信這個市場共識的看法必然是對的</strong>，此一脈絡依然是循著「心理因素在市場上扮演主要角色」而來；當【② 了解效率市場的局限】以後，我們知道無效率市場提供「錯誤定價」這個原料，所以才能藉著「第二層思考」允許一些人成為市場贏家，一些人成為輸家。</p><p>首要之務，是先【③ 準確估計實質價值】，簡稱估值。各種分析流派盡出，可簡化二分為技術面和基本面，而本書屬於後者⋯⋯總之，<strong>透過熟練的財務分析技巧，找出你對「價值」的堅定看法</strong>，能幫助你面對「價格」脫節的情況。</p><p>說到價格，是時候【④ 找出價格與價值的關係】了。再次回到「心理因素在市場上扮演主要角色」的出發點，它造成無效率市場與錯誤定價（價格與價值脫鈎才是常態），<strong>因此「第二層思考」裡必須納入「對其它投資人的洞察」</strong>然後做出因應。有時候，即使「價格」暗示你是錯的，也要勇於買進，因為你對「價值」有自己堅定不移的看法。</p><blockquote>疊加區塊：<br class="smart">∵ 心理因素在市場上扮演主要角色<br class="smart"><strong><u>➨ 無效率市場與錯誤定價（價格）</u></strong><br class="smart">➨ 第二層思考 works，<strong><u>洞察群眾心理面（價格）<br class="smart">➨ 比對自己堅定不移的（價值）看法，勇於買進</u></strong><br class="smart">並加入「我不知道」學派，派擁抱機率思維</blockquote><p>這裡和「股神」<a href="https://zh.m.wikipedia.org/zh-tw/%E6%B2%83%E4%BC%A6%C2%B7%E5%B7%B4%E8%8F%B2%E7%89%B9" rel="noopener noreferrer" target="_blank">巴菲特</a> 的思路略有不同，巴菲特投資風格是「買下好東西」偏向買進優質股權，如果以幣圈來比喻應是青睞比特幣、以太幣（雖然他本人對加密貨幣厭惡至極）；而 <a href="https://g.co/kgs/yDF7av" rel="noopener noreferrer" target="_blank">霍華・馬克斯</a> 則認為<strong>投資成功不是因為「買到好東西」而是因為「東西買的好」，</strong>所以你會在 <a href="https://g.co/kgs/EbEZgv" rel="noopener noreferrer" target="_blank">橡樹資本管理</a> 的資產組成看到很多「另類投資」，它從市場嫌惡的資產中撿便宜：</p><figure class="image"><img src="https://assets.matters.news/embed/969589db-a501-4223-a5e9-7e0e579e08b4.png" data-asset-id="969589db-a501-4223-a5e9-7e0e579e08b4" referrerpolicy="no-referrer"><figcaption><span>橡樹資本管理的資產組成有很多「另類投資」</span></figcaption></figure><p>我發現「心理因素在市場上扮演主要角色」大前提總是不斷示現，因為作者接下來拋出「周期」一說。<strong>只要有人類參與，結果就會有變動和周期更替</strong>，主要的理由是一般人有情緒，反覆無常，既不堅定也不客觀，所以沒有什麼東西可以永遠朝著同個方向前進，<strong>周期最後總是會占到優勢</strong>，造就了【⑧ 景氣循環】現象。</p><p>故【⑮ 察覺所在的景氣位置】是非常重要的事情，亦即測量市場的熱度，<strong>作者說投資市場會在ＯＯ與ＸＸ之間來回擺盪</strong>（ＯＯ與ＸＸ起碼可以套入 7 組詞彙）。我們必須【⑨ 意識到鐘擺效應】，這當中令我覺得描述地最傳神的一段是關於風險，<strong>風險是動態的</strong>。</p><blockquote>疊加區塊：<br class="smart">∵ 心理因素在市場上扮演主要角色<br class="smart">➨ 無效率市場與錯誤定價（價格）<br class="smart">➨ 第二層思考 works，洞察群眾心理面（價格）<br class="smart"><strong><u>➨ 投資市場就像鐘擺一樣，周期最後總是會占到優勢</u></strong><br class="smart">➨ 比對自己堅定不移的（價值）看法，勇於買進<br class="smart"><strong><u>➨ 比起「買到好東西」更青睞「東西買的好」</u></strong><br class="smart">並加入「我不知道」學派，擁抱機率思維</blockquote><hr><h3><strong>３・風險：在容忍與趨避之間擺盪</strong></h3><p>這個部分應該是本書最顛覆的一段。</p><p>學界認為，投資人面對風險的態度是固定不變，但作者不以為然，他以為<strong>實際上會是在風險容忍與風險趨避之間擺盪才對</strong>；學界也喜歡用「波動性」來衡量風險（這段令我深有所感，研讀過一堆用以標準差、變異數作為風險代理變數的論文），作者再度駁斥，不認為大多數投資人關心「波動性」這種風險，<strong>最重要的風險就是虧錢的可能性</strong>，直截了當。</p><p>以上是【⑤ 理解風險】，定義清楚才能【 ⑥ 確認風險】。</p><p><strong>當投資人高價進場不知規避時，這就是風險的主要來源，也是泡沫的必然特點；</strong>相信風險很低，或甚至相信「這次不一樣」風險會完全消失，是風險產生的一個主要因素，畢竟「周期最後總是會占到優勢」嘛；當投資人不害怕風險，他們會在不要求補償報酬的情況下接受風險；當提供資金的人風險趨避不足，他們放鬆借款條件與利率，而不是要求足夠的保護或預期報酬⋯⋯歸結來說，<strong>市場上增加的信心，反而會創造更多應該擔憂的風險，作者把這個現象稱之為「風險的任性」</strong>，反之亦然。</p><p>而風險不能消除，只能移轉和分散，這是【⑦ 控制風險】，在在影響了 <a href="https://g.co/kgs/EbEZgv" rel="noopener noreferrer" target="_blank">橡樹資本管理</a> 的投資策略。</p><blockquote>疊加區塊：<br class="smart">∵ 心理因素在市場上扮演主要角色<br class="smart">➨ 無效率市場與錯誤定價（價格）<br class="smart">➨ 第二層思考 works，洞察群眾心理面（價格）<br class="smart"><strong>➨ </strong>投資市場就像鐘擺一樣，<strong><u>在風險容忍與風險趨避間擺盪<br class="smart">➨ 市場上增加的信心（風險容忍），反而會創造更多應該擔憂的風險<br class="smart">➨ 市場上增加的恐懼（風險趨避），反而會減少風險</u></strong><br class="smart">➨ 周期最後總是會占到優勢<br class="smart">➨ 比對自己堅定不移的（價值）看法，勇於買進<br class="smart">➨ 比起「買到好東西」更青睞「東西買的好」<br class="smart">並加入「我不知道」學派，擁抱機率思維<br class="smart"></blockquote><hr><h3><strong>４・策略：安全邊際，保持懷疑</strong></h3><pre class="ql-syntax" spellcheck="false">投資無法掌控的部分叫做 Beta，與市場大環境有關；與之相對的是 Alpha，也就是投資人個別技巧。</pre><p>先前提到 <a href="https://g.co/kgs/EbEZgv" rel="noopener noreferrer" target="_blank">橡樹資本管理</a> 是「我不知道」學派的一員，<strong>明白投資有太多事情（Beta）無法掌控，投資人只掌控一部分（Alpha）的投資結果</strong>，所以【⑰ 採取防禦型投資策略】，這種風格強調「不是把事情做對，而是不做錯的事情」。對投資感到敬畏恐懼，戒慎行事少點犯錯，但也寫不出偉大的詩歌。追求平均上壘率，而不求揮大棒（三振率也高）擊出全壘打。</p><p>在防禦型策略底下，<strong>多頭行情時只要有平均報酬就好了，但在空頭行情的時候一定要擊敗大盤</strong>，優異的 Alpha 創造這種「績效表現的不對稱」，是我們想要的不對稱（上漲賺得績效比下跌產生的虧損幅度還大），是讓投資組合【⑲ 增加價值】的來源。</p><p>因此作者相當推崇「安全邊際」的概念，【⑫ 找出便宜標的】與【⑬ 耐心等待時機】，他買進標的物的作風，像是（美劇中）見到有住戶要喬遷，於是有一堆「待售清單」擺設在路邊，再從中挑選入手。而不是帶著「購物清單」只想持有某些特定商品。<strong>絕佳的買進機會出現在資產持有人被迫要賣出的時候</strong>，而這時候你要處在買家的位置。</p><blockquote>疊加區塊：<br class="smart">∵ 心理因素在市場上扮演主要角色<br class="smart">➨ 無效率市場與錯誤定價（價格）<br class="smart">➨ 第二層思考 works，洞察群眾心理面（價格）<br class="smart">➨ 投資市場就像鐘擺一樣，在風險容忍與風險趨避間擺盪<br class="smart">➨ 市場上增加的信心（風險容忍），反而會創造更多應該擔憂的風險<br class="smart">➨ 市場上增加的恐懼（風險趨避），反而會減少風險<br class="smart">➨ 周期最後總是會占到優勢<br class="smart">➨ 比對自己堅定不移的（價值）看法，勇於買進<br class="smart">➨ 比起「買到好東西」更青睞「東西買的好」<br class="smart"><br class="smart">並加入「我不知道」學派，擁抱機率思維<br class="smart">➨ 防禦性投資策略<br class="smart"><strong><u>➨ 推崇「安全邊際」</u></strong><br class="smart"></blockquote><p>除了基防禦型投資策略，作者也奉行【⑪ 反向投資】策略，這是基於「第二層思考」的衍生，該做的事不只是與群眾做出相反的事，<strong>還知道群眾為什麼是錯的</strong>？</p><p>這亦是一種時刻保持懷疑的態度，對於「好到不像是真的」與「壞得不像是真的」都提出質疑；對於報酬有【⑳ 合理預期】，<strong>可以養成習慣自問「為什麼是我得到這個好運」</strong>而不是別人得到？憑什麼掙得超出市場行情的報償？應該可以避開不少投資雷區，尤其是幣圈常見的資金盤詐騙手法，現實社會中也是。</p><blockquote>疊加區塊：<br class="smart">∵ 心理因素在市場上扮演主要角色<br class="smart">➨ 無效率市場與錯誤定價（價格）<br
