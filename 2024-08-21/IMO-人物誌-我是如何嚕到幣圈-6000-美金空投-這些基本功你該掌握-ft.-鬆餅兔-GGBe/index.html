    ❏ 我是如何嚕到幣圈 6000 美金空投，這些基本功你該掌握！

    ▍何時入幣圈？

    GGBear 是在 2021 年受到「NFT 狂熱」的吸引，而加入幣圈。


    ▍養成三個習慣防詐騙

    GGBear 把他的「幣圈基本面研究」用標準化方式寫下來，除了有助於系統性學習，還可以預防詐騙！

    因為過程中，他已經養成這些習慣，三層防護：

    • 避免直接透過 Google 找資料。

    • CoinMarketCap 和 DefiLlama 就是我的搜尋引擎！

    • 再三確認網址是否有誤。

    案例說明，下圖為模仿正版網址（PancakeSwap）中 nc 對調以後的山寨網址。


    ▍GGBear 的幣圈基本面研究 SOP

    1. 選擇可信任的加密貨幣資訊平台

    2. 研究專案官網（Docs 文件）

    3. 查看代幣經濟模型（尤其是代幣解鎖）

    4. 使用鏈上數據平台（DeFi 資料庫）


    ▍1. 選擇可信任的加密貨幣資訊平台

    這三個加密貨幣資訊平台功能差不多，都很好用：

    • CoinMarketCap

    • CoinGecko

    • CoinCarp

    除了查代幣也能查交易所！

    例如下圖，進入 CoinMarketCap → Exchanges → Decentralized Exchanges → Spot → PancakeSwap v3 (BSC)。

    從這裡找到並進入 PancakeSwap 官網是比較安全的做法。

    CoinCarp 特有的 Rich List 功能，可查詢某個代幣的「前Ｏ大持有者佔比」，是區塊鏈專案「去中心化程度」的參考指標。

    例如下圖，Cardano（ADA）由 3,577,963 個錢包地址所持有，其中前百大持有者佔據了代幣市場流通量的 22.16%。

    延伸閱讀：投資幣圈必看！全網最詳細 CoinMarketCap 與 CoinGecko 完整教學


    ▍2. 研究專案官網（Docs 文件檔）

    通常可以在專案官網找到 Docs 文件檔，它讓用戶、開發者、投資者或合作夥伴能夠理解該專案的目標、運作、技術細節以及使用方法。

    例如下圖是 PancakeSwap 官方文檔的路徑。


    ▍3. 查看代幣經濟模型（尤其是代幣解鎖）

    如果專案有發幣，官方文檔裡一定找得到關於「代幣經濟學」（Tokenomics）的說明，這很重要。

    例如下圖是 PancakeSwap 代幣經濟學。

    也搭配 TokenUnlocks 專門查看「代幣解鎖」時間表。

    • 右上角註冊並登入，取得較高權限。

    • 右上角搜尋代幣名稱

    • 往下滑找到 Upcoming Event

    • 往下滑找到 Vesting Schedule

    例如下圖顯示 Celestia（TIA） 71 天後（2024 年 10 月 31 日）迎來鉅額——相當於當前流通量的 85%——代幣解鎖，對照視覺化時間軸會更有感。

    先不論其他條件，這麼大的 TIA 供給，帶來的稀釋效果相當於你的代幣價格被打了 54 折（1/1.85）。


    ▍4. 使用鏈上數據平台（DeFi 資料庫）

    —— Etherscan：公鏈瀏覽器始祖

    延伸閱讀：Etherscan 是什麼？區塊鏈交易查詢、追蹤大戶、項目方動態就靠它！


    —— DeFiLlama：資訊最完整的一站式數據工具

    ▷ 有多少資金被用戶存入或質押在一個 DeFi 協議中？

    1. DeFi → Overview

    2. 總鎖定價值（Total Value Locked, TVL）

    3. 觀察 TVL 時，通常會搭配穩定幣（Stablecoins）一起看

    4. 切換至個別公鏈（Chaines）

    5. 公鏈上部署各式各樣的協議，它們的排名（Protocol Ranking）

    延伸閱讀：DeFi 是什麼？會取代傳統金融？Defi 運作方式與 4 大應用場景一次看！


    ▷ 鏈上交易活絡嗎？

    1. Volumes → Chains

    2. GGBear 習慣抓「近一天交易量＞10m」或「近七天交易量＞50m」做門檻值，初步通過篩選才有關注的必要。

    ▷ 觸動我的雷達

    有沒有不尋常的（Weekly change）增幅？

    如果有，查查看這條公鏈最近有啥新鮮事？

    於是 GGBear 決定埋伏 ZKsync Era 的基礎建設，他以「鏈上放貸」（Lending）的方式參與，存入 0.5 顆以太幣，最後拿到等值 6000 美元的空投代幣 ZK。
