<p>四月底從友人那邊接收一張 <a href="https://stepn.com/" rel="noopener noreferrer" target="_blank">STEPN</a> NFT。名曰「跑鞋」，實際上可以用走的——因為鞋子有分四種類型，我拿到是「走鞋」（walker）——也幸虧用走的就好，稍後說分明。今天就來聊聊這陣子的體驗吧！</p><figure class="image"><img src="https://assets.matters.news/embed/5ebe824b-935d-48d4-8d56-455f31f0a1ac.png" data-asset-id="5ebe824b-935d-48d4-8d56-455f31f0a1ac" referrerpolicy="no-referrer"><figcaption><span>鞋子的四種類型｜STEPN 官網</span></figcaption></figure><hr><hr><h2>■ 在幣圈，我想「腳踏實地」的賺錢</h2><p>簡單描繪一下這款 Move-to-Earn（M2E）的 <a href="https://wiki.rugdoc.io/docs/what-is-gamefi-play-to-earn-and-blockchain-games-explained/" rel="noopener noreferrer" target="_blank">GameFi</a> 區塊鏈遊戲。</p><p>首先你必須在手機上先安裝 STEPN App（<a href="https://apps.apple.com/tw/app/stepn/id1598112424" rel="noopener noreferrer" target="_blank">iOS</a>／<a href="https://play.google.com/store/apps/details?id=com.bcy.fsapp&hl=zh_TW&gl=US" rel="noopener noreferrer" target="_blank">Android</a>），它也是一款 <a href="https://zh.m.wikipedia.org/zh-tw/%E5%AF%86%E7%A0%81%E8%B4%A7%E5%B8%81%E9%92%B1%E5%8C%85" rel="noopener noreferrer" target="_blank">熱錢包</a>（Hot Wallet）。為它儲值足夠的加密貨幣，可能是 SOL 或 BNB，端看你想要在 Solana 或 BEP20 哪一條鏈上落腳？來到 App 內商城，用這些幣為自己買一雙鞋（買一張 NFT），也預留一些幣做為 <a href="https://academy.binance.com/zt/articles/what-are-blockchain-transaction-fees" rel="noopener noreferrer" target="_blank">區塊鏈交易手續費</a>——這時已經完成所有前置作業，可以開始「腳踏實地」的賺錢了（笑）。</p><figure class="image"><img src="https://assets.matters.news/embed/50909a0b-3edc-4dfd-9137-f5e09b29c67f.png" data-asset-id="50909a0b-3edc-4dfd-9137-f5e09b29c67f" referrerpolicy="no-referrer"><figcaption><span>是鏈遊 App 也是熱錢包</span></figcaption></figure><p>要玩這款手遊——或說是啟動這張 NFT 的「賦能」——相當容易，只要按下主畫面上的 START 按鈕，就會啟動猶如動漫《火影忍者》中 <a href="https://baike.baidu.hk/item/%E5%85%AB%E9%96%80%E9%81%81%E7%94%B2/13210884" rel="noopener noreferrer" target="_blank">李洛克之八門遁甲</a> 的倒數計時。</p><p>只見螢幕上有個儀表板，你必須維持移動速度（依據不同種類鞋子）落在合規的綠色區段中，<strong>將為你掙取 GST 代幣，是可以賣成「錢」的幣</strong>。時間耗盡後（能量、體力用光的意思），記得 STOP 你的鞋，因為它有耐久值消耗／維修的設定。每隔六小時會恢復 25% 能量，若想擴增能量槽上限，得成為蜈蚣，多買幾雙鞋子。</p><figure class="image"><img src="https://assets.matters.news/embed/62c86e95-8756-488d-937a-e1a2a508dc4b.png" data-asset-id="62c86e95-8756-488d-937a-e1a2a508dc4b" referrerpolicy="no-referrer"><figcaption><span>進入「八門遁甲」時間倒數</span></figcaption></figure><p>完事後，輕鬆曬出遊戲幫你記錄好的 <a href="https://zh.wikipedia.org/zh-tw/%E5%85%A8%E7%90%83%E5%AE%9A%E4%BD%8D%E7%B3%BB%E7%BB%9F" rel="noopener noreferrer" target="_blank">GPS</a> 足跡至社群上，足見其 <a href="https://golden.com/wiki/SocialFi-GZ6NRPN" rel="noopener noreferrer" target="_blank">SocialFi</a> 的潛力。</p><p>＊</p><p>先前提到「幸虧用走的就好」一事，<strong>理由是台北多雨</strong>。有時雨下一整天，這情況下出門，用走的還可以接受，沿著騎樓前行就穩妥。但若遇上滂薄大雨，還得讓人跑起來，那真是有點吃不消啊！</p><hr><hr><h2>■ 為 GameFi 而生的 NFT，觀察與思考</h2><p>非單指個別項目，想討論的是我從 <a href="https://stepn.com/" rel="noopener noreferrer" target="_blank">STEPN</a> 觀察到什麼？這種為 <a href="https://wiki.rugdoc.io/docs/what-is-gamefi-play-to-earn-and-blockchain-games-explained/" rel="noopener noreferrer" target="_blank">GameFi</a> 應運而生的 NFT 帶給我的思考，環繞著「賦能」來寫：</p><p>＊</p><p><strong>✅ 先天上的賣壓，如何緩解？</strong></p><p>所有主打 Play-to-Earn 的鏈遊，<strong>先天上勢必要面對很大一部分為了 Earn 而來的人群</strong>，所產生的「挖提賣」賣壓。如何緩解這股「供應面」力道？<strong>須打造能與之抗衡的「需求面」因應</strong>，是每個項目方在設計 <a href="https://coinmarketcap.com/alexandria/article/what-is-tokenomics" rel="noopener noreferrer" target="_blank">Tokenomics</a> 時，不可不慎的一點。</p><p>以 <a href="https://stepn.com/" rel="noopener noreferrer" target="_blank">STEPN</a> 來說，目前掙來的 GST 代幣，除了賣掉，能拿來用的情境有鑄鞋、升級鞋、開寶箱、合寶石、轉讓手續費⋯⋯等。用途似乎仍嫌不足，需要更多「消費 GST」的場景應用，或許在那些 coming soon 身上可找到答案。</p><p>＊<br class="smart"></p><p><strong>✅ 搭上「認知協調」順風車</strong></p><p>明知道運動有益身心健康，但為什麼這樣的「好習慣」如此難培養？</p><p>長銷書《<a href="https://matters.news/@penfarming/118443-%E5%B7%A6%E6%89%8B%E5%8F%AA%E6%98%AF%E6%94%AF%E6%92%90-%E4%B8%89%E5%BC%B5%E5%9C%96%E7%9C%8B-%E5%8E%9F%E5%AD%90%E7%BF%92%E6%85%A3-bafyreiboc7widlepeeh2jzltkcs4f5dqipb5z46d2p2kh4vrthpfv6nvi4" rel="noopener noreferrer" target="_blank">原子習慣</a>》告訴我們，因為「好習慣」帶來的效益不是立竿見影（需跨過某個門檻才會顯現），很多人捱不到看見成效的那一天，深陷 <a href="https://matters.news/@penfarming/118443-%E5%B7%A6%E6%89%8B%E5%8F%AA%E6%98%AF%E6%94%AF%E6%92%90-%E4%B8%89%E5%BC%B5%E5%9C%96%E7%9C%8B-%E5%8E%9F%E5%AD%90%E7%BF%92%E6%85%A3-bafyreiboc7widlepeeh2jzltkcs4f5dqipb5z46d2p2kh4vrthpfv6nvi4" rel="noopener noreferrer" target="_blank">失望之谷</a> 於是半途而廢。</p><figure class="image"><img src="https://assets.matters.news/embed/e18c5b4b-e714-48a8-9236-99036e3a1932.jpeg" data-asset-id="e18c5b4b-e714-48a8-9236-99036e3a1932" referrerpolicy="no-referrer"><figcaption><span>失望之谷｜《原子習慣》書中插圖</span></figcaption></figure><p>我覺得 <a href="https://stepn.com/" rel="noopener noreferrer" target="_blank">STEPN</a> 鏈遊存在一個優勢，它能跟大家「想要」養成的習慣綁在一起，甚至再激進一些，你不可能不走路吧！於是這「想要」有望成為一種「必要」。</p><p>總之 <a href="https://stepn.com/" rel="noopener noreferrer" target="_blank">STEPN</a> 為 Move 這件事情添具紮實的誘因，<strong>創造立即性、可視化的愉悅感受（誰不愛錢？）</strong>——就算是微小的——是給大腦的信號，當體驗令人滿足，我們比較有可能重複某一行動，尤其當這個行動是普遍認知上的「想要」或「必要」時。</p><p>玩別的手遊，即使也能賺錢，但潛意識會告訴你：「一直這樣久坐很不健康啊！」或多或少都承受了這種矛盾，心理學上的 <a href="https://zh.m.wikipedia.org/zh-tw/%E8%AA%8D%E7%9F%A5%E5%A4%B1%E8%AA%BF" rel="noopener noreferrer" target="_blank">認知失調</a>（cognitive dissonance）。</p><p>但玩 <a href="https://stepn.com/" rel="noopener noreferrer" target="_blank">STEPN</a> 少了這一層負擔，<strong>它是一張認知協調的 NFT</strong>。</p><p>＊</p><p><strong>✅ 遊戲性／社群性</strong></p><p>一款遊戲（鏈遊）能走得長遠，必須要有足夠的<strong>遊戲性</strong>。</p><p>什麼是遊戲性呢？我想從反面來論述，當一款遊戲有太多外掛，進入後充斥著各種作弊仔、掛機流⋯⋯不久的將來，應該就走向窮途末路了吧！</p><p>我不知道 <a href="https://stepn.com/" rel="noopener noreferrer" target="_blank">STEPN</a> 是否能松柏長青？但我知道官方 <a href="https://www.blocktempo.com/stepn-model-for-anti-cheating-smac/" rel="noopener noreferrer" target="_blank">打擊作弊</a> 不遺餘力。另外是關於鞋子的發展流派（請想像電玩《<a href="https://zh.wikipedia.org/zh-tw/%E6%9A%97%E9%BB%91%E7%A0%B4%E5%9D%8F%E7%A5%9E%E7%B3%BB%E5%88%97" rel="noopener noreferrer" target="_blank">暗黑破壞神</a>》的技能樹）有一定複雜度，<strong>建置上不會太無腦，不無腦就不無聊，不無聊就有得聊</strong>，於是你可以看到各種類似於 <a href="https://www.gamer.com.tw/" rel="noopener noreferrer" target="_blank">巴哈姆特</a> 的討論集散地。</p><p>這又為遊戲帶來了<strong>社群性</strong>，這不就是大 Web3 時代——尤其是 NFT 專案——人人都琅琅上口的「凝聚共識價值」嗎？</p><p>＊</p><p><strong>✅ 虛實結合到線下</strong></p><p>遊戲的「虛實結合」不是什麼大新聞，應該是從早期 <a href="https://zh.wikipedia.org/wiki/Wii" rel="noopener noreferrer" target="_blank">Wii</a> 主機的「體感」玩法開始的，到近年來（對我而言）印象比較深刻的 <a href="https://zh.wikipedia.org/wiki/Nintendo_Switch" rel="noopener noreferrer" target="_blank">Switch</a> 主機、<a href="https://zh.wikipedia.org/wiki/Pok%C3%A9mon_GO" rel="noopener noreferrer" target="_blank">Pokémon GO</a> 風潮⋯⋯咦！怎麼都是任天堂（笑）。<br class="smart"></p><p>先是鼓勵玩家屁股離開座椅就好，再到走出戶外，這是<strong>場景</strong>上的「虛與實」。</p><p>另外一種「虛與實」是指<strong>金流</strong>，如何讓 <a href="https://stepn.com/" rel="noopener noreferrer" target="_blank">STEPN</a> 逐漸擺脫這是一場龐氏騙局——後金補前金——的質疑？<strong>那就是創造更多金，除了「虛」幣以外的「實打實」金流</strong>。目前我見過 <a href="https://news.cnyes.com/news/id/4851972" rel="noopener noreferrer" target="_blank">STEPN 與知名品牌 ASICS（亞瑟士）的聯名合作</a> 計畫，至少先起了頭，希望是好彩頭。</p><hr><hr><h2>■ 尾聲</h2><p>很多人唱衰 <a href="https://stepn.com/" rel="noopener noreferrer" target="_blank">STEPN</a> 會是繼 LUNA 之後，下一個進入「死亡螺旋」的案例。</p><p>但不管怎樣，這位引爆話題，成功帶動「<a href="https://baike.baidu.hk/item/%E5%87%BA%E5%9C%88/22956271" rel="noopener noreferrer" target="_blank">出圈</a>」的大前輩，從它身上可以觀察到不少值得借鏡之處，特別是想做 GameFi NFT 的項目方。</p><p>無論最終成敗，它的每一步都將成為後繼者的養分。</p><hr><hr><figure class="image"><img src="https://assets.matters.news/embed/235d2f9b-ab0a-4fe2-8ead-49b2b898ca47.png"
