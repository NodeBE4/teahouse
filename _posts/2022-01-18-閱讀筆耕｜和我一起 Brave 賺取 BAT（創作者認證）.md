<p>承先前介紹 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E4%B8%80%E8%88%89%E4%B8%89%E5%BE%97%E7%9A%84-presearch-%E4%BD%BF%E7%94%A8%E5%BF%83%E5%BE%97-%E9%99%84%E6%95%99%E5%AD%B8-bafyreiec7mjtpjcy6bre3fgn7k53pmapqtnbziksneu6pamuvjqnbhcg5y" rel="noopener noreferrer" target="_blank">Presearch</a> 這款「邊搜邊賺（Search to Earn）」的搜尋引擎，有市民建議搭配 Brave 瀏覽器一起使用，真的是最佳拍檔，附上體驗一個多月的心得。</p><blockquote>⚠️ Brave 瀏覽器官網 <a href="https://brave.com/zh/download/" rel="noopener noreferrer" target="_blank">下載</a> 連結。<br class="smart">⚠️ Presearch 搜尋引擎 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E4%B8%80%E8%88%89%E4%B8%89%E5%BE%97%E7%9A%84-presearch-%E4%BD%BF%E7%94%A8%E5%BF%83%E5%BE%97-%E9%99%84%E6%95%99%E5%AD%B8-bafyreiec7mjtpjcy6bre3fgn7k53pmapqtnbziksneu6pamuvjqnbhcg5y" rel="noopener noreferrer" target="_blank">教學文</a> 連結。<br class="smart"></blockquote><hr><hr><h2><strong>■ 跳槽 Brave 迎來四好</strong></h2><p>以下是歸納出來，跳槽到 Brave 帶給我的好處：</p><ul><li><strong>節省效能，支援擴充</strong><br class="smart">原先是 Chrome 用戶，執行一陣子後，整台筆電會「鈍鈍的」很吃資源（不過我的 Macbook 也有點年紀）；換至 Brave 後，狀況緩解不少，而且也可（支援安裝）繼續沿用 <a href="https://chrome.google.com/webstore/category/extensions?hl=zh-TW" rel="noopener noreferrer" target="_blank">Chrome 線上應用程式商店</a> 的擴充功能。<br class="smart"></li><li><strong>內建廣告、追蹤器阻擋</strong><br class="smart">受惠於彈出式視窗的（自動）隔絕，現在比較樂於逛 <a href="https://www.potatomedia.co/user/a2a5c973-e410-4525-b94b-420ffe385348" rel="noopener noreferrer" target="_blank">Potato Media</a> 與 <a href="https://tw.appledaily.com/home/" rel="noopener noreferrer" target="_blank">蘋果新聞網</a>。被屏蔽的項目通通都進到網址欄右側的「獅子口」裡，療癒。<br class="smart"></li></ul><figure class="image"><img src="https://assets.matters.news/embed/87ad09b6-6161-47c1-8e15-aa2c3994c8a7.png" data-asset-id="87ad09b6-6161-47c1-8e15-aa2c3994c8a7" referrerpolicy="no-referrer"><figcaption><span>獅子大開口，猛吃彈跳窗</span></figcaption></figure><ul><li><strong>賺取加密貨幣（主動）</strong><br class="smart">Brave 把觀看廣告的決定權還給使用者，可以在後台（找到 Brave Rewards 選單）調節其出現頻率（每小時 0 則 ～ 每小時 10 則）以賺取名曰「<a href="https://www.coingecko.com/zh-tw/%E6%95%B8%E5%AD%97%E8%B2%A8%E5%B9%A3/%E6%B3%A8%E6%84%8F%E5%8A%9B%E5%B9%A3" rel="noopener noreferrer" target="_blank">注意力代幣（Basic Attention Token, BAT）</a>」的加密貨幣。搭配 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E4%B8%80%E8%88%89%E4%B8%89%E5%BE%97%E7%9A%84-presearch-%E4%BD%BF%E7%94%A8%E5%BF%83%E5%BE%97-%E9%99%84%E6%95%99%E5%AD%B8-bafyreiec7mjtpjcy6bre3fgn7k53pmapqtnbziksneu6pamuvjqnbhcg5y" rel="noopener noreferrer" target="_blank">Presearch</a> 搜尋引擎，不佔用額外時間，一次同時農（Farm）兩種代幣，甚好。<br class="smart"><br class="smart">就我的經驗，壓根兒感受不到那些廣告的出現，更遑論有「觀看」過。</li></ul><figure class="image"><img src="https://assets.matters.news/embed/6a9439b9-7a70-4b1d-b0b9-4e26b181d074.png" data-asset-id="6a9439b9-7a70-4b1d-b0b9-4e26b181d074" referrerpolicy="no-referrer"><figcaption><span>決定廣告（ Brave Ads ）出現頻率</span></figcaption></figure><ul><li><strong>受賞加密貨幣（被動）<br class="smart"></strong>方才提到的 BAT 加密貨幣，送禮自用兩相宜，還能用來打賞別人。具體的說，是打賞具有「<strong>藍勾勾（已驗證）</strong>」的創作者（站台），稍後詳談。</li></ul><figure class="image"><img src="https://assets.matters.news/embed/4a70f343-4939-43b8-b219-6fe58e54cc8b.png" data-asset-id="4a70f343-4939-43b8-b219-6fe58e54cc8b" referrerpolicy="no-referrer"><figcaption><span>送禮自用兩相宜的 BAT 代幣</span></figcaption></figure><hr><hr><h2><strong>■ 優化 Brave 設定</strong></h2><p>由於當初跑（註冊）流程時忘記截圖，沒辦法細到每一動都演示，僅勾勒大方向。</p><p>.</p><p>✅<strong> 設定 → 錢包</strong></p><p>瀏覽器本身<strong>內嵌</strong>一個叫做「Brave Wallet」的錢包，不好用，建議斷開與它的關聯。</p><figure class="image"><img src="https://assets.matters.news/embed/dbd6be0e-4c50-4371-b7bf-0e187e46edff.png" data-asset-id="dbd6be0e-4c50-4371-b7bf-0e187e46edff" referrerpolicy="no-referrer"><figcaption><span>錢包</span></figcaption></figure><p>.</p><p>✅<strong> 設定 → 搜尋引擎</strong></p><p>把 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E4%B8%80%E8%88%89%E4%B8%89%E5%BE%97%E7%9A%84-presearch-%E4%BD%BF%E7%94%A8%E5%BF%83%E5%BE%97-%E9%99%84%E6%95%99%E5%AD%B8-bafyreiec7mjtpjcy6bre3fgn7k53pmapqtnbziksneu6pamuvjqnbhcg5y" rel="noopener noreferrer" target="_blank">Presearch</a> 定義為預設（網址列中使用的）搜尋引擎，已收開挖 <a href="https://www.coingecko.com/zh-tw/%E6%95%B8%E5%AD%97%E8%B2%A8%E5%B9%A3/presearch" rel="noopener noreferrer" target="_blank">PRE</a> + <a href="https://www.coingecko.com/zh-tw/%E6%95%B8%E5%AD%97%E8%B2%A8%E5%B9%A3/%E6%B3%A8%E6%84%8F%E5%8A%9B%E5%B9%A3" rel="noopener noreferrer" target="_blank">BAT</a> 兩個願望一次滿足之效果。</p><figure class="image"><img src="https://assets.matters.news/embed/9a0dd752-b178-44fa-8577-a8679262b25f.png" data-asset-id="9a0dd752-b178-44fa-8577-a8679262b25f" referrerpolicy="no-referrer"><figcaption><span>改以 Presearch 做為預設搜尋引擎</span></figcaption></figure><p>.</p><p>✅<strong> 設定 → 同步</strong></p><p>假如你在多台裝置使用 Brave，至少希望「書籤」是同步的，必須靠「同步代碼」來實現。</p><figure class="image"><img src="https://assets.matters.news/embed/2fb55d05-94b0-4aa3-bf63-b3fe8cc65b60.png" data-asset-id="2fb55d05-94b0-4aa3-bf63-b3fe8cc65b60" referrerpolicy="no-referrer"><figcaption><span>多台裝置同步</span></figcaption></figure><p>.</p><p><strong>✅ 設定 → Brave Rewards</strong></p><p>決定廣告（ Brave Ads ）出現頻率，不再贅述。</p><p>這裡還有一件不急，但遲早得做的事情是「<strong>註冊一個收受 BAT 的錢包</strong>」，只能從 <a href="https://uphold.com/" rel="noopener noreferrer" target="_blank">Uphold</a> 和 <a href="https://exchange.gemini.com/" rel="noopener noreferrer" target="_blank">Gemini</a> 擇一（我選擇後者）。為什麼說不急呢？若遲遲未申請，該給你的 BAT（不論是主動／被動得到的）會先以「記帳」方式呈現，不會消失便是。</p><figure class="image"><img src="https://assets.matters.news/embed/16aae483-1eef-4984-b5e5-7d457f1656c1.png" data-asset-id="16aae483-1eef-4984-b5e5-7d457f1656c1" referrerpolicy="no-referrer"><figcaption><span>收受 BAT 的錢包：Uphold 和 Gemini 二選一</span></figcaption></figure><p>.</p><p><strong>✅ 創作者認證</strong></p><p>這是進階選項，適合在各大自媒體平台活躍，尤其是「自架站」創作者使用，請先至 <a href="https://creators.brave.com/" rel="noopener noreferrer" target="_blank">Brave Creator 網站</a> 註冊成會員，之後便可以在 <a href="https://publishers.basicattentiontoken.org/publishers/home" rel="noopener noreferrer" target="_blank">Brave Creator 後台</a> 連結錢包、添加站台（Add Channel）成為受賞管道，目前總共支援七種平台。</p><figure class="image"><img src="https://assets.matters.news/embed/6aafa9ef-d287-49d3-9128-e6d871be0798.png" data-asset-id="6aafa9ef-d287-49d3-9128-e6d871be0798" referrerpolicy="no-referrer"><figcaption><span>添加站台，進行創作者認證，支援七種平台</span></figcaption></figure><p>我添加自己的 <a href="https://www.youtube.com/channel/UCDCPZtF7qUb75rZwIHcdY-g" rel="noopener noreferrer" target="_blank">YouTube</a> 與 <a href="https://twitter.com/leo7283" rel="noopener noreferrer" target="_blank">Twitter</a> 帳戶作為測試，前者很快地就出現「藍勾勾」表示創作者已認證，站台可以順利收受他人的（BAT）打賞；不知原因，後者卡關，正好作為對照組，未認證狀態下，那些小費會先以「記帳」形式擱置著。</p><figure class="image"><img src="https://assets.matters.news/embed/b1b9e47a-cd49-4c31-9a9d-7cf75c709d57.png" data-asset-id="b1b9e47a-cd49-4c31-9a9d-7cf75c709d57" referrerpolicy="no-referrer"><figcaption><span>實驗組：已認證（藍勾勾）</span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/256e7e4e-3bdd-4d8d-87a1-83b1a8ff007c.png" data-asset-id="256e7e4e-3bdd-4d8d-87a1-83b1a8ff007c" referrerpolicy="no-referrer"><figcaption><span>對照組：未認證</span></figcaption></figure><hr><hr><h2><strong>■ 尾聲：和我一起勇敢</strong></h2><p>換用 Brave 瀏覽器並搭配 <a
