👈</p><hr><hr><p>比特幣也能發 NFT？</p><p>比特幣是區塊鏈技術的創始應用，初衷是建立一個安全且去中心化的電子交易系統，讓人們可以在不需要信任第三方的情況下進行轉帳。<strong>然而「Ordinals 協議」的問世，讓使用者能把 NFT 放在比特幣區塊鏈上</strong>，這樣的體驗讓比特幣有了新的應用方式，但也拉高了網路交易成本。</p><p>在本文中，我們將深入探討比特幣 NFT 的運作原理，包括序數理論（Ordinal Theory）和銘文（Inscriptions）。也將探討比特幣 NFT 的特色，以及與以太坊 NFT 的不同之處，最後列舉了一些熱門的比特幣 NFT 項目。</p><hr><hr><h2>▇ <strong>比特幣 NFT／比特幣 Ordinals 協議是什麼？</strong></h2><p><strong>比特幣 NFT，又稱作比特幣 Ordinals 協議，是一種在比特幣區塊鏈上創造數位資產的方法</strong>。</p><p>2023 年 1 月，由軟體工程師 Casey Rodarmor 在比特幣區塊鏈上打造的 Ordinals NFT 協議正式亮相，允許使用者在比特幣的最小單位 Satoshi（SAT） 上創建獨一無二的 NFT。</p><p>Ordinals NFT 協議按照 SAT 因挖礦被開採出來的順序進行編號，支持在 SAT 上寫入（write in）文字、圖片、影音等內容來創建獨特風格的 NFT，且可以透過瀏覽器 <a href="https://ordinals.hiro.so/" rel="noopener noreferrer" target="_blank">ordinals explorer</a> 查看。此行為被稱之為「銘文」（Inscriptions）。</p><p>Casey Rodarmor 強調不同於以太坊等區塊鏈的 NFT 項目，往往將電子檔案儲存於中心化伺服器，需要定期審查其智能合約以確定有無後門、漏洞等疑慮；但銘文則沒有這些缺陷，所有資訊直接寫入無法竄改的比特幣區塊鏈中。</p><hr><hr><h2>▇ <strong>比特幣 NFT 運作原理</strong></h2><h3><strong>≣ 序數理論（Ordinal Theory）</strong></h3><p>首先，NFT 是一種專門紀錄「非同質化」資產的資料單位，亦有著「不可分割」的性質。</p><pre class="ql-syntax" spellcheck="false">💡 區塊鏈小知識：同質化 vs 非同質化首先「同質化」是指資產可與同類型資產互換、或是可拆分的特性。舉例來說，比特幣、以太幣和其它加密貨幣，又或是你手邊的新台幣，都是一種同質化的資產，因為與同類型資產之間，價值相等且能夠切割成更小的單位。如果今天我拿 1 張千元大鈔，跟你的 10 張枚百元鈔票做替換，你會覺得無所謂，因為兩者的價值相同。而「非同質化」自然是指獨一無二，無法與同類型資產互換、拆分的資產。舉例來說，畢卡索的畫作，蘇富比的拍賣品，又或是不動產權狀，每件資產都與眾不同，你無法找到第二件一模一樣的資產。就算出自於同一人（畢卡索）之手，他的每一件作品都擁有不同的價值，彼此無法互相替換。要注意的是，同質化與非同質化的定義是可以改變的。如果一件同質化資產被賦予內在價值或獨有特徵，便可重新定義為一件獨一無二的非同質化資產。例如，我的千元大鈔擁有周杰倫的簽名，那它就變成非同質化資產，反之亦然。</pre><p>而由 Casey Rodarmor 所發展的序數理論（Ordinal Theory） ，這套技術巧妙利用了比特幣「不可分割」的最小單位 SAT，把每個 SAT 都加上編號，類似大家在鈔票上看到的編號那樣，讓大家方便識別出每個 SAT，進而可以追蹤和轉移「特定」的 SAT。</p><p><strong>序數理論給了 SAT 個體身份</strong>，這讓原本彼此同質化的 SAT 現在有了「非同質化」的標記，允許它們被追蹤、轉移並賦予意義。</p><p><br></p><h3><strong>≣ 銘文（Inscriptions ）</strong></h3><p>再者，NFT 是透過「元數據」（metadata）傳達相關附加資訊，像是作者、創建日期、描述等，也可以是指向 NFT 所代表實際資產（例如藝術品或音樂）的超連結。</p><p><strong>銘文正是比特幣 NFT 的元數據概念</strong>，透過這項技術把資料「刻」在 SAT 上，如此一來每個聰 SAT 就會指向到特定的資訊，至此 SAT 在本質上已經是 NFT 了。</p><figure class="image"><img src="https://imagedelivery.net/kDRCweMmqLnTPNlbum-pYA/prod/embed/5fdfe3c4-c2e3-4d2e-894d-29bccdfeb9e6.jpeg/public" data-asset-id="5fdfe3c4-c2e3-4d2e-894d-29bccdfeb9e6" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><hr><hr><h2>▇ <strong>比特幣 NFT 特色，和以太坊 NFT 有哪些不同？</strong></h2><h3><strong>≣ 比特幣 NFT 儲存在鏈上</strong></h3><p>承前所述，銘文被「直接」寫入 SAT 中，<strong>也就是儲存在比特幣區塊鏈上</strong>，無法引用鏈下內容，這一點造就了比特幣 NFT 和其他區塊鏈 NFT 之間的巨大差異。</p><p>拿以太坊 NFT 為例，其本質是以太坊區塊鏈上的代幣，透過代幣指向作品本身。也就是説，以太坊 NFT 使用一種「間接」的方式，代幣像是一個路標，引導我們找到作品的位置；然而，<strong>問題在於儲存作品的地方通常在鏈下</strong>，具有一定程度的遺失風險，也可以被修改，與百分之百儲存在鏈上的比特幣 NFT 相比截然不同。</p><h3><strong>≣ 比特幣 NFT 沒有系列概念</strong></h3><p>在比特幣區塊鏈上，沒有智能合約的概念，因此也沒有「系列」的概念。比特幣 NFT 只能依靠中心化或共識機制，來把比特幣 NFT 歸類成群；而在以太坊區塊鏈上，可透過智能合約來創建系列，例如有名的 CryptoPunk、無聊猿（Bored Ape Yaught Club, BAYC） NFT 系列就是透過智能合約撰寫而成。</p><h3><strong>≣ 比特幣 NFT 玩法單調</strong></h3><p>沒有智能合約的比特幣區塊鏈，其 NFT 很難進行修改和更新，大部分比特幣 NFT 都是靜態形式；而以太坊 NFT 由於有智能合約，因此可以進行很多豐富有趣的玩法，例如合成、突變、進化、燒毀等。</p><figure class="image"><img src="https://imagedelivery.net/kDRCweMmqLnTPNlbum-pYA/prod/embed/adfbd7e3-a168-4168-9ed8-baa25e0ffbbb.jpeg/public" data-asset-id="adfbd7e3-a168-4168-9ed8-baa25e0ffbbb" referrerpolicy="no-referrer"><figcaption><span>比特幣 NFT 特色，和以太坊 NFT 有哪些不同？</span></figcaption></figure><hr><hr><h2>▇ <strong>比特幣 NFT 熱門項目</strong></h2><h3><strong>≣ Bitcoin Punks</strong></h3><p>Bitcoin Punks 是第一個使用 Ordinals 協議，將以太坊 CryptoPunk 成功上傳到比特幣區塊鏈的項目，所有比特幣 NFT 皆由收藏家免費鑄造，不到一天的時間就已經完成所有鑄造。</p><p>Bitcoin Punks 官方稱：「為了製作這個網站，我們正在檢查上傳到 Ordinals 的每張圖片雜湊值（Hash），並將其與原始的 1 萬張 CryptoPunk 圖片進行比對。」</p><figure class="image"><img src="https://imagedelivery.net/kDRCweMmqLnTPNlbum-pYA/prod/embed/83784add-154a-46e7-8497-ea817e908fd8.png/public" data-asset-id="83784add-154a-46e7-8497-ea817e908fd8" referrerpolicy="no-referrer"><figcaption><span>Bitcoin Punks</span></figcaption></figure><h3><strong>≣ Ordinal Punks</strong></h3><p><strong>Ordinals Punks 是一個向「以太坊 NFT 始祖」CryptoPunks 致敬的項目</strong>。作為比特幣區塊鏈前 650 個鑄造系列之一，Ordinals Punks 系列由 100 個獨特的 NFT 組成，每個 NFT 都擁有一個由匿名 Web3 創建者 FlowStay 所開發的開源算法，所生成的 192 × 192 像素圖像。</p><figure class="image"><img src="https://imagedelivery.net/kDRCweMmqLnTPNlbum-pYA/prod/embed/54d225d7-17a1-43df-88ce-405937a70327.png/public" data-asset-id="54d225d7-17a1-43df-88ce-405937a70327" referrerpolicy="no-referrer"><figcaption><span>Ordinal Punks</span></figcaption></figure><h3><strong>≣ TwelveFold</strong></h3><p>TwelveFold 是「NFT 領域 IP 霸主」Yuga Labs 透過 Ordinals 協議所鑄造發行的比特幣 NFT。</p><p><strong>與此前 Yuga Labs 主要推出的頭像（PFP） NFT 不同，TwelveFold 是一款生成式藝術 NFT</strong>，每件作品都是 12×12 網格，結合 3D 圖形和手繪特徵；限量發行 300 件，其中 288 個以拍賣方式進行（預留 12 個用作公益），其中最高出價者為 7.11 BTC ，最低出價為 2.25 BTC，短短 24 小時便創造了超過 1600 萬美元的收入。</p><p>然而 TwelveFold 的拍賣流程極為簡陋，引發了加密社群的不滿，因為用戶需要兩個比特幣錢包地址，且必須將出價的 BTC 發送到 Yuga Labs 控制的 BTC 地址，退款也需要手動處理，被譏諷為「石器時代」。</p><figure
