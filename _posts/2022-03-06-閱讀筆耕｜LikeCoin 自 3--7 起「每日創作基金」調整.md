每日創作基金調整</strong></h2><p>爬到一則與大家攸關，是 <a href="https://matters.news/@edmond/like-coin-chain-proposal-23-endorse-a-committee-to-manage-the-creators-fund-bafyreiefu5a4tm7jnnq2vr7kqqs3v7tb2iosurroa57hj2bj6qxfqn3laq" rel="noopener noreferrer" target="_blank">創作基金委員會</a>（Creators Fund Committee, CFC）將於 3/7 起調整「<strong>每日創作基金</strong>」的公告，就是你在 Liker Land App 中查看，每日中午更新蹦出的「<strong>空心</strong>」長條狀部分，裡頭起的名字叫做「<strong>配對基金</strong>」。</p><p>通常 @Dasiy 會熱心地彙整好每次會議內容並發布至 Matters，<a href="https://matters.news/@daisy/%E6%89%93%E5%92%97%E9%87%9D-%E7%84%A1%E5%BE%97%E9%A3%B2%E8%8C%B6-%E8%A6%8B%E5%94%94%E5%88%B0%E5%AD%AB%E4%BB%94-%E5%9B%A0%E7%82%BA%E4%B8%AD%E5%92%97-covid-bafyreigwyplofba7wohouwxiuyu6nyuw2css2lheax562c57sdiixp3uzy" rel="noopener noreferrer" target="_blank">然近期微恙</a>。請容許我自作主張，把已知的消息張貼上來，並以「新人視點」稍加注釋，<strong><u>希冀降低溝通成本</u></strong>。</p><p>原文這樣提到：</p><figure class="image"><img src="https://assets.matters.news/embed/985b7346-522a-49fa-bb48-f4724e557910.png" data-asset-id="985b7346-522a-49fa-bb48-f4724e557910" referrerpolicy="no-referrer"><figcaption><span>資料來源：讚賞公民共和國 Discord 社群</span></figcaption></figure><h2><strong>✅ 第一段</strong></h2><blockquote><em>3/3（四）晚創作基金委員會第三次會議，討論後決議 2022 年 3 月 7 日起，每日創作基金撥款調整為 13,300 LIKE。</em></blockquote><p>現時「每日創作基金」的配發為 20,000 LIKE，這意謂著，若你保持原本的寫作步調（其它條件不變），<strong><u>預期所獲讚賞的 LIKE 數額大約打 66 折</u></strong>。</p><p>＊</p><h2><strong>✅ 第二段</strong></h2><blockquote><em>按委員會第一次會議共識的撥款原則，創作基金每月撥款金額是按讀者實際付出的贊助額，以 1:1 比例配對。然而估計上月讀者的每月實際贊助額只有 62,500 LIKE 左右，比原先大幅下跌；為了令創作者的收入不會一下子受太大影響，委員會決定在原本 1:1 的配對金額以外，再額外撥出一筆款項，令每月基金贊助額維持於 40 萬 LIKE，即大約每日分發 13,300 LIKE 為期一個月再檢討。</em></blockquote><p>首先大家必須明白那 62,500 LIKE 是怎麼來的？它是從「Civic Liker 讚賞公民」節點的總委託數額（理解為本金）乘上收益率所估算，其金額用作「<strong>讚賞公民</strong>」拍手時的 LIKE 派發，是 Liker Land App 中「<strong>實心</strong>」長條狀部分，亦為公告中的「<strong>贊助額</strong>」。</p><p>追蹤 <a class="mention" href="https://matters.news/@baoshin" target="_blank" data-display-name="鑫大叔" data-user-name="baoshin" data-id="VXNlcjo1OTc0OA">﻿<span>@鑫大叔</span>﻿</a> 製作的 <a href="https://datastudio.google.com/u/0/reporting/d801e66d-5de6-4046-8244-dcfd77a15d47/page/p_dhohm7ddsc" rel="noopener noreferrer" target="_blank">讚賞公民 Web 3報表</a>，目前「本金💰」約為 400 萬 LIKE 出頭。</p><figure class="image"><img src="https://assets.matters.news/embed/2ddbfbe3-de4d-42b9-b249-8341bf8fe58e.png" data-asset-id="2ddbfbe3-de4d-42b9-b249-8341bf8fe58e" referrerpolicy="no-referrer"><figcaption><span>讚賞公民 Web 3報表</span></figcaption></figure><p>這裡採用 Liker Land App 裡的名詞，便於大家對照。</p><p>委員會乃依據「<strong>讚賞公民（實心）</strong>」金額，採 1:1 比例，去映射「<strong>配對基金（空心）</strong>」金額。若照現在情況挹注，這每月 62,500 LIKE 除下來，相當於每日 2,080 LIKE 創作基金，<strong><u>降幅實在劇烈。故額外補錢，幫 62,500 LIKE 充值到 40 萬 LIKE</u></strong>，這就是每日 13,300 LIKE 創作基金的來由。</p><p>＊</p><h2><strong>✅ 第三段</strong></h2><blockquote><em>當作者實際贊助金額（即讚賞公民委託回報總數）上升，委員會將參考目前撥款額度再調整補助總額。委員會期望讀者的贊助能漸漸成為作者的主要收入，創作基金的輔助角色能愈來愈輕，最終協助作者建立獨立的經濟循環。</em></blockquote><p>委員會期望有朝一日「<strong>讚賞公民（實心）</strong>」能成長到自給自足，屆時無須再受「<strong>配對基金（空心）」</strong>輔助。白話地說，就是方才提到的「本金💰」持續壯大。欲成就此事，可以拆解為兩個面向來看：</p><ol><li><strong>增加讚賞公民 Web 3 新用戶的「人數」</strong><br class="smart">參考 <a class="mention" href="https://matters.news/@baoshin" target="_blank" data-display-name="鑫大叔" data-user-name="baoshin"
