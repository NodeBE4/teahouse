<p>4 月 14 日收聽 vocus 舉辦的 <a href="https://discord.com/channels/911982679602372631/911982679602372635/961936214359953408" rel="noopener noreferrer" target="_blank"><strong>Web3 X 創作者經濟！創作者與粉絲如何共創價值</strong></a><strong> </strong>AMA（Ask Me Anything）活動，邀請的來賓是《M觀點》／《科技巨頭解碼》的 Miula，聊聊從創作者的角度來看待 NFT，並介紹他的 NFT 發行計畫。</p><figure class="image"><img src="https://assets.matters.news/embed/3a5b9ffc-8e3f-4efc-b6e9-5759594aee30.png" data-asset-id="3a5b9ffc-8e3f-4efc-b6e9-5759594aee30" referrerpolicy="no-referrer"><figcaption><span>Web3 X 創作者經濟！創作者與粉絲如何共創價值 AMA</span></figcaption></figure><p>做了一些筆記整理，鎖定在前半段內容，分享之。</p><hr><hr><h2><strong>■ 為什麼要買這張「會員卡」？</strong></h2><p>我自己是《M 觀點》的忠實聽眾，知道 Miula 擅長以生活中遇到的事件，去類比（舉例）一個概念，讓大家都聽得懂；撇開 NFT 技術層面，要各位創作者先忘記 NFT 的事情，以「買家」的視角——<strong>最容易的想像莫過於「會員卡」了</strong>——給他們一個理由：</p><blockquote><em>為什麼要買這張會員卡？</em></blockquote><p>也許是單純「<strong>經濟上</strong>」的動機，事後賣掉價更高。或其它因素，也就是大家琅琅上口的「<strong>賦能（Utility）</strong>」實際用途。當然若項目方本身魅力值滿點，基於虛榮、炫耀、觀賞性等「<strong>收藏</strong>」考量，粉絲願意買帳亦是可行。</p><p>總之，如果你是一位想幫自己發行 NFT 的創作者，Miula 建議的首要之務，先拿張白紙逐一條列式盤點，<strong><u>從消費者的立場去揣摩，思考你為這 NFT 塞的功能，到底具備多少價值？賦予的價值究竟夠不夠</u>？</strong></p><p>把 NFT 看做是一種除了訂閱制，<strong><u>另一種為創作者帶來收入的手段，但不是目的</u></strong>，千萬別以為發行就一定賺錢，也別較真市場是熊是牛的「擇時（market timing）」問題，這些都不是最攸關的考量。</p><hr><hr><h2><strong>■ 理想 NFT 樣貌</strong></h2><p>接下來 Miula 以創作者角度，提出觀察與想法，是他心目中期許的理想 NFT 樣貌，或稱之為價值體系，歸結為三點：</p><ol><li><strong>NFT 是一個價值承載的載體<br class="smart"></strong>承上一大段討論，思考為 NFT 塞多少功能？創作者賦予 NFT 的價值必須<strong>具體，</strong>不要是很空泛的（騙錢的項目還不夠多嗎）。<br class="smart"><br class="smart"></li><li><strong>創造的價值隨時間經過而提高（增值）</strong><br class="smart">不是發行完就一灘死水。<br class="smart">這 NFT 的價值不是固定， 它融合了對未來的想像，讓大家朝著這願景，一齊努力去打造價值，讓它與時俱進。<br class="smart"><br class="smart"></li><li><strong>每一位參與者也是貢獻者，讓價值增長，再共享利益</strong><br class="smart">焦點不應擺在共享利益，絕對不是單指分潤（把錢分一分）這麼簡單，<strong>而是關注共同創建價值的這一段過程</strong>。<br class="smart"><br class="smart"></li></ol><p>至此，Miula 又舉了一個「荒地開墾」的例子來類比方才的論述。</p><p>把這塊土地切成 130 多份（此時台下聽眾數）做分配，大夥兒各有所長，與自己的人脈，齊貢獻一份心力：有人開超商，有人開餐廳，有人開診所，有人拉了一條公路⋯⋯群策群力讓這片區域共好、繁榮。</p><p>透過 NFT 工具，讓一群人開創價值，價值提高，每個人獲利也愈多。</p><hr><hr><h2><strong>■ 我的感想</strong></h2><p>話說回來，如何說服一群人「買地」，又扣回初時「為什麼要買這張會員卡」的理由，<strong><u>以及如何讓一群人有「拓荒」的動機</u></strong>，值得細細玩味。</p><blockquote><em>以前，是我單純看好你，我投資你，那你得好好幹啊！怎麼現時來到 Web3 時代，我也得一起下場搞事，我傻了嗎？給我一個心甘情願的說法，好不？</em></blockquote><p>那 NFT <strong>公開透明</strong>的（智能合約）分配機制，或許能添具一點誘因，但絕對不是全部，這是每一個想要發行 NFT 的創作者，都應該要好好思索的部分。</p><hr><hr><figure class="image"><img src="https://assets.matters.news/embed/235d2f9b-ab0a-4fe2-8ead-49b2b898ca47.png" referrerpolicy="no-referrer"><figcaption><span>閱讀筆耕的簽名檔</span></figcaption></figure><blockquote>🌱
