<div>
<p>本文為大家演示查看方格子 <a href="https://vocus.cc/vaf" rel="noopener noreferrer" target="_blank">vocus and friends</a> NFTs（簡稱VAF）的三個管道，過程中跑到 TOFUNFT 次級市場逛逛，最後把觀念帶回 OpenSea 平台如法泡製。</p><hr><hr><h2><strong>■ 方格子後台</strong></h2><p>昨晚是方格子 <a href="https://vocus.cc/vaf" rel="noopener noreferrer" target="_blank">VAF</a>「冷凍艙」投放的日子，意指尚不能瀏覽外觀，還需在特定時日「喚醒」裡面「休眠中」的角色，寫到這兒腦海里開始浮現電影《<a href="https://zh.wikipedia.org/zh-hant/%E6%99%AE%E7%BD%97%E7%B1%B3%E4%BF%AE%E6%96%AF_(%E7%94%B5%E5%BD%B1)" rel="noopener noreferrer" target="_blank">普羅米修斯</a>》的畫面。</p><figure class="image"><img src="https://assets.matters.news/embed/cbfd91a9-d327-4aeb-9c5c-97dbaeec9ad7.png" data-asset-id="cbfd91a9-d327-4aeb-9c5c-97dbaeec9ad7" referrerpolicy="no-referrer"><figcaption><span>後台查詢 VAF 空投</span></figcaption></figure><hr><hr><h2><strong>■ 豆腐交易所</strong></h2><p>除了在【❶ <strong>方格子後台</strong>】查看以外，我們也可以去日後 VAF 的次級市場【❷ <a href="https://tofunft.com/" rel="noopener noreferrer" target="_blank"><strong>TOFUNFT</strong></a>】交易所逛逛。</p><p>由於 VAF 鑄造於 Arbitrum 鏈上，得先幫小狐狸錢包添加對應網路（編按：如果你尚未安裝錢包，或是還不熟悉「手動」添加網路，詳見 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E7%82%BA%E9%A6%AC%E7%89%B9%E5%B8%82-nft-%E8%A8%88%E7%95%AB%E5%81%9A%E6%BA%96%E5%82%99-02-meta-mask-%E5%B0%8F%E7%8B%90%E7%8B%B8%E9%8C%A2%E5%8C%85-bafyreih6hktykp5bzxlp373aph5ck3wvyoba4z6ex4s5ccoelirqxqiyva" rel="noopener noreferrer" target="_blank">這篇</a> 教學文），極速懶人法是進到 <a href="https://bridge.arbitrum.io/" rel="noopener noreferrer" target="_blank">Arbitrum One Bridge</a> 觸發簽署動作，土法煉鋼法則是自己鍵入數值：</p><figure class="image"><img src="https://assets.matters.news/embed/83319c4b-31d3-43be-9e16-4291c007a7b8.png" data-asset-id="83319c4b-31d3-43be-9e16-4291c007a7b8" referrerpolicy="no-referrer"><figcaption><span>極速懶人法：至 Arbitrum One Bridge 添加網路</span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/19bc7023-b9cf-4138-a0e9-be82e97cedde.png" data-asset-id="19bc7023-b9cf-4138-a0e9-be82e97cedde" referrerpolicy="no-referrer"><figcaption><span>土法煉鋼法：至小狐狸錢包後台自己鍵入</span></figcaption></figure><pre class="ql-syntax" spellcheck="false">◍ 網路名稱：Arbitrum One◍ Network URL: https://arb1.arbitrum.io/rpc◍ 鏈 ID: 42161◍ Currency Symbol : ETH◍ 區塊鏈瀏覽器: https://arbiscan.io/</pre><p>＊</p><p>萬事俱備，終於可以來「吃豆腐」啦！</p><p>把網路切至 Arbitrum One 上，進入 <a href="https://tofunft.com/" rel="noopener noreferrer" target="_blank">TOFUNFT</a> 交易所，點擊右上角頭像，連結你的小狐狸錢包，授權完成後，找到「My NFTs」就可看到 VAF 乖乖地躺在那兒：</p><figure class="image"><img src="https://assets.matters.news/embed/0f4a3d87-14d5-46a2-b725-5c0547c6d300.png" data-asset-id="0f4a3d87-14d5-46a2-b725-5c0547c6d300" referrerpolicy="no-referrer"><figcaption><span>TOFUNFT 後台</span></figcaption></figure><p>可以切換語系，也能修改個資，選擇任何一件作品進去細瞧，輕輕地用游標碰觸圖案，順勢帶出如何幫自己「換頭像」的選單：</p><figure class="image"><img src="https://assets.matters.news/embed/83cef983-b392-4eeb-a583-863e3a14bf6b.png" data-asset-id="83cef983-b392-4eeb-a583-863e3a14bf6b" referrerpolicy="no-referrer"><figcaption><span>換上 Travelogger #53 頭像</span></figcaption></figure><hr><hr><h2><strong>■ 手機上把玩</strong></h2><p>「教練，我想在【<strong>❸ 小狐狸錢包 App</strong>】上把玩 VAF。」</p><p>好的，請先在手機上「手動」添加網路喔（詳見 <a href="https://matters.news/@penfarming/%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E7%82%BA%E9%A6%AC%E7%89%B9%E5%B8%82-nft-%E8%A8%88%E7%95%AB%E5%81%9A%E6%BA%96%E5%82%99-02-meta-mask-%E5%B0%8F%E7%8B%90%E7%8B%B8%E9%8C%A2%E5%8C%85-bafyreih6hktykp5bzxlp373aph5ck3wvyoba4z6ex4s5ccoelirqxqiyva" rel="noopener noreferrer" target="_blank">這篇</a> 教學文）！搞定後，並切換至 Arbitrum One 網路 ，發現小狐狸錢包 App 不會「自動」帶出 VAF，<strong>得自己添加收藏品</strong>。</p><p>所以為了取得 VAF 的資訊，我們回到 <a href="https://tofunft.com/" src="https://assets.matters.news/embed/d69671b7-289d-4429-993e-03056784e94f.png" data-asset-id="d69671b7-289d-4429-993e-03056784e94f" referrerpolicy="no-referrer"><figcaption><span>查找 CONTRACT 與 TOKEN ID</span></figcaption></figure><p>想看到完整合約地址（中間被「...」略過），於是點開它來到區塊鏈瀏覽器，利用歷史交易記錄裡的搜尋框（是正中間右側的放大鏡，可別找到右上角去了），輸入 TOKEN ID 檢索出全部資訊，複製 CONTRACT 完整地址：</p><figure class="image"><img src="https://assets.matters.news/embed/25d39d35-cff1-468d-aa3d-cd4be002437f.png" data-asset-id="25d39d35-cff1-468d-aa3d-cd4be002437f" referrerpolicy="no-referrer"><figcaption><span>輸入 TOKEN ID 檢索</span></figcaption></figure><figure class="image"><img src="https://assets.matters.news/embed/4eda135e-1883-4302-80c1-473270ca4098.png" data-asset-id="4eda135e-1883-4302-80c1-473270ca4098" referrerpolicy="no-referrer"><figcaption><span>複製 CONTRACT 完整地址</span></figcaption></figure><p>最後一動，只消在小狐狸錢包 App 裡「<strong>添加收藏品</strong>」，就可以把 VAF 展示出來，只是目前還沒解盲，所以灰濛濛的。</p><figure class="image"><img src="https://assets.matters.news/embed/b88bbfe5-3a79-4213-bf4d-616cef43cb65.png" data-asset-id="b88bbfe5-3a79-4213-bf4d-616cef43cb65" referrerpolicy="no-referrer"><figcaption><span>在小狐狸錢包 App
