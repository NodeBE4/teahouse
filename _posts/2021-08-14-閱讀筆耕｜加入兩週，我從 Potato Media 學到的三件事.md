<div>
<p>兩週前（七月底），我在 <a href="https://www.potatomedia.co/profile/a2a5c973-e410-4525-b94b-420ffe385348" target="_blank">Potato Media</a> 審核通過第一篇文章，從此以一天一篇的速度上架，都搬舊文；或許你會納悶，<strong>兵貴神速</strong>，怎麼拖了這麼久才拿這平台做一篇？主要是想多蒐集一些切入點，積累到一定程度，才與大家討論。</p><p>這篇文章不談「利」，試著從其它視點來品嚐馬鈴薯的滋味。<br class="smart"></p><hr><hr><h2><strong>■ 兩週的體驗</strong></h2><pre class="ql-syntax">✅ 首頁：老奶奶的碎花拼布</pre><p>一進首頁，迎面而來的是兩個<strong>【浮動廣告視窗】</strong>，得先花個 1～2 秒「清零」避免視覺干擾；文章分類以<strong>【五顏六色區塊】</strong>印上令人有點尷尬的小圖案呈現，像極了老奶奶的碎花拼布；不過目前是 Beta<strong> </strong>公測版本，未來理當會改善 UI 才是，應該吧！</p><figure class="image"><img src="https://assets.matters.news/embed/b0b58e96-c8e4-4d09-9355-fe808bb4e49a.png" data-asset-id="b0b58e96-c8e4-4d09-9355-fe808bb4e49a" referrerpolicy="no-referrer"><figcaption><span>Potato Media 首頁截圖</span></figcaption></figure><p>改以小螢幕瀏覽，手機模式下，貌似捨去掉分類選單（我找不到），變得比較清爽。</p><pre class="ql-syntax">✅ 編輯器：由奢入儉難</pre><p>必須說<strong>【文章編輯器陽春】</strong>到一個極致，若你早已玩遍其它寫作平台，必定會有「由奢入儉」的感概；然對於平素只接觸社群媒體（FB、IG、Twitter⋯⋯）的生手而言，有這些功能已是眼前一亮，要「由儉入奢」是相對容易的，這也是 Potato Media 鎖定的藍海 TA（Target Audience）吧！</p><figure class="image"><img src="https://assets.matters.news/embed/3998a702-e42d-4170-8716-6037fbdf50f8.jpeg" data-asset-id="3998a702-e42d-4170-8716-6037fbdf50f8" referrerpolicy="no-referrer"><figcaption><span>Potato Media 文章編輯器</span></figcaption></figure><pre class="ql-syntax">✅ 審查制：不遺餘力抓廢文</pre><p>聊一下<strong>【文章審核制度】</strong>，表定顯示要等上 1～3 天，雖然不利於時效性的文章，例如股市，但覺得整體而言<strong>利大於弊</strong>。</p><p>文章素質的「地板」墊高了，一些違法亂紀的行為自然不在話下（見 <a href="https://www.potatomedia.co/legal/terms" target="_blank">全站規定</a>），比較有趣的是那「擋水文」的舉措，除了<strong>【至少 200 字及一張圖】</strong>的客觀標準外，更加上<strong>【內容空洞、不明所以】</strong>的主觀判定⋯⋯。</p><p>看到前者的字數底限，或許有些詩人／畫家要跳腳了，但起碼是具體準則，沒有模糊空間；而後者的「言論審查」令人嘖嘖稱奇，<strong>人家馬特市、方格子的官方是精挑細選精華文，而馬鈴薯的小編則是不遺餘力抓廢文</strong>。</p><p>根據 <a class="mention" href="https://matters.news/@abundancelife" target="_blank" data-display-name="開掛人生" data-user-name="abundancelife" data-id="VXNlcjo0MDU3Nw">﻿<span>@開掛人生</span>﻿</a> <a href="https://matters.news/@abundancelife/potato-media%E6%96%87%E7%AB%A0%E5%AF%A9%E6%A0%B8%E5%A4%B1%E6%95%97%E7%90%86%E7%94%B1-and-%E8%A7%A3%E6%B1%BA%E6%96%B9%E6%B3%95-bafyreihiugqje77zdhnnrhju2vhjezrwwm2zytozxpsbb5gugrkcymrxhe" target="_blank">的經驗分享</a>，單純「心情抒發」的流水帳可過不了關。要讓文章具有「建設性、實用性」，提出自己的見解，才能上得了檯面。</p><p>這也呼應前面說法，Potato Media 吸納了一票來自社群平台的生手，卻也希望投稿的文字，通俗歸通俗，還是得和「社群文」做出區隔。</p><hr><hr><h2><strong>■ 學到三件事</strong></h2><p>有一回，說一篇舊文 <a href="https://matters.news/@penfarming/%E8%9C%9C%E8%9C%82%E8%88%87%E9%81%A0%E9%9B%B7-%E8%88%87%E4%B8%83%E9%BE%8D%E7%8F%A0%E7%8B%82%E6%83%B3-bafyreiavj2gvy5qcybwrxaonlwilbmfjy4cwttf36ccx7gh2l3gliuksdq" target="_blank">《蜜蜂與遠雷》與七龍珠狂想</a> 因為「複製文句過多」而退件；倒是沒想到附上原文連結這招，我直接把文章中的第二大段（狂想）砍掉後重送，竟順利過關。</p><p>也想起 Potato Media 的文章分類沒有「閱讀」，只能往「生活娛樂」裡送審。可見站方不是很重視這塊領域，先天不足，再加上 TA 屬性，後天失調——故「因地制宜」有其必要性，把落落長的文章，精萃成簡明扼要的版本。</p><blockquote><strong>⚠️ 第一件事：練習「減法」的藝術。</strong><br class="smart"></blockquote><p>於是反覆試驗，馬鈴薯很大度，容忍我委婉地吐槽 😭：</p><figure class="image"><img src="https://assets.matters.news/embed/79d292b6-469e-49fa-895c-ec923a2d528c.png" data-asset-id="79d292b6-469e-49fa-895c-ec923a2d528c" referrerpolicy="no-referrer"><figcaption><span>張貼於 Potato Media 的文章節錄：《蜜蜂與遠雷》與七龍珠狂想</span></figcaption></figure><p>.</p><p>以前很懶惰（現在還是會），文章【<strong>標題】</strong>直接套上書名，清湯掛麵扔上去就完事了；把握良機，跟早期的文章「重修舊好」，仔細琢磨這次該下什麼標題好呢？</p><p>另外是文章的<strong>【封面圖】</strong>。向來習慣為閱畢的書籍拍一張照，由於書是直立的，很直覺地也用直拍拍攝，結果直式照片遇上橫幅顯示，文章預覽圖一整個被裁切得醜醜的；這次一律橫拍，還要統一背景色系。</p><blockquote><strong>⚠️ 第二件事：與舊文「重修舊好」。</strong></blockquote><figure class="image"><img src="https://assets.matters.news/embed/8ecb2591-47b2-436d-9734-aa601c0eb10f.png" data-asset-id="8ecb2591-47b2-436d-9734-aa601c0eb10f" referrerpolicy="no-referrer"><figcaption><span>舊有文章的重新打磨</span></figcaption></figure><p>.</p><p>目前 Potato Media 處在一個「砸錢拼市占」的階段，宛如蝦皮當年的 0 元運費補貼策略。</p><p>雖不知海水退潮後，有多少人會留下來，但這「蜜月期」的流量是可觀的；由於功能還不是很完善（文章編輯器最有感），不如順水推舟，讓更多人知道尚有「奢侈」的選項存在，也是為自己既有的平台增添能見度。</p><blockquote><strong>⚠️ 第三件事：引流，增加曝光。</strong></blockquote><figure class="image"><img src="https://assets.matters.news/embed/1349497c-c355-4faa-8c2f-57c327a5829e.png" data-asset-id="1349497c-c355-4faa-8c2f-57c327a5829e" referrerpolicy="no-referrer"><figcaption><span>Potato Media
