<div>
<p>說到 NFT，許多人腦海裡第一個浮現的是 <a href="https://zh.wikipedia.org/zh-tw/%E4%BB%A5%E5%A4%AA%E5%9D%8A" rel="noopener noreferrer" target="_blank">以太坊</a>（Ethereum）與它的擴容，不過在 <a href="https://g.co/kgs/wkDWHR" rel="noopener noreferrer" target="_blank">Solana</a> 上亦有許多值得關注的作品，而且它們已經正式登陸於 <a href="https://zh.wikipedia.org/zh-tw/OpenSea" rel="noopener noreferrer" target="_blank">OpenSea</a> 交易市場了呢！ </p><p>「好想參與 Solana 生態系項目啊！」</p><p>但是 <a href="https://matters.news/@penfarming/188962-%E9%96%B1%E8%AE%80%E7%AD%86%E8%80%95-%E7%82%BA%E9%A6%AC%E7%89%B9%E5%B8%82-nft-%E8%A8%88%E7%95%AB%E5%81%9A%E6%BA%96%E5%82%99-02-meta-mask-%E5%B0%8F%E7%8B%90%E7%8B%B8%E9%8C%A2%E5%8C%85-bafyreih6hktykp5bzxlp373aph5ck3wvyoba4z6ex4s5ccoelirqxqiyva" rel="noopener noreferrer" target="_blank">小狐狸錢包</a>（MetaMask Wallet）並不支援 Solana。沒關係，今天要為大家介紹的 <a href="https://phantom.app/" rel="noopener noreferrer" target="_blank">鬼鬼錢包</a>（Phantom Wallet）是我在玩 <a href="https://twitter.com/Stepnofficial" rel="noopener noreferrer" target="_blank">STEPN</a> 時——首次參與 Solana NFT 項目——接觸研究後，最推薦的 Solana 錢包。簡單易用好上手，而且還很可愛（笑），一起來瞧瞧吧！</p><hr><hr><h2><strong>■ Phantom 簡介</strong></h2><p>Phantom 是一個新創團隊，致力為用戶打造易用性（making crypto easier for the world to use）的加密貨幣體驗，<a href="https://phantom.app/" rel="noopener noreferrer" target="_blank">鬼鬼錢包</a>（幻影錢包） 是它們的第一個產品。</p><figure class="image"><img src="https://assets.matters.news/embed/bad39ff1-5ddb-4375-9f8d-2a0cbd28caf9.png" data-asset-id="bad39ff1-5ddb-4375-9f8d-2a0cbd28caf9" referrerpolicy="no-referrer"><figcaption><span>資料來源：Phantom 官網</span></figcaption></figure><p>Phantom 於 2021 夏季取得 900 萬美元的Ａ輪融資，由 <a href="https://zh.wikipedia.org/zh-tw/Andreessen_Horowitz" rel="noopener noreferrer" target="_blank">a16z</a> 領投，Variant Fund、Jump Capital、Solana Foundation 也參與投資。</p><figure class="image"><img src="https://assets.matters.news/embed/19a97179-fcb5-4a40-91d8-03c525cd7396.png" data-asset-id="19a97179-fcb5-4a40-91d8-03c525cd7396" referrerpolicy="no-referrer"><figcaption><span>資料來源：Phantom 官網</span></figcaption></figure><hr><hr><h2><strong>■ 鬼鬼錢包的特色</strong></h2><ul><li><strong><u>支援眾多瀏覽器、裝置。</u></strong></li><li><strong><u>支援「繁體」中文。</u></strong></li></ul><p>做為插件，適用於各大瀏覽器，這是在電腦上操作。亦有手機（<a href="https://apps.apple.com/app/phantom-solana-wallet/1598432977" rel="noopener noreferrer" target="_blank">iOS</a>／<a href="https://play.google.com/store/apps/details?id=app.phantom" rel="noopener noreferrer" target="_blank">Android</a>） App 版本。語系同時支援繁體、簡體中文。</p><figure class="image"><img src="https://assets.matters.news/embed/a37cd8fd-cf28-4635-97bf-6e258dcac194.png" data-asset-id="a37cd8fd-cf28-4635-97bf-6e258dcac194" referrerpolicy="no-referrer"><figcaption><span>資料來源：Phantom 官網</span></figcaption></figure><p>＊</p><ul><li><strong><u>可以收藏 NFT。</u></strong></li></ul><p>「難道我買了一個 NFT，還得打開電腦，才能炫耀給朋友看嗎？」</p><p>不論是在電腦、手機上使用（基於隨時隨地秀一波的理由，後者尤為重要），錢包裡特別獨立出 NFT 展示櫃，讓你的每一件收藏品，都美美的收納在一起。</p><figure class="image"><img src="https://assets.matters.news/embed/ae00bd28-0d82-47a4-97bc-0838a6217f87.png" data-asset-id="ae00bd28-0d82-47a4-97bc-0838a6217f87" referrerpolicy="no-referrer"><figcaption><span>資料來源：Phantom 官網</span></figcaption></figure><p>＊</p><ul><li><strong><u>賺取 SOL 代幣。</u></strong></li><li><strong><u>又快又便宜的兌幣。</u></strong></li></ul><p>點進 SOL 代幣，就會看到一個「<strong>🌟 開始賺取 SOL 代幣收益</strong>」的選項，透過質押（Stake）給驗證人的方式，賺取收益。</p><p>我也把從 <a href="https://twitter.com/Stepnofficial" rel="noopener noreferrer" target="_blank">STEPN</a> 掙得的 GST 代幣，在 <a href="https://phantom.app/" rel="noopener noreferrer" target="_blank">鬼鬼錢包</a> 裡兌換成 USDC 代幣，因為跑鞋 App 內建的錢包不太好用，常常發生兌幣失敗情況。</p><figure class="image"><img 官網</span></figcaption></figure><p>＊</p><ul><li><strong><u> 連接去中心化程式應用（DApp）。</u></strong></li></ul><p>和 <a 交易。暢遊 Web3 不是電腦的專利，用手機也通喔！<br class="smart"></p><figure class="image"><img src="https://assets.matters.news/embed/6b95c4b0-b6cd-40d9-8b1e-4485bb35a4ff.png"
