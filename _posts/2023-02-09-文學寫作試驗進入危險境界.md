referrerpolicy="no-referrer"><figcaption><span>神秘的黄昏，野鸭在游泳</span></figcaption></figure><p><strong>小引：</strong></p><p>我与【一百零四 / 特洛伊】进行的网路互动式文学写作竞赛既是一种艺术竞赛，也是一种科学竞赛，因为竞赛参加者要测试对手以及受众的反应并作出相应的调整。这种测试和调整以及调整的测试无疑都属于科学的范畴。</p><p>简而言之，我与【一百零四 / 特洛伊】竞赛的主题是文学评论与写作。TA在这样的竞赛中突如其来地高叫并炫耀TA会写电脑程式，会使用Python等编程语言。TA的这种表现清楚地显示了TA的傻。这就像在讨论烹调艺术的场合，有人突如其来在一边大叫他擅长污水处理，或在讨论婴儿百岁庆祝安排的场合，有人在一边突然大叫他擅长殡葬安排。</p><p>这里的问题不是污水处理或殡葬安排。污水处理或殡葬安排当然也是非常重要的事情，是社会生活正常运作不可或缺的。但在不恰当的场合坚持大喊大叫污水处理和殡葬安排就是犯傻，甚至是傻得没边了。不幸的是，当今中国的教育欠缺基本的人文教育和逻辑思维教育，因此盛产这种傻缺产品。【一百零四 / 特洛伊】便是典型的这种残缺教育的残缺产品。</p><p>或许部分是由于我发表的《（小說）津輕海峽與特洛伊首次線下會面（4）》给了TA过大的刺激，TA作出了强烈的、激烈的反应。我必须说，TA的反应让我感到非常担心，因为TA在其反应中不但重弹老调和傻调，而且还清楚地展示出了疯狂，比如说TA的长篇幅反应的第一段：</p><blockquote>咋不直接掏出Ulysses，关于文学，如果你一本正经就是在找死（除非你自己写出过传世的佳作）；掏出一本《线性代数》才是对家庭妇女绝对的打击。</blockquote><p>在这里，读者可以看到【一百零四 / 特洛伊】一如既往地语无伦次，一如既往地晒TA的傻书单（注：《尤利西斯》Ulysses是西方现代派文学名著），一如既往地晒TA的傻数学，外加一如既往地展示TA的仇女/厌女的意识和下意识。读者也可以看到，TA虽然持续自称女性，却对女性的身体和智力极尽讽刺挖苦之能事，好像不知道在当今世界以及古代世界，女性的数学能力一点也不比男性差，现在世界一流的数学家女性也不在少数。</p><p>对我这个研究文学和社会科学的人来说，【一百零四 / 特洛伊】在这段话中最让我感到惊讶乃至惊恐的是TA所清晰展示出来的反学术、反社会、反文明乃至反人类的意识或下意识。【关于文学，如果你一本正经就是在找死（除非你自己写出过传世的佳作）】，TA这话的意思显然是，自人类有史以来，99%99999的学者都是垃圾，都是找死，都不值得一提。TA显然完全不知道文明和学术建设从来都是一个日积月累集腋成裘的过程。</p><p>【一百零四 / 特洛伊】为什么会说这样的疯话？难道真的是因为读了我的《（小說）津輕海峽與特洛伊首次線下會面（4）》受到的刺激太强烈而发疯了吗？想到这里，我不禁感到非常不安，脊梁上满是冷汗。我一直认为，无论是艺术试验还是科学试验，试验者必须关怀被试者的福祉与安全，这是试验者在设计和执行试验时必须坚持的伦理道德底线．</p><p>与此同时，作为一个人文和科学研究者，我也知道我不应当想当然，看到【一百零四 / 特洛伊】说疯话就立即想当然地以为TA已经精神失常．在精神科大夫给TA进行彻底全面的检查之前，我认为负责任的看法和说法应当是：【一百零四 / 特洛伊】明显是一个喜好抖机灵的傻叉，因此我应当本着诲人不倦的精神刺激TA，教育TA，开导TA，启发TA．</p><p>罗马不是一天建成的，愚昧也不是一天可以破除的，我必须不断提醒自己不能着急，要慢慢来．（说实话，当教师有时候让我感到很有成就感，有时候又让我感到丧气．遇到【一百零四 / 特洛伊】这样的傻呼呼还喜欢穷抖机灵的学生就让我丧气．唉～）</p><p>【一百零四 / 特洛伊】对我发表的《（小說）津輕海峽與特洛伊首次線下會面（4）》作出了评论．以下是我给出的回应．这一次，我仍是没有按照先来后到的次序把TATA的评论放在前面，还是因为TA的文笔比较差，写出来的东西虽然催眠效果强大，但娱乐性贫弱．我不想考验读者的耐心，只想给读者娱乐．所以，我就把我自以为比较娱乐的回应放在前面了．</p><p>我当然也希望读者有正确的阅读态度，要尽力避免因人废言．假如哪位读者认为【一百零四 / 特洛伊】的疯言疯语有什么可取之处，我也愿意洗耳恭听．<br class="smart">－－－－－－－－－－－－－－－</p><p>亲爱的傻大姐【一百零四 / 特洛伊】呀，您憋了好几天，最终还在这里高叫【我会缝扣子】呐？傻大姐，您傻成这样，也是简直了。您先前不是显摆会写两句英语嘛，怎么又不写啦？现在您又在您的傻乎乎的编程语言上加上《线性代数》啦？傻大姐呀，您就是一个活脱脱的傻的化身呐。</p><p>我觉得王小波的文章常常失之于点到为止，不能深入。在描述和刻画傻大姐的时候，显然王小波也有这样的缺陷。王只是说了傻大姐动辄高叫【我会缝扣子】，没说傻大姐还会叫【我会缝铜扣子】，【我会缝塑料扣子】，【我会用棉线缝扣子】，【我会用尼龙线缝扣子】，......。</p><p>傻大姐呀，您给我洋洋洒洒留言，很难得，但您一开头就再度展示了您的不可救药的傻，我只有叹息。只要您继续傻，我作为一个正常人是没法跟您进行对话甚至没法进行一般的交谈的。这个道理，您明白嘛？</p><p>为什么说您傻得没边，让正常人无法跟您对话呢？下边这篇文有非常简单清楚的解说。我当然知道您很可能看不懂。不要紧，您可以找您的朋友帮您看，帮您解读。</p><p>当然，我也知道您的朋友大概率也是傻。但我希望其中有几个还不是傻得没边。</p><p>唉，我也是爱心泛滥，与人为善之心泛滥呐。怕是也有点傻了。</p><p>===============</p><p>亲爱的［一百零四／特洛伊］呀，说您傻什么意思呢？唉，我再次跟您讲讲道理吧，虽然明知您傻得没边，大概率是听不懂的。</p><p>比如说，一群人聚在一起讨论如何修理马桶和下水道，我因为某种原因（比如说，由于也有兴趣，由于家里也有马桶和下水道），也去参加讨论。这是没问题的。绝对没问题的。但在讨论的过程中，我突然大叫，而且反复大叫：【我懂日语，你们懂吗？】并且还以此来挑战所有的人，那我就是一个傻逼了。纯粹的傻逼，就是一个笑料了。（谢天谢地，我没那么傻）</p><p>再比如说，一群人聚在一起讨论文学，一个人（比如说您）由于某种原因也跟过去参加讨论。这当然没问题。但那个人在讨论过程中突然大叫，而且反复反复大叫：【我会电脑编程，你们会吗？你们写给我看看】，并以此挑战在场的所有的人，那个人就是个纯傻逼，就是笑料了。您就是这样的一个傻逼。很不幸的是，您就是个笑料，您傻得没边。到现在还继续展示您的傻。</p><p>王小波的傻表姐为什么傻？也是同样的道理。别人讨论天气，她会在一边大叫：我会缝扣子。别人讨论如何做糖醋鱼片，她会在一边大叫：我会缝扣子。她傻并不是因为她会缝扣子，您傻也并不是因为您会写两行电脑程式，而是因为她跟您一样不懂得自己的那点技能是狭窄的，不懂得不能傻乎乎用自己的狭窄知识去越界碾压别人。实际上，她和您也碾压不了任何人，她和您只是众人的笑料。</p><p>亲爱的，我希望您能早一点明白这个道理。但我知道您傻，你大概不会明白。您太笨了。</p><p><strong>附　［一百零四］留言评论</strong>《（小說）津輕海峽與特洛伊首次線下會面（4）》<strong>：</strong></p><blockquote>但在这时候，津轻海峡弯腰从随身带的包包里掏出两本书。足足一斤重的那一本是大名鼎鼎的文艺批评理论书The
