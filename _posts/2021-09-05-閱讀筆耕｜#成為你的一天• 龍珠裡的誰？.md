<p>如果可以體驗漫畫裡面的人生，你想進入哪一部作品？</p><p>■</p><p>我會毫不猶豫地選擇《<a href="https://g.co/kgs/6RLGaQ" target="_blank">七龍珠</a>》，原因無它，因為<strong>最強</strong>，震碎戰鬥力觀測器。</p><figure class="image"><img src="https://assets.matters.news/embed/8aafd59f-b61f-47e2-96f8-fe8581ec15a9.jpeg" data-asset-id="8aafd59f-b61f-47e2-96f8-fe8581ec15a9" referrerpolicy="no-referrer"><figcaption><span>動畫《七龍珠》</span></figcaption></figure><p>要知道 <a href="https://zh.wikipedia.org/zh-tw/%E5%BC%97%E5%88%A9%E6%B2%99" target="_blank">弗利沙</a> 的設定，<strong>可以在沒有空氣的宇宙生存</strong>，厲害了，只可惜他是大反派，哭哭。還是循規蹈矩地回到正派角色身上呢！</p><p>我的野心很小，只要學會 <a href="https://baike.baidu.hk/item/%E8%88%9E%E7%A9%BA%E8%A1%93/15442338" target="_blank">舞空術</a> 就好，如此一來，上班、上學就不會塞車，避免遲到；但某種程度上，依然要「親力親為」，若想體驗「自動駕駛」的悠閒，還得弄一台 <a href="https://baike.baidu.hk/item/%E7%AD%8B%E6%96%97%E9%9B%B2/23608348" target="_blank">筋斗雲</a> 呢！再更貪心一點，追求最高的移動效率，那麼 <a href="https://baike.baidu.hk/item/%E7%9E%AC%E9%96%93%E7%A7%BB%E5%8B%95/3948876" target="_blank">瞬間移動</a> 就是必備。</p><blockquote>⚠️ 要同時滿足以上條件，我想成為 <a href="https://zh.wikipedia.org/wiki/%E5%AD%99%E6%82%9F%E7%A9%BA_(%E4%B8%83%E9%BE%99%E7%8F%A0)" target="_blank">孫悟空（卡卡洛特）</a>。</blockquote><p>■<br class="smart"></p><p>然而，和《<a href="https://g.co/kgs/q95rFs" target="_blank">名偵探柯南</a>》如出一轍。</p><p>有柯南在的地方，就有人犯下謀殺案；有孫悟空在的世界，屢有地球爆炸的災難，於是救亡圖存的重責大任落在這群 <a href="https://zh.wikipedia.org/wiki/%E8%B3%BD%E4%BA%9E%E4%BA%BA" target="_blank">超級賽亞人</a> 身上。天吶！<strong>能力愈強，責任愈大</strong>。</p><p>孫悟空本人似乎早有自知之明，不論是太平盛世，抑或是兵馬倥傯，<strong>ㄧ生都是不斷地修煉，除了練功還是練功</strong>，落下 <a href="https://g.co/kgs/zqBK5p" target="_blank">琪琪</a> 獨守空閨，事業與家庭完全失衡。</p><p>但孫悟空對太太還是很尊重的，琪琪對丈夫的「工作」也算包容，只立下別讓 <a href="https://g.co/kgs/bNmQ1q" target="_blank">悟飯</a>「學壞」這條規矩（但他後來還是「學壞」了）。</p><blockquote>⚠️ 假如我變成孫悟空，第一件事情就是「<strong>齊家</strong>」，搖身一變，從拋妻棄子，變成相妻教子的模範先生。</blockquote><figure class="image"><img src="https://assets.matters.news/embed/3ec48712-93b6-4226-89e7-458eeb2316e8.jpeg" data-asset-id="3ec48712-93b6-4226-89e7-458eeb2316e8" referrerpolicy="no-referrer"><figcaption><span>動畫《七龍珠》</span></figcaption></figure><p>■</p><p>事情沒這麼簡單，對吧！孫悟空就像一塊吸鐵，雖不去招惹，但時間一到，邪惡勢力紛紛找上門⋯⋯。</p><p>嚴格來說，拯救地球也是拯救家人，是一種犧牲奉獻、大愛無私。</p><p>快去吧！勿做籠中鳥，天空才是你翱翔的舞台。<br
